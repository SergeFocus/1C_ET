Разработанный, чтобы загрузить или обновить элементы данных каталогов, списки, документы или записи зарегистрировать базу данных.

Основной задачей института

Быстрый и легкий импорт необработанных данных - список ввода имени, остаточная стоимость его цены.

Тем не менее, он может быть использован для других каталогов и импорта регистра (такие как цена поставщика).

После выбора инструмента импорта для выбора, который помещается в столбцы в таблице, в которой данные
В примере 1, колонка - название, 2 - статья (включить в настройках), 3 - наименование изготовителя (активация счета производителей товаров установка), 4 - оставшееся количество, 5 - цену.

При публикации указанного количества и (или) цена может быть четко прописано в импорте товаров.
Может быть, в течение небольшого числа различных магазинов.Склад может быть определено для каждой строки данных ( "Импорт объектов"), и весь столбец (далее в качестве вкладки "поведение").
С видом той же цене - в сложных расчетных цен можно загрузить несколько типов цен.Посмотреть цена может быть определена для каждой строки или весь столбец.

Значения столбцов вручную.Можно задать один и тот же столбец более одного раза (например, название общей группы, чтобы установить имя производителя и груза).

Галки В разделе «Если пусто - с предыдущим" означает, что столбец не может заполнить все строки.Например, производители называют один раз, а затем все следующие строки - о своем продукте.Просто, когда он заканчивается блок данных одного источника - значения в этом столбце ячеек вновь производителя (с именем следующего блока данных).Таким образом, программа (включена ОУПЖ) сохраненное значение, и назначить его в колонке все пустые ячейки.Если проверка не прошла - программа для чтения каждой строки значения и использовать его в качестве пространства незаполненной.

Колонка Галки "Создать" позволяет создавать при импорте напрямую связана с простой каталог.Только в качестве ссылки детального типа.
Таким образом, в этом примере показывает, что если элемент каталога "производители" не по имени (значение в этой колонке) найден, вы будете создавать новое имя каталога элементов "производителя", и он уже установлен для производителей проект.

Если последовательный учет может быть сделано немедленно импортировать именованный информацию о конкретной семье.Если столбец данных пикап серии, то этот ряд будет найден / создан, по имени импорта об этом.Тогда имя может быть различным "серии" линия "повтор".
При импорте отгрузки остатков, необходимо колонке "Нет", и "1" заполнена.Любое другое число больше нуля, программа приведет к одному.
Если вы не укажете номер или ноль, серия будет создана, но регистрационные данные не будут находиться в равновесии.

С помощью команды в этом блоке, вы можете сохранить общие параметры импорта версии и восстановления.А также сохранить внешний файл, чтение из файла.

Давайте посмотрим на странице "поведение":


Есть единственный способ указать данные, которые мы использовали, чтобы определить оценку каждой линии.В этом примере, каждый из которых определяется уникальным именем номенклатуры, а - продукты.То есть, если повторяющиеся имена и код - это та же терминология.Не забудьте установить уникальный для того, чтобы избежать повторения параметров процесса импорта.

Анг, "только обновление" указывает на то, что элементы, новые данные не будут созданы.Не было обнаружено из-за строк данных в поле Ключ (блок 1) будет просто игнорировать.Если такие данные (в данном случае, номенклатура) будет обнаружен - программа будет проверять детали и будет обновлять таблицу в счет импорта.
Галки "Пропустить существующие" предполагает, что если в процессе импорта существующие функции будут найдены в приоритетных областях, они пропустят программу и не будет обновляться, так как данные из таблицы.
Если две вороны пустые, то программа будет обновлять существующие элементы данных и добавлять новые.
Когда Галки "Новая группа" Импорт данных иерархических каталогов (например, имя), появится.Если группа дается в таблице данных в отдельном столбце, то вы можете установить стоимость клещами - Создать новую группу (если не найден) или нет.

На балансе данных документа и цены будут падать.
