Die Hauptaufgabe des Mechanismus

Schnelle und einfache Import von Rohdaten - Eintrag Liste der Nomenklatur, der Preis, den Restwert.

Es kann jedoch für die Einfuhr von anderen Verzeichnissen und Registern verwendet werden (zB Lieferanten Preis).

Nach dem Importfunktion die Auswahl, in der Spalte der Tabelle zu wählen, platziert die Daten
In Beispiel 1, Spalte - Name, 2 - Artikel (in den Einstellungen aktiviert ist), 3 - Name des Herstellers (Konto aktiviert Warenhersteller in den Einstellungen), 4 - die Rückstandsmenge, 5 - der Preis.
Wenn die Anzahl oder Preis nicht erwähnt wird - wird einfach ein Element Nomenklatur Verzeichnis erstellen.
Wenn die Menge angibt und (oder) Preis können sich registrieren, werden genau die Entsendung der Ware bei der Einfuhr.
Vielleicht ist die Zahl für mehrere verschiedene Geschäfte.Warehouse kann für jede Zeile von Daten bestimmt werden (auf der "Import-Objekt") und für die gesamte Spalte (auf dem "Verhalten" tab).
Das gleiche mit dem Blick auf den Preis - die komplexen Verrechnungspreise können verschiedene Arten von Preisen laden.Preise ansehen können für jede Zeile oder für die gesamte Spalte bestimmt werden.

Spaltennummern werden manuell eingestellt.Die gleiche Spalte kann mehr als einmal angegeben werden (zum Beispiel den Namen des Herstellers zu setzen und, wie der Name einer gewöhnlichen Gruppe von Waren).

Dohle in der "Wenn leer - nehmen Sie die vorherigen" bedeutet, dass die Spalte nicht in allen Zeilen gefüllt werden kann.Zum Beispiel wird der Hersteller auf einmal bezeichnet, und dann alle Zeilen unten - über Produkte von ihm.Und nur, wenn es einen Datenblock endet eine einzige Quelle - in dieser Spalte Zellenwert erscheint wieder Produzent (der nächste Datenblock-Nomenklatur).So (mit aktiviertem daw) Programm speichert den Wert und verteilt sie an alle die leeren Zellen dieser Spalte.Wenn eine Prüfung nicht geliefert wird - ein Programm, um die Werte in jeder Zeile zu lesen und es als ungefüllte leer nehmen würde.

Dohle in der Spalte "Erstellen" können Sie erstellen, wenn sie direkt einfachen Verzeichnisse im Zusammenhang importieren.Es ist nur relevant für die Art der Link Details.
Also in diesem Beispiel zeigt, dass, wenn das Verzeichnis Element "Produzenten" kann nicht nach dem Namen (der Wert in dieser Spalte) zu finden, erstellen Sie ein neues Verzeichnis Element "Produzenten" mit diesem Namen und schon wird es als Hersteller installiert werden, für der Gegenstand.

Wann kann die Serien Buchhaltung sofort Informationen über bestimmte Serie Nomenklatur importieren erfolgen.Wenn die Serie Datenspalte ausgewählt, dann wird diese Serie gefunden / erstellt werden und wird mit dem Import dieser Nomenklatur beteiligt sein.Dann können Nomenklatur "repeat" in Linien mit unterschiedlichen "Serie".
Wenn Sie eine Reihe von Waren Rückstände importieren ist es notwendig, die Spalte "Anzahl" zu setzen und füllen Sie ihn mit "1".Jede andere Zahl als Null ist das Programm zu einem führen.
Wenn die Nummer nicht angegeben oder gleich Null ist, wird die Serie erstellt, wird aber nicht in den Registrierungsdaten auf die Waage bringen.

Für den Treffer zum 1 Block angeben wir eine Methode, um die Einzigartigkeit jeder Reihe von Daten zu bestimmen.In diesem Beispiel bestimmt die Einzigartigkeit eines jeden der Nomenklatur durch den Namen und dann - Artikel.Das heißt, wenn der Name und der Code wiederholt werden - das ist die gleiche Nomenklatur.Achten Sie darauf, beim Import der Parameter eindeutig vermeiden Duplikate zu setzen.

Daw "nur aktualisieren" zeigt an, dass neue Datenelemente werden nicht erstellt werden.Datenleitungen, die durch das Tastenfeld (Block 1) nicht einfach ignoriert werden, erkannt haben.Und wenn solche Daten (in diesem Fall wird die Nomenklatur) werden erkannt - das Programm wird ihre Daten prüfen und die Tabelle, auf Kosten der Importe zu aktualisieren.
Dohle "Überspringen bestehende" legt nahe, dass, wenn während der Import bestehender Elemente werden in Schlüsselbereichen gefunden werden, werden sie das Programm verpassen und werden nicht aufgrund der Daten aus dieser Tabelle aktualisiert werden.
Wenn beide Saatkrähen leer sind, dann wird das Programm die Daten in den vorhandenen Elemente zu aktualisieren und neue hinzufügen.
Dohle "Neue Gruppe" wird angezeigt, wenn hierarchische Verzeichnisse Importieren von Daten (zB Nomenklatur).Wenn die Gruppe eine eigene Spalte in der Datentabelle angegeben wird, dann auf Kosten der Zecke kann so eingestellt werden - eine neue Gruppe zu erstellen (wenn nicht gefunden) oder nicht.
