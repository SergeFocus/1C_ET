Deleting marked objects

Processing allows you to make the removal of the information database of objects that are marked for deletion.

On the first page it is possible to choose between deleting all objects marked for deletion mode ("Complete removal") and interactive selection of all objects of a subset (mode "Selective removal").

WYSIWYG removal

To completely remove all marked for deletion objects should select "Wipe" and click "Remove". The processing searches for all objects in the knowledge base, marked for deletion, and makes an attempt to remove them. Removal is to control the referential integrity, at the end of the operation, not all objects can be deleted, as a part of them may be in a non-removable links objects.

After completion of the erase operation occurs to a page showing the results of removal, i.e. how many objects were successfully removed and the removal of many objects were forbidden to control the referential integrity. If you have successfully removed all the objects, the only acceptable action (in addition to the display reference) remains closed. Otherwise, it is possible to go to a page that shows links to information on the basis of the objects that failed to remove, grouped in 3 hurovnevom tree on the project metadata and do not link the remote object. Selecting an object in the tree containing a reference to the remote object is not to open the window showing the value, for example, reference books will be open form of the element or group.

Selective removal

To selectively remove the objects marked for deletion, select the mode "Selective removal" and click "Next & gt; & gt;". This will return all marked for deletion objects and open the page where they will be presented in the form of 2-hurovnevogo tree with notes, grouped under the project metadata. This tree marks, select the objects you want to remove, then click button "Remove".

Further work will be selective removal as well as the work of the complete removal of the exception, which will be removed, not all items marked for deletion, but only those that were selected by the user.
