Deleting marked objects

Processing allows you to make the removal of the information base of objects marked for deletion.

On the first page it is possible to choose between deleting all objects marked for deletion (mode "Complete removal") and interactive selection of all objects of a subset ( "Selective removal" mode).

Complete removal mode

Mode should be chosen to completely remove all objects marked for deletion "Complete removal" and click "RemoveProcessing searches all objects of the knowledge base, marked for deletion, and makes an attempt to remove them.Removal takes place with the control referential integrity, not all objects at the end of the operation can be removed becausea portion of which may be referenced in non-removable objects.

After completion of the erase operation occurs to a page showing the results of removal, i.e.how many objects were successfully removed and removal of many objects were forbidden to control referential integrity.If you have successfully removed all the objects, the only acceptable action (in addition to the display reference) remains closed.

Otherwise, it is possible to go to a page that shows links to information on the basis of the objects that could not be removed, grouped in 3 hurovnevom tree metadata object and the link is not a remote object.Selecting an object in the tree that contains a reference to the remote object is not to open the window showing the value, for example, reference books will be open form of the element or group.

Selective removal mode

To selectively remove objects marked for deletion, select the mode "Selective removal" and click "Next & gt; & gt;This will find all the objects marked for deletion and open the page where they will be presented in the form of 2-hurovnevogo tree with notes, grouped according to the metadata object.This tree marks, select the objects you want to remove, then click "Remove

Next, the operation of selective removal will occur as well as the work of the full removal of the exception, that does not remove all items marked for deletion, but only those that have been selected by the user.
