Suppression d'objets marqués

Traitement vous permet d'effectuer le retrait de la base de données d'objets marqués pour suppression.

Sur la première page, il est possible de choisir entre la suppression de tous les objets marqués pour suppression (mode "retrait complet") et la sélection interactive de tous les objets d'un sous-ensemble (en mode «élimination sélective").
