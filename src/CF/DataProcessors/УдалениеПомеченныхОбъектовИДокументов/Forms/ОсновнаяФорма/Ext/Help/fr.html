Suppression d'objets marqués

Traitement vous permet de faire la suppression de la base de données d'objets qui sont marqués pour suppression.

Sur la première page, il est possible de choisir entre la suppression de tous les objets marqués pour mode de suppression ("d'élimination complète") et la sélection interactive de tous les objets de un sous-ensemble (mode «élimination sélective").

Élimination WYSIWYG

Pour supprimer complètement tous marqués pour les objets de suppression devrait choisir "Wipe" et cliquez sur "Retirer". Les recherches de traitement pour tous les objets dans la base de connaissances, marqué pour la suppression, et fait une tentative de les enlever. Enlèvement est de contrôler l'intégrité référentielle, à la fin de l'opération, tous les objets ne peuvent être supprimés, comme une partie d'entre eux peut être dans un non-amovibles objets liens.

Après l'achèvement de l'opération d'effacement se produit à une page montrant les résultats d'enlèvement, à savoir le nombre d'objets ont été supprimés avec succès et la suppression de nombreux objets ont été interdit de contrôler l'intégrité référentielle. Si vous avez réussi à retirer tous les objets, la seule action acceptable (en plus de la référence d'affichage) reste fermée. Sinon, il est possible d'accéder à une page qui montre des liens vers des informations sur la base des objets qui ont échoué à enlever, regroupés en 3 arbre hurovnevom sur les métadonnées du projet et ne lie pas l'objet distant. Sélection d'un objet dans l'arborescence contenant une référence à l'objet distant est de ne pas ouvrir la fenêtre montrant la valeur, par exemple, des livres de référence seront forme ouverte de l'élément ou de groupe.

L'élimination sélective

Pour supprimer sélectivement les objets marqués pour suppression, sélectionner le mode "élimination sélective" et cliquez sur "Suivant & gt; & Gt;". Ce sera de retour tous marqués pour les objets de suppression et d'ouvrir la page où ils seront présentés sous la forme d'un arbre 2-hurovnevogo avec des notes, regroupés sous les métadonnées du projet. Cet arbre marques, sélectionnez les objets que vous souhaitez supprimer, puis cliquez sur bouton "Retirer».

D'autres travaux seront élimination sélective ainsi que le travail de l'élimination complète de l'exception, qui sera enlevé, pas tous les éléments marqués pour suppression, mais seulement ceux qui ont été sélectionnés par l'utilisateur.
