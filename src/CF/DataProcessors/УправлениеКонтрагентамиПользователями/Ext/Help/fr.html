Système de gestion des entrepreneurs par l'utilisateur

Traitement vous permet de poser pour une relation client ou fournisseur, avec une base de données clients.

On suppose que l'utilisateur (ou les utilisateurs) avec le rôle de la «Société Client" ( "Enterprise Fournisseur") peuvent être écrits un certain nombre de clients (fournisseurs) qui sera en mesure de travailler dans le système.Chaque contrepartie sera original "mot de code" qui l'identifie et installer les pré-sélections dans les rapports et les listes.
