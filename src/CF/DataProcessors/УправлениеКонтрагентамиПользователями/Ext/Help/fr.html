Le système de contrôle contreparties utilisateurs

Traitement vous permet de poser pour la relation client ou fournisseur avec une base de données clients.

Il est supposé que l'utilisateur (ou utilisateurs) avec le rôle de la «Société Client" ("Fournisseur Société») peuvent être écrits un certain nombre de clients (fournisseurs) qui peuvent travailler dans le système. Chaque telle contrepartie est le "Le mot de code" original qui l'identifie et installer la pré-sélection des rapports et des listes.
