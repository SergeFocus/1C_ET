  • тільки для адміністратора: в розділі програмипо командіГрупове зміна реквізитів або в разі підключення в якості зовнішньої обробки - у відповідному розділі програми по командіДодаткові обробки - Групове зміна реквізитів

Зміна виділених елементів списку

Зміна значень реквізитів (а також додаткових реквізитів і відомостей) у вибраних елементів списку.

Введення нових значень реквізитів обраних елементів списку

  • Доступні для зміни поля обраних елементів списку відображаються в рядках колонкиРеквізит Можна змінити відразу кілька реквізитів.
  • Для того щоб змінити значення реквізитів (а також додаткових реквізитів і відомостей), в потрібних рядках необхідно заповнитиНове значення
  • Відкрийте рядок для введення за допомогою подвійного клацання миші.Можна, можливо:
    • ввести значення для пошуку в списку;
    • за допомогою командиПоказати все (клавіші ) Вибрати значення з відповідного списку;
    • ввести нове значення в список за допомогою командиСтворити
  • При введенні нового значення автоматично включається прапорець
  • НатиснітьЗмінити реквізити

Очистити значення полів обраних елементів списку

  • Для очищення значення реквізиту потрібно включити прапорець у відповідному рядку, а колонкуНове значення не заповнювати.
  • НатиснітьЗмінити реквізити

Результат зміни з помилками

  • Список незмінених елементів списку виводиться в окремому вікні. У списку по кожному елементу наводиться причина помилки.
  • Натисніть відкрити, Щоб переглянути елементи, які не вдалося змінити.
  • НатиснітьПовторити зміна, Щоб спробувати повторно змінити елементи списку з тими ж параметрами.
  • Для виправлення помилки, можливо, доведеться змінити значення реквізитів перерахованих елементів на інші.

Вибір раніше збережених настройок по зміні реквізитів

  • Скористайтеся командоюРаніше змінені реквізити
  • У списку оберіть потрібну настройку.У списку зберігаються настройки останніх 20 змін реквізитів.

Зміна заблокованих реквізитів

  • У списку можуть бути реквізити, за замовчуванням недоступні для зміни і які відображаються сірим кольором.
  • Ці реквізити є "ключовими" реквізитами, вони можуть вплинути на інші списки та документи програми. Тому вони захищені від випадкових змін.
  • При спробі введення нових значень програма пропонує їх розблокувати.У разі особливої ​​необхідності їх зміни, можна підтвердити свій вибір, натиснувши на кнопкуДа

Особливості зміни проведених документів

  • При зміні проведених документів виконується їх перепроведення.

  • Для перегляду вибраних для зміни елементів скористайтеся посиланнямЗмінювати все елементи

  • Після вибору необхідних реквізитів скористайтесястандартними командами для їх зміни.
