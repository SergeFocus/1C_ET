Сохранить регистр конфигурационный файл шаблона

Эта форма была установлена ​​шаблонные параметры файла конфигурации сгенерированные.

  • Имя Шаблона - Короткое название модели.Появляются в спискевыбор дизайна
  • Описание шаблона - Установить описание шаблона (более продвинутый, чем имя шаблона).Отображается в правой панеливыбор дизайна
  • Это указывает на то, что каталог должен спрашивать ли пользователь выбрать шаблон в режиме транзакции.

После нажатия кнопки,ОК Сохранение и другой файл цифровой шаблон.Сохраните файл шаблона в занятому номеру невозможно.
