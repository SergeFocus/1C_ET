Salvare i registri di file di configurazione del modello

Questa forma viene fatto impostando i parametri generati file di configurazione del modello.

  • Nome modello - Nome del modello corto.Appare nella casella di riepilogoscegliere il disegno
  • Descrizione del modello - Impostare la descrizione del modello (più avanzato rispetto al nome del modello).Viene visualizzata nel riquadro di destrascegliere il disegno
  • Indicare i modi in cui le directory dovrebbero chiedere all'utente se la transazione utilizzando un modello.

Dopo aver fatto clicBene è il salvataggio del file di modello con un altro numero.Salvare il file di modello nel numero occupato è impossibile.
