Selezionare un file di configurazione del modello

In questo modulo è possibile scegliere il file di configurazione del modello, in base al quale verrà creato il file di registri di configurazione.

Modelli file di configurazione può essere costruito (in questo caso sono contrassegnati davanti al modello di immagine) e l'utente (questa immagine è assente).

È anche possibile eliminare un modello utente.Per fare questo, selezionare squadraCancella all'elenco dei modelli menu di scelta rapida.Questo comando è disponibile solo per i modelli personalizzati.

Dopo aver fatto clicBene, Il modello verrà caricato nel file di configurazione dell'editor.
