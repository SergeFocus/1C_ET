Selezione del modello di file di configurazione

In questo modulo è possibile selezionare il modello file di configurazione in base al quale verrà creato la configurazione del registro tecnologico desiderato.

Modelli file di configurazione può essere costruito (in questo caso sono contrassegnate con una foto davanti al nome del pattern) e l'utente (questa immagine è assente).

E 'anche possibile eliminare un modello personalizzato.Per fare questo, selezionare il comandoCancellare nella lista modelli di menu contestuale.Questo comando è disponibile solo per i modelli personalizzati.

Dopo aver fatto clicBene, Il modello selezionato verrà caricato nel file di configurazione dell'editor.
