Saving Technology modèle de fichier de configuration de magasins

Dans ce formulaire, vous installez les options générées modèle de fichier de configuration.

  • Nom modèle - Un nom court pour le modèle.Affiche la liste de la fenêtrela sélection d'un modèle
  • Description modèle - Compte tenu de la description du modèle (plus avancé que le nom du modèle).Affichée sur le côté droit de la fenêtrela sélection d'un modèle
  • Il spécifie le chemin d'accès au répertoire devrait inciter l'utilisateur une opération de sélection de modèle.

Après avoir cliqué surBien fichier modèle sauve avec le numéro suivant.Enregistrez le fichier de modèle ne peut pas être utilisé sous le numéro.
