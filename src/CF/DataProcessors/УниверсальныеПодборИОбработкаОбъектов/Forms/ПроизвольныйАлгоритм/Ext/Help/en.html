"Arbitrary" algorithm processing.

Executes an arbitrary algorithm built-in language.

To execute arbitrary algorithm, in the edit box to enter text on the integrated language and click"Run"(keyDefine a custom algorithm will be executed sequentially for each sample of the object included in the processing.Appeal to the object from the embedded language should be carried out by means of the variable "object".
