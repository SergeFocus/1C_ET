L'elaborazione di "algoritmo arbitrario".

Esegue algoritmo arbitrario nello script.

Per eseguire l'algoritmo arbitrario deve essere nella casella di modifica per inserire il testo nello script e fare clic su "Correre" (chiave F9). Definire un algoritmo personalizzato viene eseguito in sequenza per ogni oggetto campione inclusi nel processo. Titolo per l'oggetto dal linguaggio incorporato dovrebbe essere attuata attraverso una variabile "oggetto".

Esempio di scrittura di un algoritmo arbitrario nello script:

Obekt.Naimenovanie = "nome nuovo";

Obekt.Zapisat ();
