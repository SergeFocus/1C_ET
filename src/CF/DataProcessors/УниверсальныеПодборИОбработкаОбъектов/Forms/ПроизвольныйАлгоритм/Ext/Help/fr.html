Traitement "algorithme arbitraire."

Exécute algorithme arbitraire dans le script.

Pour exécuter l'algorithme arbitraire doit être dans la zone d'édition pour saisir du texte dans le script, puis cliquez sur "Courir" (clé F9). Définir un algorithme personnalisé est exécutée séquentiellement pour chaque objet de l'échantillon inclus dans le processus. Titre de l'objet à partir de la langue incorporé doit être mis en œuvre par une variable «objet».

Exemple d'écrire un algorithme arbitraire dans le script:

Obekt.Naimenovanie = "nouveau nom";

Obekt.Zapisat ();
