"Ixtiyari alqoritm." Emal

Script ixtiyari alqoritm həyata keçirir.

Ixtiyari alqoritm icra script mətn daxil edin və basın edit qutusuna olmalıdır "Run" (Əsas F9). Xüsusi alqoritm prosesinə daxil hər nümunə obyekt üçün ardıcıl icra edilir tanımlayın. Əlaqədar dil obyekt Title dəyişən vasitəsilə həyata keçirilməlidir "obyekt".

Script bir ixtiyari alqoritm yazı nümunəsi:

Obekt.Naimenovanie = "yeni adı";

Obekt.Zapisat ();
