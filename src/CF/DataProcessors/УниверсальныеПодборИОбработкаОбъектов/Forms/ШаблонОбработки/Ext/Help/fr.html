Traitement "de modèle de traitement".

Ce formulaire peut être utilisé comme modèle lors de la création de nouveaux traitements.Pour cela, dans le mode de configurateur pour copier la forme de "ShablonObrabotki" et changer le nom et les formes synonymes.Ensuite, le traitement est affichée dans un arbre de traitements disponibles. Dans l'avenir, il est nécessaire de modifier le traitement de la forme et le module du formulaire.
