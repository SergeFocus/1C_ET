Обработка "деталей установки."

Для получения более подробной информации о выполнении установки.

Для получения дополнительной информации по установке должны быть в форме, подробно колонке список коробки "Значение" Выберите новое значение соответствующей информации, в том числе "," Подробности терапии изменилось флагов состояния в строке, а затем нажмите "Выполнить" (Ключ F9). Детали установки будут последовательно обработки каждого образца, чтобы включить объект выполняется.
