Обработка "добавить детали."

Дополнительные сведения о выполнении установки.

Подробный перечень информации в списке полей, чтобы выполнить необходимые сведения об установке"стоимость" Выберите новое значение соответствующие детали, включая подробные сведения о процессе изменения состояния государственного флага в «вверх» линии, нажмите кнопку"бег"(Самое важное). УСТАНАВЛИВАЕТЕ детали будут сделаны объект, включенный в обработку каждого образца в очереди.
