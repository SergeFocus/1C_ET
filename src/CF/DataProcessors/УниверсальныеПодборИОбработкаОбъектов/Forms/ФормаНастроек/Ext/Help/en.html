Form settings for selection and display of objects

The form is intended to edit the settings for selection and display of objects, such as:

  • Display details - details columns displayed in the table of found objects.
  • Tackling - a set of options for selecting subjects.
  • Sort - a sequence of objects in the output table.

To set the order of output fields, select the field and added using the corresponding buttons to put them in the appropriate table field.

Setting selection is carried out on the "selection" from the command panel.
