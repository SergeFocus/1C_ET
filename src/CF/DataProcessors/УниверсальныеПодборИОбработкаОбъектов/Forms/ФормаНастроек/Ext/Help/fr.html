Paramètres de formulaire pour la sélection et l'affichage des objets

Le formulaire est destiné à modifier les paramètres pour la sélection et l'affichage des objets, tels que:

  • Afficher les détails - de détails colonnes affichées dans le tableau d'objets trouvés.
  • Tackling - un ensemble d'options de sélection des sujets.
  • Tri - une séquence d'objets dans la table de sortie.

Pour définir l'ordre des champs de sortie, sélectionnez le champ et a ajouté en utilisant les touches correspondantes pour les mettre dans le champ de la table appropriée.

Réglage de sélection est effectuée sur la "sélection" à partir du panneau de commande.
