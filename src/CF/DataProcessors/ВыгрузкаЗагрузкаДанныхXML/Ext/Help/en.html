Processing allows for both full and partial unloading of the data base information to a file.Setting composition data is unloaded in the dialogue by setting flags in the wood column that displays the metadata objects for which data can be downloaded. Additional column flags "if necessary", establishes the need for unloading objects of this type "link".That is, if the box is checked only in the "if necessary", the data for this object will not be discharged in full, and only to the extent that is necessary to maintain referential integrity in the information base, which will load the file upload.
