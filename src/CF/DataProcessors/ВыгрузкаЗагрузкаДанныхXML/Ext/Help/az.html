Processing "Boşaltma və yükləmə XML data"

Əməliyyat rejimi

Yetishan (istifadəçi tərəfindən bu məlumatların yaradılması bir faylı) və yük (bu mövcud data file yaradılan eponymous oxu rejimi, və qeyd boşaltmaq) Processing iki əməliyyat rejimi həyata keçirir. Mode sahəsində seçerek rejimi qurulması.

Bu və ya digər rejimi (Run düyməsini basın) başlamazdan əvvəl, fayl upload adını daxil etmək lazımdır və ya "Ad Fayl" in özünüz daxil və ya düyməsini basın sahəsində və standart fayl seçimi informasiya seçin istifadə edərək.

Boot rejimində, bu download sürəti təsir göstərə bilər girişdə qeydiyyat istifadə nəticələrini dəyişdirmək mümkündür.
Bayraq "data download zamanı istifadə nəticələri redaktə etmək imkanı Enable" üçün müəyyən edilir və məlumatların yüklənməsi istifadə nəticələrinin dərslik nəzarət üçün istifadə olunur zaman Buttons "nəticələr Disable" və "nəticələr daxil" mövcuddur.

Tətbiq emal şərait

Processing yalnız məlumat boşaldılır olan məlumat bazası, və data dolu olan bir homogen (eyni konfiqurasiya data dəyişə bilər) var hallarda istifadə edilə bilər.Ya bütün obyektlərin tərkibi və növü məlumat və masa hissələri, xassələri "aparıcı" obyekt metadata və s demək olar ki, tamamilə eyni xəstəxanadanBu məhdudiyyətlər ilə əlaqədar emal əsasən homogen IB arasında mübadilə üçün nəzərdə tutulmuşdur ki, qeyd etmək lazımdır.

Mübadilə planına header hissəsində boşaltma zaman yaradılan format faylı fərqli bir fayl format bərpa edin.Fayl formatları bu hissəsində planları mübadiləsi axıdılması eynidir eyni XML-serialization mexanizmi istifadə edərək müalicə (s elementləri təlimatlar, qeydiyyat və uçotunun dəstləri) data download.

Axıdılması tərkibinin müəyyən edilməsi

Processing bir fayl məlumat bazası informasiya, həm də tam və qismən boşaltma üçün imkan verir.Setting tərkibi data data yükləyə bilərsiniz olan metadata obyektlərin göstərir ağac sütun bayraqları qəbulu ilə dialoq boşaldılır. Əlavə sütun bayraqlar "lazım olsa", bu növü "link" obyektlərin boşaldılması üçün ehtiyac yaradır.Box yalnız yoxlanılır ki, əgər, bu obyekt üçün data "zəruri hallarda" tam azad və olmayacaq yalnız fayl yük informasiya bazasında sened bütövlüyünü qorumaq lazımdır dərəcədə üçün bərpa edin.

Açmaq zaman forması emal informasiya bazasının parçası xəstəxanadan sened bütövlüyünü təmin bütün obyektlərin bağlantısını yüklemek üçün bayraq edir.

Düyməsinə tıklayarak, emal təhlil data tam axıdılması bir ibrət yüklü müəssisələrində aparıla bilər istinadları "körpüsünə bərpa data müəyyən" və avtomatik yüklemek üçün ehtiyac göstərir bayraqları sütun doldurur link.Obyekt artıq tam axıdılması qutusu bir sıra varsa, bu halda, axıdılması link müəyyən deyil edin.

Mümkün applications

Bu müalicə istifadə, məsələn, məlumat bazaları arasında məlumat mübadiləsinin tam və ya qismən backup data, eləcə də məlumat bazalarının probleminin azaldılması bir köməkçi alət yaratmaq bilər.
