Processing "Upload və download XML"

Modu

(Eyni rejim tərəfindən yaradılmış fayl oxumaq boşaltmaq və mövcud məlumat yazmaq) axıdılması (fayl yaradılması boşaltma istifadəçi müəyyən data) və yük Qenerasiya 2 əməliyyat rejimi satır. Installation rejimi seçilməsi sağ rejimində edilir.

"Fayl adı" bu daxil və ya bu sahəyə və standart fayl seçimi informasiya seçmək üçün düyməsini istifadə əl fayl upload adını daxil olmalıdır (Run tıklayarak) xüsusi rejimi başlamaq və ya əvvəl.

Boot rejimində, bu download sürəti təsir göstərə bilər girişdə qeydiyyat istifadə nəticələrini dəyişdirmək mümkündür.
Buttons "nəticələr Disable" və bayraq və məlumatların loading zaman nəticələr istifadə manual nəzarət rejimi üçün istifadə olunur "data loading zaman nəticələr istifadə redaktə imkanı Enable" zaman mövcud "qədər dönün".

Tətbiq emal şərtləri

Bu məhdudiyyətlər ilə bağlı, əsasən heterogen IB arasında mübadilə üçün nəzərdə idarə ki, qeyd etmək lazımdır.

File format plan mübadiləsi, adı iştirak boşaltma zamanı yaradılmış fayl formatı boşaldılması fərqlidir.Məlumat boşaltmaq üçün boşaltma və fayl formatları bu hissəsində planları bölüşmək eynidir eyni XML-serialization mexanizmi istifadə edərək, emal (arayış elementləri, rekord qeydə, və s. edir).

Axıdılması Definition

Qenerasiya məlumat bazası fayl tam və qismən axıdılması data üçün imkan verir.Daşıyanlar data Setting obyektləri metadata, qəbul edilə bilər verilerini gösterir sütun ağac bayraqları qəbulu bir dialoq keçirilir. Əlavə sütun bayraqlar "lazım olsa", bu növü "link" obyektləri boşaltmaq ehtiyacı yaradır.Bayraq lakin yalnız bir fayl upload download məlumat bazasında sened bütövlüyünü qorumaq lazımdır dərəcədə tam bu mövzu vyvantazhuvatymutsya haqqında, 'zəruri' in yalnız data müəyyən edilir ki, edir.

Bir form emal xüsusiyyət link müəyyən açtığınızda fraqmenti məlumat bazası boşaldılır links bütövlüyünü zəmanət bütün obyektlərin boşaltmaq.

Ehtiyac link yerinə yndytsyruyut axıdılması tam dəsti xüsusiyyət emal data links müəssisələrində olan bilər analiz "Əlavə data link müəyyən" avtomatik olaraq doldurur sütun bayraqları tıklayarak.Obyekt artıq link boşaltma qutusu müəyyən olmayan halda bir onay qutusu tam axıdılması, varsa.

Mümkün applications

Mümkün bu müalicə istifadə, misal üçün, məlumat bazalarının arasında məlumat mübadiləsi məlumatların tam və ya qismən backup, eləcə də narahat verilənlər bazası bərpa yardımçı alət yaratmaq.
