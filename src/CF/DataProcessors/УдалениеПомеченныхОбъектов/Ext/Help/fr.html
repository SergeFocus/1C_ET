Suppression marqué objets disponibles pour un utilisateur avec des droits administratifs. L'élimination est effectuée pour contrôler l'intégrité de la base de l'information. Cela signifie que si l'un ou l'autre objet du système (document, l'entrée de répertoire, etc.) référencé par d'autres objets puis supprimez l'objet échouera. Toutefois, il convient de rappeler que la suppression d'objets - une opération irréversible.

Enlèvement des objets précédemment marqués peut se faire de deux manières:

  • "Supprimer automatiquement tous les objets »- Une tentative est faite pour supprimer automatiquement tous les objets qui sont marqués pour suppression.
  • "L'élimination sélective des objets"- Ce mode vous permet de sélectionner les fichiers à supprimer.

À la fin de la suppression d'une liste d'objets dont l'enlèvement ne tient pas pour l'intégrité référentielle de la base de l'information. Dans cette liste objets sont regroupés par type et pour chaque objet objets qui ne peuvent pas être supprimés affichés.

Vous pouvez modifier les objets contre la suppression d'un objet particulier marqué pour suppression, puis répéter la procédure de retrait.
