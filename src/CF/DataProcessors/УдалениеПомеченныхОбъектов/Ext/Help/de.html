Löschen markierte Objekte für einen Benutzer mit administrativen Rechten. Entfernen wird durchgeführt, um die Integrität des Informationsbasis zu steuern. Das heißt, wenn das eine oder andere System-Objekt (Dokument der Verzeichniseintrag, etc.) von anderen Objekten referenziert löschen Sie das Objekt, wird fehlschlagen. Es sollte jedoch nicht vergessen, dass die Entfernung von Objekten - eine irreversible Operation.

Entfernen des zuvor markierten Objekte kann auf zwei Arten erfolgen:

  • "Alle Objekte werden automatisch gelöscht "- Es wird versucht, alle Objekte, die zum Löschen markiert sind, automatisch gelöscht werden.
  • "Selektives Entfernen von Objekten"- In diesem Modus können Sie die Dateien zum Löschen wählen.

Nach Abschluss der Entfernung von einer Liste von Objekten, deren Entfernung nicht für die referentielle Integrität der Informationsbasis zu halten. In dieser Liste Objekte werden nach Typ gruppiert und für jedes Objekt angezeigten Objekte, die nicht gelöscht werden kann.

Sie können die Objekte gegen die Entfernung von einem bestimmten Objekt zum Löschen markiert bearbeiten, und wiederholen Sie das Verfahren zur Entfernung.
