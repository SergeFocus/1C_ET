Entfernen von ausgewählten Objekten zur Verfügung, um Benutzer mit Administratorrechten. Entfernung erfolgt, um die Integrität des Informationsbasis zu steuern. Das bedeutet, dass, wenn der eine oder andere System-Objekt (Dokument, Verzeichnis-Eintrag, etc.) werden von anderen Objekten, die das Löschen fehlschlägt Objekt referenziert. Dennoch sollte daran erinnert werden, dass die Entfernung von Objekten - ein irreversibler Vorgang.

Entfernen von zuvor markierten Objekten kann auf zwei Arten erfolgen:

  • "Alle Objekte werden automatisch gelöscht "- Es wird versucht, alle Objekte, die zum Löschen markiert sind, automatisch gelöscht werden.
  • "Selektives Entfernen von Objekten"- In diesem Modus können Sie die zu löschenden Dateien auswählen.

Nachdem die Lösch Listen von Objekten, deren Entfernung nicht für die referenzielle Integrität der Informationsbasis zu halten. In dieser Liste Objekte werden nach Typ gruppiert und für jedes Objekt die Objekte, deren Entfernung zu verhindern gefunden.

Sie können die Objekte gegen die Entfernung eines bestimmten Objekts zum Löschen markiert bearbeiten, und wiederholen Sie dann die Deinstallation.
