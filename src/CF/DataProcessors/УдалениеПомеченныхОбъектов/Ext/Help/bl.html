Выдаленне выбраных аб'ектаў, даступных для карыстальніка з правамі адміністратара. Выдаленне робіцца для кантролю цэласнасці інфармацыйнай базы. Гэта азначае, што калі адзін ці іншы аб'ект сістэмы (дакумент, запіс каталога і г.д.) спасылаюцца іншыя аб'екты, якія пярэчаць не ўдаецца выдаліць файл. Тым не менш, варта памятаць, што выдаленне аб'ектаў - дарогі назад няма.

Выдаленне раней пазначаныя аб'екты можна зрабіць двума спосабамі:

  • "Аўтаматычна выдаляць ўсе аб'екты "- Спроба аўтаматычна выдаліць усе аб'екты, якія адзначаны для выдалення.
  • "Выбарачнае выдаленне аб'ектаў»- Гэты рэжым дазваляе выбраць файлы для выдалення.

Пасля выдалення спісы аб'ектаў, выдаленне не выконваецца для спасылачныя цэласнасці інфармацыйнай базы. У гэтым спісе аб'екты згрупаваныя па тыпу і па кожнаму аб'екту знайшлі тыя аб'екты, якія перашкаджаюць яго выдаленне.

Вы можаце рэдагаваць аб'екты супраць зняцця канкрэтнага аб'екта адзначаныя для выдалення, а затым паўтарыце працэдуру выдалення.
