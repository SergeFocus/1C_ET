Выдаленне вылучаных аб'ектаў, даступных карыстачу з правамі адміністратара. Выдаленне выконваецца кантраляваць цэласнасць інфармацыйнай базы. Гэта азначае, што, калі адзін ці іншы аб'ект сістэмы (дакумент, запіс каталога, і г.д.) спасылацца іншыя аб'екты затым выдаліць аб'ект не будзе выканана. Тым не менш, варта памятаць, што выдаленне аб'ектаў - незваротнай аперацыі.

Выдаленне раней пазначаных аб'ектаў можна зрабіць двума спосабамі:

  • "Аўтаматычна выдаліць усе аб'екты "- Спроба аўтаматычна выдаліць усе аб'екты, якія адзначаны для выдалення.
  • "Выбарачнае выдаленне аб'ектаў"- Гэты рэжым дазваляе выбраць файлы для выдалення.

Пасля завяршэння выдалення спісу аб'ектаў, ухіленне якіх не мае для спасылачныя цэласнасці інфармацыйнай базы. У гэтым спісе аб'ектаў згрупаваныя па тыпу і для кожнага аб'екта паказу аб'ектаў, якія не могуць быць выдаленыя.

Вы можаце рэдагаваць аб'екты на выдаленні канкрэтнага аб'екта адзначаныя для выдалення, а затым паўтарыце працэдуру выдалення.
