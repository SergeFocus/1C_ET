Eingabeverarbeitungstabellen

Es ermöglicht Ihnen, zum direkten Eingabe von Daten füllen (geben) die Tabelle.

Kopieren Sie beispielsweise - fügen Sie Daten von externen Tabellendatei.

Geben Sie an, welches Feld immer gefüllt ist (darauf wird das Programm zu stoppen, wenn erlaubt größere Anzahl).

Die erste Zeile der Tabelle wird als Titel verwendet und nicht geladen.
