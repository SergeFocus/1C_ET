Система управления доступом пользователей

Система призвана предоставить руководителям в интерактивном режиме определить для пользователей их уровни доступа к объектам, выполнить данную настройку более тонко с учетом условий ведения бизнеса.

По умолчанию у пользователя установлены все права.

Предусмотрены следующие уровни доступа:

  • Полный доступ (полный безусловный доступ)
  • Без условия - чтение (полный доступ, но при условии. При невыполнения условия - просмотр данных)
  • Доступ при условии (полный доступ, но при условии. При невыполнении условия - отказ)
  • Только чтение (только просмотр данных)
  • Чтение при условии (только просмотр данных и только при условии. При невыполнении условия - отказ)
  • Нет доступа (отказ)

Условие заполняется в отдельном поле на языке 1С. Например: Объект.Наименование = "Пример"

Другой пример: Объект.Ответственный = ПараметрыСеанса.ТекущийПользователь

Наличие в интерфейсе пользователя тех или иных объектов определяется "ролью" пользователя. Но доступность этих элементов можно закрепить за счет этой системы.
