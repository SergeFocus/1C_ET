Выгрузка и загрузка данных. Обмен с аналогичной программой

В данной обработке предусмотрены способы обмена данными с базами данных аналогичной конфигурации:

Существует четыре способа обмена данными за период:

1. Выгрузка - Загрузка данных о реализации. Данные о реализации это данные о движениях товара на отдельном складе или в отдельной торговой точке с самостоятельным учетом. Они включают в себя только данные по расходам, возвратам и списанию товаров за период. Деньги, состояние склада и взаиморасчетов не выгружаются. Опционально можно скрыть и цену реализованного товара. Этот вид обмена нужен для получения данных от реализатора с независимым учетом (доходом). Такие данные нужны, чтобы отслеживать остатки товара, переданного ему на реализацию. При загрузке такого пакета формируются документы типа расход для реализации и списания товара, поступления для случаев возврата товара. При загрузке можно свернуть все движения за период до одного документа, если нет необходимости отслеживать учет по контрагентам. Предусмотрены другие настройки процесса загрузки: "Параметры новых документов", "Параметры номенклатуры", "Параметры нового клиента".

2. Выгрузка - Загрузка данных руководителю. Это все данные базы данных. Такой обмен необходим в тех случаях, когда руководитель наблюдает за учетом торговой точки - загружает все данные и дополняет учет своими данными (например ценами поступления) для расчета действительных показателей прибыли и рентабельности. При загрузке такого пакета в системе появятся копии всех документов за период и элементы справочников, которые в них упоминаются. Если документы с таким номером уже есть, - они будут пропущены. Программа при загрузке прверяет документы по номеру/дате, а элементы справочников по наименованию. Номенклатуру так же по основному штрих-коду и артикулу. Контрагентов по ИНН/ОКПО. Рекомендуется не менять данные в карточках элементов справочников в программе руководителя, а только дополнять.

3. Выгрузка - Загрузка остатков товаров, цен, остатков денег, показателей взаиморасчетов, - необходимо для одноразового переноса данных в другую копию программы, когда нет возможности или необходимости перенести весь предыдущий учет, а достаточно только данных на определенную дату. При загрузке такого пакета формируется документ корректировка.

4. Выгрузка - Загрузка настроек программы, - для обмена всеми настройками с другой копией программы.

Отборы данных по Складам, Номенклатурным Группам, Производителям и Регионам накладывают фильтр при выгрузке и загрузке данных.

Помимо данной обработки, существует подсистема импорт-экспорта данных через файл, для параметрируемого обмена с другими базами данных.

Внимание: обмен работает только для аналогичной версии конфигурации!
