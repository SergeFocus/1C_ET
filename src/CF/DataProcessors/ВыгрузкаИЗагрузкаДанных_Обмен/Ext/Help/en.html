Upload and download data. Exchange with a similar program

In this treatment, methods for exchanging data with a database of similar configuration:

There are four ways to exchange data for the period:

1. Unloading - Loading data on implementation. Data on the implementation of this movement data Item a separate stock or in a single outlet with a separate account. They include only expenditure data, refunds and cancellation of goods during the period. Money, the state warehouse and settlements are not preempted. Optionally, you can hide the price of the goods sold. This type of exchange is needed to collect data from an independent marketer of account (income). Such data are needed to track the remnants of the goods transferred to him to sell. When you load such a package formed type documents consumption for implementation and cancellation of the goods, receipts cases for return of goods. When you download, you can turn all the movements for the period up to one document, if there is no need to keep track of accounting counterparties. There are other boot configuration: "Parameters of new documents," "nomenclature Options", "Parameters of the new client."

2. Unloading - Loading data manager. This is all the database data. Such an exchange is necessary in cases where the head is watching considering outlet - loads all the data and complements its accounting data (eg prices of receipt) to calculate the actual earnings and profitability. When you load such a package in the system will copy all the documents for the period elements and guides, which are mentioned. If documents with the number already there - they will be ignored. Program at boot checked up documentation by number / date and reference elements by name. Nomenclature also in the main barcode and the article. Counterparties by INN / of Enterprise. It is recommended not to change the data elements in the reference cards in the program manager, but only supplement.

3. Unloading - Loading residues goods, prices, money balances, indicators of mutual- Need for a single data transfer to another copy of the program, there is no possibility or need to move all the previous records, and only enough data for a specific date. When you load such a package is formed document adjustment.

4. Upload - Download the program settings- For all exchange settings with another copy of the program.

Selections data Warehouse,Item groups,Manufacturers and Regions filter applied for the unloading and loading data.

In addition to this processing, there is subsystem imports and exports data through a file for parameterized exchange with other databases.

Attention: sharing only works for the same version of the configuration!
