검색 값을 대체

범용 처리 "찾기 및 교체 값은"1C의 다른 정보 기반 개체의 참조 값의 검색 및 교체를위한 것입니다 : 기업입니다.

동일한 상대방이 에러 기준을 수축하는 과정에서 두 번 입력 값이 여러 문헌에 사용 된 경우, 이러한 동작에 대한 필요성은 예를 들어, 발생할 수있다.이있는 요소가 사용되는 이러한 경우에, 디렉터리의 요소는 제거되기 전에 하나 (있는 경우 및 기타 항목) 모든 문서를 수정한다.이러한 처리는 가능한 자동적 절차를 수행 할 수있다.

주의!!!
치료는 보편적 방식 정보 데이터베이스 객체의 값을 대체한다.이 계정에 구성의 특정 기능을하지 않습니다.처리를 사용하여 값을 수동으로 조정에 대한 구성을 지원하는 개체의 논리적 관계 값을 방해 할 수 있습니다.

그것은 정보 기반의 백업 복사본을 만들기 위해 치료를 사용하기 전에 권장합니다.치료에 사용 후 변경이 제대로 이루어 왔으며 가치의 교환을 수반 데이터 객체의 논리에 장해가 발생하지 않았 음을 확인.필요한 경우 수동으로 값을 수정 치료의 값에 따라 데이터를 변경해야합니다.

"대체 객체"쌍의 값 목록을 채울 수 있어야 교체하십시오. "교체하는 무엇"의 값으로 당신은 잘못 입력되어있는 디렉토리 항목을 선택 할 수 있으며, "무엇 교체"의 값으로 - 정말이 요소는 문서, 디렉토리와 다른 사이트에서 사용 할 수 있습니다.

두 번 클릭하여 "대체 개체"목록에서 또는를 눌러 키 입력
사용자는 커서 위치의 값을 열 수있다.

다음 버튼 "검색 링크"를 누릅니다.이 프로그램은 아래에 선택된 값을 사용 하였다 오브젝트 (문서, 매뉴얼 등)를 검색합니다 "무엇 교체를."검색 시간이 오래 걸릴 수 있습니다.

참고 문헌 목록은 개별 개체에 대한 값의 현재 쌍의 교체를 취소 찾을 수 있습니다.이렇게 목록에 해당 상자를 해제하십시오.

값은 클릭해야 교체하려면 "실​​행 교체 값을."프로그램 폐지되지 않은 여분하는 시설들에 값을 대체하려고 시도 할 것이다.
