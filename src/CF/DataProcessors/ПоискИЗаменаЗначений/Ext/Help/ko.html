검색 값을 대체

범용 처리 "찾아 값을 교체"를 찾아 다른 사이트 정보베이스 (1C)에서 참조 값을 대체하기 위해 설계되었습니다 : 기업.

작업 디렉토리 계약자 실수 동일한 상대방이 회 입력 한 경우와 같은 동작을 수행하기위한 필요성은, 예를 들면, 일 수 있고, 두 값이 서로 다른 문서들에 사용된다.이 경우, 전 (있는 경우 및 기타 항목) 가이드의 한 요소는 항목을 사용하는 모든 문서를 수정해야 제거한다.이 치료는 자동으로이 절차를 수행 할 수 있습니다.

주의!!!
처리는 대체 시설 정보베이스 보편적 인 방법으로 값을 사용합니다.이 계정에 특정 구성의 특수성을 고려하지 않습니다.수동 조정 값으로 구성을 지원하는 개체의 논리적 상관 값을 깰 수있는 핸들을 사용하여.

사용하기 전에 권장 치료 정보 기반을 다시 확인합니다.사용 후 처리 변경이 제대로 만들어지고 대체 값을 행할 데이터 객체의 논리에 장해가 발생하지 않았다는 것을 보장한다.필요한 경우 수동으로 해당 값이 변경된 처리 값에 따라 데이터를 변경해야합니다.

대체 값 쌍의 목록을 완료되어야한다 수행하려면 "기능을 대체됩니다." 으로 당신이 잘못 도입 된 요소 가이드를 선택할 수 있습니다뿐만 아니라 "무엇을 대체 할"을 "는 대체"하는 - 정말 문서, 디렉토리와 다른 사이트에서 사용되어야하는 요소.

두 번 클릭하거나 Enter 키를 눌러 "대체 개체"목록에서
당신은 커서에서 값을 열 수 있습니다.

그럼 당신은 클릭 '링크 찾기 "를해야합니다.이 프로그램은 아래에 선택된 값을 사용 하였다 오브젝트 (문서, 매뉴얼 등)를 검색합니다 "무엇 교체를."검색 시간이 오래 걸릴 수 있습니다.

발견 링크 목록에서 특정 개체의 현재 대체 값 쌍을 취소 할 수 있습니다.이렇게 목록에 해당 상자를 해제하십시오.

"대체 값"을 클릭합니다 대체 값을 수행합니다.이 프로그램은 취소 교체하지 마십시오 객체의 대체 값을 수행한다.
