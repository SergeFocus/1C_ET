Поиск и замена значений

Обработка общего назначения "Найти и заменить значение" используется для поиска по сайту для опорного значения различных репозиториев 1С и замены: Предприятие.

Такая операция требует может произойти, например, если вы неправильно ввести те же противники дважды в течение ручных коллегами работы, и эти два значения различных файлов были использованы. В этом случае, если вы удалите элемент каталога, которые должны быть исправлены все документы (и другие предметы, если таковые имеются), в котором компонента. Этот процесс делает возможным автоматически выполнять такой процесс.

ВНИМАНИЕ!
Лечение заменить общий способ, значение соответствующих объектов в базе знаний. Это не принимать во внимание специфические особенности строения. Значение для использования в лечении логических отношений может подорвать поддержку ручной регулировки величины объектов конфигурации. Перед использованием рекомендуется лечение, сделать резервную копию репозитория. Использование, пост-обработка, гарантировать, что изменения были выполнены правильно, это не вызвало бунты логика заменяется значением объекта данных. При необходимости, вы должны вручную изменить значение зависит от значения модифицированного обработки данных.

Чтобы заменить значения должны быть завершены "замена объекта" список. Значение "заменить" вы можете ввести правильные записи каталогов и в качестве значения "в Заменить" - элементы, которые действительно должны быть использованы в файлах, директориях и других сайтов.

В "альтернатив", дважды щелкнув по списку или нажмите Enter
Вы можете открыть значение положения курсора.

Затем нажмите кнопку ", чтобы найти ссылку." Программа будет искать с помощью выберите объекты стоимостью (файлы, справочники и т.д.), "Что это заменить." Поиск может занять много времени.

Список ссылок можно найти, чтобы заменить текущий, которые будут использоваться, чтобы отменить значение каждого объекта. Чтобы сделать это, выключите соответствующие флажки в списке.

Чтобы заменить значение, нажмите кнопку "Выполнить восстановительной стоимости." Программа будет пытаться заменить значение этих объектов и заменить его не будет отменено.
