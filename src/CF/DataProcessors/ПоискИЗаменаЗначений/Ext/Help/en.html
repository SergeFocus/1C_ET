Search and replace values

Universal processing "Find and Replace values" is intended for search and replacement of the reference values ​​in the different sites information base 1C: Enterprise.

The need for such an operation may occur, for example, if during operation in the manual counterparts mistakenly same counterparty entered twice, and both values ​​have been used in various documents. In this case, before you remove one element of the directory should be corrected all the documents (and other items, if any) in which this element is used. This treatment makes it possible to perform such a procedure automatically.

ATTENTION!!!
Treatment replaces the values ​​of the objects in the knowledge base in a universal manner. It does not take into account the specific features of the configuration. Using a treatment can disrupt logical relationship values ​​of objects that support the configuration for manual adjustment of values. Recommended treatment before use to make a backup copy of the information base. After use, processing, make sure that the changes have been made correctly and did not cause disturbances in the logic of the data objects that were replaced values. If necessary, you must manually change the data values ​​depend on the values ​​of the modified treatment.

To replace should complete list of pairs of values ​​of "substitute objects". The value "that the replacement" you can select the directory entry that is entered incorrectly, and as the value of "On that replace" - the element that really should be used in documents, directories and other sites.

In the list of "substitute objects" by double-clicking or by pressing the Enter key
You can open the value at the cursor position.

Then press the button "Find links." The program will search for objects (documents, manuals, etc.) that used the value selected under "What to replace." The search can take a long time.

The list of references can be found to cancel the replacement of the current pair of values ​​for individual objects. To do this, turn off the appropriate boxes in the list.

To replace the values, press the "Run replacement values." The program will attempt to replace the values ​​in those facilities, the replacement of which is not canceled.
