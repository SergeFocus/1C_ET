Search and replace values

Universal processing "Find and Replace values" is intended for search and replace reference values ​​for different objects infobase 1C: Enterprise.

The need for such an operation may occur, for example, if during operation in the manual counterparts mistakenly same counterparty introduced twice, and both values ​​are used in different documents. In this case, before you remove one of the elements of the directory should fix all the documents (and other objects, if they exist) in which this element. This treatment makes it possible to perform this procedure automatically.

WARNING!
Processing replaces the values ​​in the knowledge base objects in a universal manner. Without taking into account the specific features of the configuration. Using a treatment can disrupt the logical relationship of object values ​​that are supported by the case of manual adjustment values. Recommended before using data to make a backup copy of the information base. After using the treatment, make sure that the changes have been made correctly and not cause disturbances in the logical data of those objects which made replacement values. If necessary, you should manually change the data, the values ​​of which depend on the values ​​of the modified treatment.

To replace should complete list of pairs of values ​​"substitute objects". The value of the "What to replace" you can choose the directory entry that is entered incorrectly, and as the value of "What to replace" - an element that really should be used in documents, references and other objects.

In the list of "substitute objects" by double-clicking or pressing Enter
You can open the value at the cursor position.

Next, press the button "Search button". The program will search for objects (documents, manuals, etc.) in which we used the values ​​selected in the column "What to replace." The search may take a long time.

In the list of options, you can cancel the current replacement value pairs for the individual objects. To do this, turn off the appropriate boxes in the list.

To replace values ​​should click "Run replacement values." The program will attempt to replace the values ​​in those facilities, replacement of which has not been abolished.
