Пошук будь-яких даних в інформаційній базі.

Приклад пошуку Результат
«Продаж 00016422» Знайде документ "продаж товару" з номером "00016422"
«Оплата 120000» Знайде документи "оплата" на суми 120000 і 120000,50
«Іванов Акація» Знайде всі документи, виписані Івановим для контрагента ТОВ «Акація»
«Продаж Акація» Знайде всі документи «Продаж товару» по контрагенту ТОВ «Акація»
«Акація 05.07» Знайде всі документи по контрагенту ТОВ «Акація» за 05 липня 2008 року
«Кросівки» Знайде всі згадки товару типу «кросівки» по порядку:
• Кросівки білі (Товар)
• Кросівки чорні (Товар)
• Прихід товару № 00086822
• Витрата товару № 00016422

Поради:

  • Якщо знайшлося багато даних, то уточніть пошуковий запит, додавши до нього будь-який уточнення (дату, суму, назву контрагента)
  • Якщо знайшлося мало даних, то спростите запит, виключивши з нього якесь слово
  • Для точного пошуку використовуйте наступні ключові слова:
  • Лапки - для пошуку точної фрази, наприклад: «" міні пилосос "»
  • АБО - для пошуку одного з двох слів, наприклад «Акація АБО Стройпромвсе» - знайде всі згадки контрагентів ТОВ "Акація" та ЗАТ "Стройпромвсе".
  • Дужки - для угруповання виразів, наприклад пошук «Іванов (Акація АБО Стройпромвсе)» знайде всі документи, виписані Івановим по контрагентам ТОВ "Акація" та ЗАТ "Стройпромвсе".
