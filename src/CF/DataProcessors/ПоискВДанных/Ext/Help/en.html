Example search Result
"Sale 00016422" Find the document "sale of goods" with the number "00016422"
"Payment 120000" Finds documents "payment" amount of 120,000 and 120,000.50
"Ivanov Acacia" Find all documents that prescribed for the contractor Ivanov LLC "Acacia"
"Sale Acacia" Find all documents "sale of goods" counterparty LLC "Acacia"
"Acacia 05.07" Find all documents counterparty LLC "Acacia" for July 5, 2008
"Sneakers" Find all references to products such as "sneakers" in order:
• Sneakers White (Product)
• Sneakers Black (Product)
• Parish Item number 00086822
• Flow Item number 00016422

Tips:

  • If there was a lot of data, then specify the search query by adding some refinement (date, amount, name of the contractor)
  • If found little evidence that simplify the query to exclude any word of it
  • To refine your search using the keywords:
  • Quotes - to search for an exact phrase, eg, "a" mini vacuum cleaner ""
  • OR - search for one of the two words, such as "Acacia OR Stroypromvse" - will find all references to counterparties LLC "Acacia" and ZAO "Stroypromvse."
  • Brackets - for grouping expressions, for example search for "Ivanov (Acacia Stroypromvse OR)" finds all documents drawn Ivanov counterparty LLC "Acacia" and ZAO "Stroypromvse."
