Abrufen aller Daten in der Informationsdatenbank.

Beispiel Such Ergebnis
"Der Verkauf von 00016422" Wird das Dokument "Verkauf von Waren" finden mit "00016422" No.
"Die Zahlung von 120000» Werden die Dokumente "Zahlung" in der Höhe von 120.000 finden und 120,000.50
"Ivanov Acacia" Dadurch werden alle Dokumente gezogen Ivanov für den Auftragnehmer von "Acacia" finden
"Der Verkauf von Acacia" Dadurch werden alle Dokumente "Verkauf von Waren" von der Gegenpartei von "Acacia" finden
"Acacia 05.07" Dadurch werden alle Dokumente auf dem Gegenstück zu "Acacia" von 5. Juli 2008 finden
"Turnschuhe" Dadurch werden alle Verweise auf Waren wie "Turnschuhe", um zu finden:
• Trainer White (Waren)
• Schwarze Turnschuhe (Güter)
• Die Ankunft der Waren Nummer 00086822
• Verbrauch Waren Nummer 00016422

Rat:

  • Wenn es eine Menge von Daten war, geben Sie eine Suchabfrage durch Hinzufügen einer Spezifikation (Datum, Betrag, Name des Auftragnehmers)
  • Wenn es ein wenig Daten war, vereinfacht die Abfrage davon auszuschließen jedes Wort
  • Zur Suche verfeinern die Schlüsselwörter verwenden:
  • Zitate - die Suche nach einem exakten Ausdruck, wie "ein" Mini-Staubsauger ' "
  • ODER - Suche nach einer der beiden Wörter, wie "Akazie oder Stroypromvse" - werden alle Verweise auf die Gegenparteien der OOO "Acacia" und CJSC "Stroypromvse" finden.
  • Brackets - gruppieren Ausdrücke, zum Beispiel, wenn nach "Ivanov (Akazie oder Stroypromvse)" findet alle Dokumente Ivanov Kontrahenten Ltd. "Acacia" gezogen und GAG "Stroypromvse".

Cm.ebenfalls: Ausführliche Beschreibung des Formats von Suchausdrücken
