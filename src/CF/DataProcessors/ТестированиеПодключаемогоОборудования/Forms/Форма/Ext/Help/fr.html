Forme de test "des exemples d'un appareil externe"

Ce formulaire vous permet de démontrer le fonctionnement du mécanisme de l'appareil externe. Formulaire implémente les fonctionnalités suivantes à travailler avec de l'équipement:

  • Connexion des équipements
    Il permet la connexion de l'équipement pour effectuer la VO de type (à l'ouverture du formulaire) ou par ID (fournies par l'utilisateur pour sélectionner un périphérique connecté à partir de la liste).
  • Équipement arrêt
    Vous permet de tenir à distance sur le type d'équipement VO (en fermant le formulaire) ou par ID (fourni par l'utilisateur de sélectionner un périphérique connecté à partir de la liste).
  • Éteignez tous les équipements connectés
    Il vous permet de tenir à distance (forcé, quel que soit le nombre d'utilisateurs d'un dispositif) de tous les équipements connectés.
  • Réinitialiser les paramètres mis en cache
    Pour tester le fonctionnement du mécanisme de nettoyage lorsque la mise en mémoire cache des valeurs
  • Production et transformation des événements externes
    Lets get l'événement à partir d'un périphérique externe à transmettre au mécanisme de traitement pour recevoir les données traitées par le mécanisme d'alerte et d'afficher le résultat sur l'écran
  • Travailler avec les registraires fiscales
    Il vous permet de connecter un registre financier, effectuez l'une des commandes d'interface d'informer sur l'erreur et puis éteignez le DF.
  • Travailler avec des terminaux de collecte de données
    Il vous permet de connecter terminal de collecte de données, décharger des données de terminaux de données ou de chargement prédéfinis à partir du terminal pour afficher sur l'écran, puis débranchez le DNT.
