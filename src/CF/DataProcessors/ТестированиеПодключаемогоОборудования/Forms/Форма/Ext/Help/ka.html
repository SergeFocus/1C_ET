ტესტი ფორმა "მაგალითები გარე მოწყობილობა"

ეს ფორმა საშუალებას გაძლევთ დემონსტრირება მუშაობის მექანიზმი გარე მოწყობილობები. ფორმა ახორციელებს შემდეგ ფუნქციებს მუშაობა აღჭურვილობა:

  • დამაკავშირებელი აპარატურა
    ეს საშუალებას იძლევა კავშირი აღჭურვილობა ჩაატაროს ტიპის VO (გახსნაზე სახით), ან ID (მომხმარებლის მიერ შესარჩევად დაკავშირებული მოწყობილობა სიიდან).
  • აღჭურვილობა ემუქრება
    გაძლევთ გამართავს off ტექნიკის ტიპის VO (როდესაც დახურვის სახით), ან ID (მომხმარებლის მიერ შესარჩევად დაკავშირებული მოწყობილობა სიიდან).
  • გამორთეთ ყველა დაკავშირებული მოწყობილობა
    ეს გაძლევთ საშუალებას გამართავს off (იძულებული გახდა, მიუხედავად იმისა, რომ მომხმარებელთა რაოდენობის მოწყობილობა) ყველა დაკავშირებული მოწყობილობების.
  • რესეტი ქეშირებული პარამეტრები
    შესამოწმებლად ფუნქციონირების მექანიზმი, როდესაც დასუფთავების ქეშირებული ღირებულებები
  • წარმოება და დამუშავება გარე მოვლენები
    საშუალებას მიიღოთ ღონისძიება გარე მოწყობილობის გადასცეს იგი დამუშავება მექანიზმი მიიღოს დამუშავებული მონაცემების მზადყოფნაში მექანიზმი და ცარიელია შედეგი ეკრანზე
  • სამუშაო ფისკალური რეგისტრატორები
    ეს საშუალებას გაძლევთ დაკავშირება ფისკალური რეესტრში, შეასრულოს ერთი ინტერფეისი ბრძანება ინფორმირება შეცდომა და შემდეგ გამორთეთ DF.

    ეს საშუალებას გაძლევთ დაკავშირება მონაცემთა შეგროვების ტერმინალში, განტვირთავს განსაზღვრული მონაცემები ტერმინალში ან დატვირთვის მონაცემები ტერმინალში არიან ეკრანზე, და შემდეგ შეუსაბამობა tsd.
