Test form "Examples from external equipment"

This form allows you to demonstrate the operation of the mechanism of external equipment. Form implements the following functionality to work with the equipment:

  • Connecting Equipment
    It allows connection of the equipment to conduct the type VO (at the opening of the form) or by ID (provided by the user to select a connected device from the list).
  • Equipment Shutdown
    Allows you to hold off on equipment type VO (when closing the form) or by ID (provided by the user to select a connected device from the list).
  • Turn off all connected equipment
    It allows you to hold off (forced, regardless of the number of users of a device) of all connected equipment.
  • Reset the cached settings
    To test the operation of the mechanism when the cleaning cached values
  • Production and processing of external events
    Lets get the event from an external device to transmit it to the processing mechanism to receive the processed data from the alert mechanism and display the result on the screen
  • Working with the fiscal registrars
    It allows you to connect a fiscal register, perform one of the interface command to inform about the error and then turn off the DF.
  • Working with data collection terminals
    It allows you to connect data collection terminal, unload predefined data terminal or loading data from the terminal to display on the screen, and then disconnect the TSD.
