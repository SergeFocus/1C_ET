Test forması "Xarici avadanlıq nümunələr"

Bu forma xarici avadanlıqların mexanizminin əməliyyat nümayiş etdirmək üçün imkan verir. Form avadanlıqları ilə işləmək üçün aşağıdakı funksionallığı həyata keçirir:

  • Connecting Equipment
    Bu (siyahıdan bir bağlı cihaz seçin istifadəçi tərəfindən təmin) (forma açılışında) və ya ID növü VO aparmaq avadanlıq keçid imkan verir.
  • Equipment Kapatma
    Siz (forma bağlayarkən) avadanlıq növü VO off tutma və ya ID (istifadəçi tərəfindən təmin siyahıdan bir bağlı cihaz seçin) imkan verir.
  • Bütün bağlı avadanlıq söndürün
    Bu bütün əlaqədar avadanlıq (bir cihaz istifadəçilərinin sayı asılı olmayaraq, məcburi) off saxlamaq üçün imkan verir.
  • Cached parametrlərini yenidən qurun
    Təmizləmə dəyərlər cached zaman mexanizminin əməliyyat test üçün
  • İstehsal və xarici hadisələr emalı
    Alert mexanizmi emal məlumat almaq və ekranda nəticə göstərmək üçün emal mexanizmi onu ötürmək üçün xarici cihaz hadisə almaq imkan verir
  • Maliyyə qeydiyyatçılar İş
    Bu, maliyyə reyestrinin qoşulmaq səhv haqqında məlumat və sonra DF söndürmək üçün interface komanda birini imkan verir.
  • Məlumatların toplanması terminalları ilə iş
    Bu, məlumatların toplanması terminal qoşulmaq ekranda keçirmək üçün terminalından əvvəlcədən təyin edilmiş məlumat terminal və ya loading data boşaltmaq, sonra TSD ayırmaq üçün imkan verir.
