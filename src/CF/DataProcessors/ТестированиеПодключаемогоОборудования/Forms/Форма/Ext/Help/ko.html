시험 형태 "외부 기기에서 예"

이 양식은 외부 장비의기구의 운영을 입증 할 수 있습니다. 양식 장비와 함께 작업 할 수있는 다음과 같은 기능을 구현합니다

  • 연결 장비
    그것은 (목록에서 접속 장치를 선택하기 위해 사용자에 의해 제공되는) (개구부의 형태로), 또는 ID가 입력 VO를 수행하는 장치의 연결을 허용한다.
  • 장비 종료
    당신이 (폼을 닫을 때) 장비 유형 VO에 보류 또는 ID로 (사용자가 제공 한 목록에서 연결된 장치를 선택) 할 수 있습니다.
  • 연결된 모든 장비의 전원을 끄고
    그것은 당신이 연결된 모든 장비 (장치의 사용자 수에 관계없이 강제로) 보류 할 수 있습니다.
  • 캐시 설정을 재설정
    클리닝 값을 캐시 할 때기구의 작동을 테스트
  • 생산 및 외부 이벤트의 처리
    경보기구로부터 처리 된 데이터를 수신하고, 결과를 화면에 표시하는 처리기구로 전달하는 외부 장치로부터 이벤트를 얻을 수
  • 재정 등록 작업
    그것은 당신이, 재정 레지스터를 연결 오류에 대한 정보를 제공하고 다음 DF을 해제 할 수있는 인터페이스 명령 중 하나를 수행 할 수 있습니다.
  • 데이터 수집 단말기와 협력
    그것은 당신이, 데이터 수집 단말기를 연결 화면에 표시 할 터미널에서 미리 정의 된 데이터 터미널이나로드 데이터를 언로드 한 다음 TSD를 분리 할 수​​ 있습니다.
