Обробка ДРУК ЕТИКЕТОК

Виконує наступні функції:

  • призначена для друку етикетки спільно зі створенням товару при його надходженні (F12)
  • або для друку етикеток для всієї таблиці товарів певного документа

При друку етикетки можуть використовуються формули формування штрих-коду і формули формування тексту етикетки.

Увага! Для коректного друку Штрих-коду повинна бути встановлена ​​компонента 1С: Друк штрихкодів.

У етикетках може фігурувати ціна.

Тутже можна роздрукувати етикетки (штрих-коди) контрагентів, організації, позиціям для послуг за час.
