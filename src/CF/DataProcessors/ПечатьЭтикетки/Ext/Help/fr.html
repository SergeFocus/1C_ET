Impression de l'étiquette de traitement

Il assure les fonctions suivantes:

  • conçu pour l'impression d'étiquettes, ainsi que la création de la marchandise quand il arrive (F12)
  • ou pour imprimer les étiquettes de tous les produits d'une table de document particulier

Lorsque vous imprimez des étiquettes peuvent être utilisés Formule formant code à barre formant texte et formules étiquettes.

Attention! Pour l'impression de code à barres devrait être mise en composant 1C: Impression de codes à barres.

Les étiquettes de prix peuvent apparaître.

Tutzhe peut imprimer des étiquettes (codes à barres) les entrepreneurs, les entreprises, les postes pour les services à l'époque.
