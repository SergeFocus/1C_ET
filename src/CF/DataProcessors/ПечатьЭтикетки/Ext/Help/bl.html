Друк этыкетак Апрацоўка

Ён выконвае наступныя функцыі:

  • прызначаны для друку этыкетак, разам са стварэннем тавару, калі яна прыбывае (F12)
  • або для друку этыкетак для ўсіх тавараў пэўнага планшэце

Пры друку этыкеткі могуць быць выкарыстаны Формула фарміраванне штрых-код фарміравання тэкставых і формул этыкеткі.

Увага! Для правільнага друку штрых-кода павінен быць усталяваны кампанент 1С: Друк штрых-кодаў,

Можа апынуцца цэннікі.

Тутже можа друкаваць этыкеткі (штрых-коды) падрадчыкі, кампаніі, пасады паслуг ў той час.
