Апрацоўка Друк этыкетак

Выконвае наступныя функцыі:

  • прызначаны для друку этыкетак ў спалучэнні са стварэннем тавару prihodovanii (F12)
  • або для друку этыкетак для ўсіх табліцы тавараў канкрэтнага дакумента

Пры друку этыкетак можа выкарыстоўвацца формула фарміраванне штрых-код фарміравання тэксту і сцвярджае пазнаку.

Увага! Для правільнай друку штрых-кода павінен быць усталяваны кампанент 1С: Друк штрых-кодаў.

У цэнніках могуць з'явіцца.

Тутже можа друкаваць этыкеткі (штрых-коды) падрадчыкі, арганізацыі, паслугі на пасады ў часу.
