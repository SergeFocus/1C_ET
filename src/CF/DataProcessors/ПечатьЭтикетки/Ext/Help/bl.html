Апрацоўка друку этыкетак

Наступныя функцыі:

  • прызначаныя для друку этыкетак, разам са стварэннем добра, калі ён прыбыў (F12)
  • або для друку этыкетак для ўсіх прадуктаў канкрэтнага планшэце

Пры друку этыкетак могуць быць выкарыстаныформулыфарміраваннештрых-код і фарміраванне формул тэкставых метак.

Увага! Для карэктнай друку штрых-кодаў кампанент для ўстаноўкі1С: Друк штрых-кодаў

Этыкеткі могуць з'явіцца ў цане.

Акрамя таго, можна друкаваць этыкеткі (штрых-коды) падрадчыкаў, арганізацый, пазіцыі на аказанне паслуг па часе.
