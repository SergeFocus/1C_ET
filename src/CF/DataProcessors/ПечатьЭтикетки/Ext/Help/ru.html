Обработка печати этикеток

Выполняет следующие функции:

  • предназначена для печати этикетки совместно с созданием товара при его поступлении (F12)
  • или для печати этикеток для всей таблицы товара определенного документа

 

При печати этикетки могут используются формулы формирования штрих-кода и формулы формирования текста этикетки.

Внимание! Для Корректной печати Штрих-кода должна быть установлена компонента 1С:Печать ШтрихКодов.

В этикетках может фигурировать цена.

Тут же можно распечатать этикетки (штрих-коды) контрагентов, организации, позициям для услуг за время.
