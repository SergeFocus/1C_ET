Consider the functions in the "Customer data".

The "Create a new customer with the name" allows the program to automatically create a card company. At the same time, it is necessary in the "new group of contractors" to specify the folder in which will be recorded all new counterparties. It should also make cards predefined settings "outstanding company." To do this, you must fill in the "Settlements with buyers", "supplier conditions," "Advances to customers" and "Advances to suppliers" in the "Account accounting calculations" cards of unknown buyer. These settings will be the default program included in the new card company.

The "Create a contract with the customer is automatically" allows the program to create a new company to automatically create a contract with him.

In the "product information" should be in the "Group of a new range of products" to specify the folder in which will be recorded all the elements of the nomenclature of the directory. At the same time, you should also configure the parameters of predefined cards "Unknown Product". By analogy with the filling of the program settings card settings of the new company made to the card predefined "outstanding counterparty", the program will fill the new configuration card product range according to the settings entered by default in the predefined card "Unknown product".

The function of this section "Generate name product with the order number in front of" the program allows you to specify the order number in the name of the product, which is enough important in custom production, with a kind of "uniqueness" of each the produced products.

Consider the adjustment made to the section "Information materials":

Checking the box "Download to the range of materials involved in the specification," will allow the program to automatically download in the range of reference materials, the brought in completing the specifications of the products. In the "Group of a new range of materials" must specify the folder in which you will be entered all the new elements of the specifications of products.
In this section, you must also fill out the parameters of predefined cards "unknown material" that the program will be used for automatic filling in the new range of materials.

Checking the box "Use Products counterparties synchronization nomenclature" will allow the program to automatically synchronize reference nomenclature, using the articles of counterparties.

After making the settings, you are ready to receive data. To do this, click the "Upload list of all the data." Note that the download is only for items that are mentioned in the new orders.

When you select "Test mode" data download will not be written into the program. This allows you to virtually put forward to check the settings and, if necessary, make further adjustments settings.
