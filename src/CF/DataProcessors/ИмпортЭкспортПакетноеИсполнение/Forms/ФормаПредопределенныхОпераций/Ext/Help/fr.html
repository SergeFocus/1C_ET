Considérons les fonctions dans les «données du client»:

Le "Créer un nouveau client avec le nom" du programme permet de créer automatiquement une compagnie de carte. Dans le même temps, il est nécessaire dans le "nouveau groupe d'entrepreneurs" pour spécifier le dossier dans lequel seront enregistrées toutes les nouvelles contreparties. Il convient également de faire des cartes des paramètres prédéfinis "de la société en circulation." Pour ce faire, vous devez remplir les «établissements avec les acheteurs», «conditions de fournisseur», «Avances à la clientèle» et «avances aux fournisseurs" dans les "compte calculs comptables" des cartes de l'acheteur inconnu. Ces paramètres seront le programme par défaut inclus dans la nouvelle société de la carte.

Le "Créer un contrat avec le client est automatiquement" permet au programme de créer une nouvelle société pour créer automatiquement un contrat avec lui.

Dans le "information produit" devrait être dans le "Groupe d'une nouvelle gamme de produits" pour spécifier le dossier dans lequel seront enregistrés tous les éléments de la nomenclature du répertoire. Dans le même temps, vous devez également configurer les paramètres de cartes prédéfini "Inconnu produit". Par analogie avec le remplissage des paramètres de la carte des paramètres du programme de la nouvelle société effectué sur la carte prédéfinie "contrepartie exceptionnelle», le programme va remplir la nouvelle gamme de produits de cartes de configuration selon les paramètres saisis par défaut dans la carte prédéfinie "Produit inconnu" .

La fonction de cette section "Générer nom produit avec le numéro de commande devant" le programme vous permet de spécifier le numéro de commande dans le nom du produit, ce qui est assez important dans la production de coutume, avec une sorte de «caractère unique» de chaque les produits obtenus.
