중요! 사용자 이름의 비교에 의해 수행된다.

이 사용자 정보 데이터베이스 파일을 갱신 할 수 있다면 파일과 같은 정보베이스 자명.이렇게하려면 체크 박스를 선택묶여 사용자 이름은 사용자의 등록 정보를 업데이트그렇지 않으면, 사용자 데이터 정보베이스 이름 우연 그대로 유지된다.

당신이 얻을해야하는 경우 부트 프로토콜은 플래그를 지정한다부트 프로토콜을 형성하도록 다운로드하기 전에.이 경우, 텍스트 파일은 각 사용자에 대한 정보가 포함 된 파일을 다운로드하여 생성된다 :

  • 사용자 : 사용자 이름.변경되지정보 기반에서의 사용자 파일.그 데이터는 변경되지 않습니다.
  • 사용자 : 사용자 이름.파일의 업데이트 된 데이터입니다. 정보 기반에서의 사용자 파일.데이터는 파일의 데이터를 갱신한다.
  • 사용자 : 사용자 이름.새로 만들었습니다. 지정된 이름으로 새로운 고객 정보 데이터베이스.
