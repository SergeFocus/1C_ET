중대한! 사용자 이름의 비교에 의해 수행된다.

파일의 데이터를 데이터베이스에 사용자 정보를 갱신 할 가능성이있을 경우 파일과 같은 데이터베이스에서의 사용자의 이름.이렇게하려면 체크 박스를 선택동일한 사용자 이름, 업데이트, 사용자의 호텔그렇지 않으면 사용자 정보 데이터베이스에있는 동일한 이름이 변경되지 않습니다.

부트 프로토콜을 얻고 싶은 경우에, 당신은 상자를 지정해야합니다부트 프로토콜을 형성하도록 다운로드하기 전에.이 경우, 텍스트 파일은 각 사용자로부터 파일을 다운로드하는 방법에 대한 정보가 포함되어 생성된다 :

  • 사용자 : 사용자 이름.변경되지파일의 사용자 정보 데이터베이스에서 발견 하였다.그의 데이터는 변경되지 않습니다.
  • 사용자 : 사용자 이름.파일에서 업데이트 된 데이터. 파일의 사용자 정보 데이터베이스에서 발견 하였다.그 데이터는 파일의 데이터로 업데이트됩니다.
  • 사용자 : 사용자 이름.새. 지정된 이름으로 새로운 회원 정보 데이터베이스.
