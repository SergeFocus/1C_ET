Upload und Download von Benutzerinformationen-Datenbank

Die Behandlung soll die Upload-und Download eine Liste der Benutzer von der Informationsbasis.

Um den Benutzer zu entladen muss drücken Benutzer laden .... Nach der Bildung der erforderlichen Daten, werden Sie aufgefordert, die Datei, die ausgeführt werden sollen Entladen. Vorgeschlagene Standard-Dateiname: IBUsers.xml.

Zum Download müssen die Benutzer klicken Laden Sie Benutzer .... Sie werden aufgefordert, eine Datei mit einer Liste von Benutzern (aus der Wissensbasis vor-Download) wählen und dann die Liste der Benutzer herunterlädt. Vorgeschlagene Standard-Dateiname: IBUsers.xml.

WICHTIG! User-Zuordnung nach Namen gemacht.

Es ist eine Gelegenheit, die Benutzerinformationen in der Informationsdatenbank der Datendatei zu aktualisieren, wenn der Benutzername in der Datei und in der gleichen Wissensbasis. Um dies zu tun, markieren Sie das Kästchen Mit den gleichen Namen Benutzern, Benutzereigenschaften aktualisieren. Andernfalls, wenn die Namen der Benutzerdaten in der Informationsbasis entsprechen, werden unverändert bleiben.

Wenn Sie brauchen, um das Boot-Protokoll zu erhalten, geben Sie das Feld Bilden Sie die Boot-Protokoll vor dem Download. In diesem Fall wird eine Textdatei mit Informationen zum Laden der Datei der einzelnen Benutzer zu erstellen:

  • User: Username. Nicht geändert. User-Datei wird in der Informationsdatenbank gefunden. Seine Figuren bleiben unverändert.
  • User: Username. Aktualisierte Daten aus der Datei. User-Datei wird in der Informationsdatenbank gefunden. Seine Daten werden mit den Daten aus der Datei aktualisiert.
  • User: Username. Eine neue. Die neue Benutzerinformationen-Datenbank mit dem angegebenen Namen.
