Upload und Download der Benutzerinformations-Datenbank

Die Behandlung soll Upload und Download eine Liste der Benutzer des Informationsbasis.

Um den Benutzer zu entladen müssen drücken Nutzer hochladen .... Nach der Bildung der erforderlichen Daten, werden Sie aufgefordert, um die Datei anzugeben, die ausgeführt entladen sein. Der Standardname der Datei: IBUsers.xml.

So laden Sie eine Benutzer muss klicken Laden Sie die Benutzer .... Sie werden aufgefordert, eine Datei, die eine Liste von Benutzern (zuvor von der Informationsbasis heruntergeladen) enthält, aus, und lädt dann die Benutzerliste. Der Standardname der Datei: IBUsers.xml.

WICHTIG! Benutzerzuordnung wird durch Namen.

Gibt es eine Möglichkeit, die Benutzerinformationen in der Datenbank mit den Daten aus einer Datei zu aktualisieren, wenn die Benutzernamen in der Datei und in der Datenbank der gleiche. Um dies zu tun, markieren Sie das Kästchen Mit identischen Benutzernamen, Update Benutzereigenschaften. Andernfalls mit dem gleichen Namen in der Benutzerinformations-Datenbank bleiben unverändert.

Wenn Sie eine Boot-Protokoll erhalten möchten, müssen Sie überprüfen, geben Sie Um das Boot-Protokoll bilden vor dem Download. In diesem Fall wird es eine Textdatei, die Informationen über Datei jedes Benutzers Laden zu bilden:

  • Benutzer: Username. Nicht geändert. Der Benutzer der Datei in der Informationsdatenbank gefunden. Seine Daten bleiben unverändert.
  • Benutzer: Username. Aktualisierten Daten aus der Datei. Der Benutzer der Datei in der Informationsdatenbank gefunden. Dessen Daten mit den Daten aus der Datei aktualisiert.
  • Benutzer: Username. Eine neue. Die neue Benutzerinformations-Datenbank mit dem angegebenen Namen.
