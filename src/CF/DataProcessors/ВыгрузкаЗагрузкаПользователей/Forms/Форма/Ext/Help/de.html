WICHTIG! Vergleich der Benutzer wird durch Namen durchgeführt.

Es besteht die Möglichkeit, Benutzerinformationen in der Datenbank der Daten aus einer Datei, falls der Name des Benutzers in der Datei und in der Datenbank desselben zu aktualisieren.Dazu markieren Sie das KästchenBei identischen Benutzernamen, Update-BenutzereigenschaftenAnsonsten ist der gleiche Name in der Informationsdatenbank Benutzer bleiben unverändert.

Wenn Sie das Boot-Protokoll erhalten möchten, müssen Sie das Feld angebenUm das Boot-Protokoll bilden vor dem Download.In diesem Fall wird die Textdatei erstellt, die Informationen zum Herunterladen von Dateien von jedem Benutzer enthält:

  • Benutzer: Username.Nicht geändertDer Benutzer der Datei in der Informationsdatenbank gefunden.Seine Daten bleiben unverändert.
  • Benutzer: Username.Aktualisierte Daten aus der Datei. Der Benutzer der Datei in der Informationsdatenbank gefunden.Seine Daten werden mit Daten aus der Datei aktualisiert.
  • Benutzer: Username.Ein neues. Ein neues Mitglied Informations-Datenbank mit dem angegebenen Namen.
