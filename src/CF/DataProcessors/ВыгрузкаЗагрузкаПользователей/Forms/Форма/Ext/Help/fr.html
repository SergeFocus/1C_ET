Charger et télécharger la base de données d'informations d'utilisateur

Le traitement est conçu pour charger et télécharger une liste de la base d'information des utilisateurs.

Pour télécharger l'utilisateur doit appuyer surAjouter les utilisateurs ...Après la formation des données nécessaires, le système vous demandera pour le fichier dans lequel la décharge est terminée.Le nom de fichier par défaut proposé:

Pour télécharger l'utilisateur doit appuyer sur le boutonTélécharger le mode ...Vous serez invité à sélectionner un fichier contenant la liste des utilisateurs (précédemment téléchargé à partir de la base de données), puis allez charger une liste d'utilisateurs.Le nom de fichier par défaut proposé:

IMPORTANT! Comparaison des utilisateurs est effectuée par son nom.

Il est possible de mettre à jour les informations utilisateur dans la base de données des données d'un fichier, si le nom de l'utilisateur dans le fichier et dans la base de ce dernier.Pour ce faire, cochez la caseAvec les noms d'utilisateur identiques, propriétés de l'utilisateur de mise à jourSinon, le même nom dans le mode base de données d'information restera inchangé.

Si vous voulez obtenir le protocole de démarrage, vous devez spécifier la boîtePour former le protocole d'initialisation avant de télécharger.Dans ce cas, le fichier texte sera créé qui contient des informations sur la façon de télécharger des fichiers à partir de chaque utilisateur:

  • Utilisateur: UserName.InchangéL'utilisateur du fichier trouvé dans la base de l'information.Ses données reste inchangé.
  • Utilisateur: UserName.Données mises à jour à partir du fichier. L'utilisateur du fichier trouvé dans la base de l'information.Ses données sont mis à jour avec les données du fichier.
  • Utilisateur: UserName.Un nouveau. Une nouvelle base de données d'informations sur les membres avec le nom spécifié.
