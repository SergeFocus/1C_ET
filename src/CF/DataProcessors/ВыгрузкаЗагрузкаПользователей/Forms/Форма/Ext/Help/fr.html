Charger et télécharger la base de données des informations utilisateur

Le traitement est destiné à charger et télécharger une liste d'utilisateurs de la base de l'information.

Pour décharger l'utilisateur doit appuyer sur Ajouter les utilisateurs .... Après la formation des données nécessaires, le système vous invite à spécifier le fichier d'être exécuté le déchargement. Le nom du fichier par défaut: IBUsers.xml.

Pour télécharger un utilisateur doit cliquer Télécharger le mode .... Vous serez invité à sélectionner un fichier qui contient une liste d'utilisateurs (précédemment téléchargé à partir de la base de données), puis télécharge la liste de l'utilisateur. Le nom du fichier par défaut: IBUsers.xml.

IMPORTANT! Cartographie de l'utilisateur est effectué par nom.

Il ya une possibilité de mettre à jour les informations de l'utilisateur dans la base de données des données d'un fichier, si le nom d'utilisateur dans le fichier et dans la base de données de la même. Pour ce faire, cochez la case Avec les noms d'utilisateur identiques, propriétés de l'utilisateur de mise à jour. Sinon, avec le même nom dans le mode base de données d'information resteront inchangés.

Si vous voulez obtenir un protocole de démarrage, vous devez spécifier chèque Pour former le protocole d'initialisation avant de télécharger. Dans ce cas, il formera un fichier texte contenant des informations sur le chargement du fichier de chaque utilisateur:

  • Utilisateur: UserName. Inchangé. L'utilisateur du fichier trouvé dans la base de données de l'information. Ses données reste inchangée.
  • Utilisateur: UserName. Données mises à jour à partir du fichier. L'utilisateur du fichier trouvé dans la base de données de l'information. Ses données sont mis à jour avec les données du fichier.
  • Utilisateur: UserName. Un nouveau. La nouvelle base de données de l'utilisateur avec le nom spécifié.
