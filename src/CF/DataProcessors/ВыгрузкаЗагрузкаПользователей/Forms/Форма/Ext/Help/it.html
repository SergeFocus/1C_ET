Caricare e scaricare le informazioni del database utenti

Il trattamento è stato progettato per caricare e scaricare un elenco della base dati degli utenti.

Per caricare utente deve premereCarica utenti ...Dopo la formazione dei dati necessari, il sistema chiederà per il file in cui lo scarico è completato.Il nome del file di default proposto:

Per scaricare l'utente deve premere il pulsanteScarica il manuale ...Verrà richiesto di selezionare un file contenente l'elenco degli utenti (precedentemente scaricato dalla base dati), e quindi si caricare un elenco di utenti.Il nome del file di default proposto:

IMPORTANTE! Confronto di utenti viene eseguita per nome.

C'è la possibilità di aggiornare le informazioni utente nel database dei dati da un file, se il nome dell'utente nel fascicolo e nella banca dati dello stesso.Per fare questo, selezionare la casellaCon nomi utente identici, proprietà utente aggiornamentoAltrimenti lo stesso nome della base dati delle informazioni utente rimarrà invariata.

Se si desidera ottenere il protocollo di avvio, è necessario specificare la casellaPer formare il protocollo di avvio prima di scaricare.In questo caso, verrà creato il file di testo che contiene le informazioni su come scaricare i file da ogni utente:

  • Utente: NomeUtente.Non cambiatoL'utente del file trovato nel database di informazioni.La sua tecnica è rimasto invariato.
  • Utente: NomeUtente.I dati aggiornati dal file. L'utente del file trovato nel database di informazioni.La sua tecnica è aggiornato con i dati del file.
  • Utente: NomeUtente.Un nuovo. Un nuovo database di informazioni membro con il nome specificato.
