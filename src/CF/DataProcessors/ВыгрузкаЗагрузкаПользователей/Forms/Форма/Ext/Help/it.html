Carica e scarica di database di informazioni dell'utente

Il trattamento ha lo scopo di caricare e scaricare un elenco di utenti della base dati.

Per scaricare l'utente deve premere Carica utenti .... Dopo la formazione dei dati necessari, il sistema richiede di specificare il file di scarico essere eseguito. Il nome predefinito del file: IBUsers.xml.

Per scaricare un utente deve fare clic su Scarica il manuale .... Vi verrà richiesto di selezionare un file che contiene un elenco di utenti (precedentemente scaricato dalla base dati), e poi scarica l'elenco degli utenti. Il nome predefinito del file: IBUsers.xml.

IMPORTANTE! Mappatura degli utenti avviene in base al nome.

Vi è la possibilità di aggiornare le informazioni utente nel database dei dati da un file, se il nome utente nel file e nella banca dati dello stesso. Per fare questo, selezionare la casella Con nomi utente identici, le proprietà di aggiornamento degli utenti. Altrimenti, con lo stesso nome utente nella banca dati di informazioni rimarrà invariata.

Se si desidera ottenere un protocollo di avvio, è necessario specificare di controllo Per formare il protocollo di avvio prima di scaricare. In questo caso, si formerà un file di testo contenente informazioni sul caricamento del file di ciascun utente:

  • Utente: NomeUtente. Non è cambiato. L'utente del file trovato nel database di informazioni. La sua tecnica è rimasto invariato.
  • Utente: NomeUtente. I dati aggiornati dal file. L'utente del file trovato nel database di informazioni. La sua tecnica è aggiornato con i dati del file.
  • Utente: NomeUtente. Un nuovo. Il nuovo database le informazioni dell'utente con il nome specificato.
