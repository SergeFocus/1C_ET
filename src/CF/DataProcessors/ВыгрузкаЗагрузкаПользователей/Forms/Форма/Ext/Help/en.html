Upload and download of user information database

Treatment is intended to upload and download a list of users of the information base.

To unload the user must press Users upload .... After the formation of the necessary data, the system prompts you to specify the file to be executed unloading. Suggested default file name: IBUsers.xml.

To download users must click Download user .... You will be prompted to select a file containing a list of users (pre-downloaded from the knowledge base), and then downloads the list of users. Suggested default file name: IBUsers.xml.

IMPORTANT! User mapping is done by name.

There is an opportunity to update the user information in the information database of the data file, if the user name in the file and in the same knowledge base. To do this, check the box With the same name users to update user properties. Otherwise, when the names match the user data in the information base will remain unchanged.

If you need to get the boot protocol, specify the box Form the boot protocol before downloading. In this case will generate a text file containing information about loading the file of each user:

  • User: UserName. Not changed. User file is found in the information database. His figures remain unchanged.
  • User: UserName. Updated data from the file. User file is found in the information database. His data are updated with data from the file.
  • User: UserName. A new. The new user information database with the specified name.
