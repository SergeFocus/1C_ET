Upload and download of user information database

Treatment is intended to upload and download a list of users of the information base.

To unload the user must press Upload users .... After the formation of the necessary data, the system prompts you to specify the file to be executed unloading. The default name of the file: IBUsers.xml.

To download a user must click Download the user .... You will be prompted to select a file that contains a list of users (previously downloaded from the information base), and then downloads the user list. The default name of the file: IBUsers.xml.

IMPORTANT! User Mapping is done by name.

There is an opportunity to update the user information in the database of the data from a file, if the user name in the file and in the database of the same. To do this, check the box With identical usernames, update user properties. Otherwise, with the same name in the user information data base will remain unchanged.

If you want to get a boot protocol, you must specify check To form the boot protocol before downloading. In this case, it will form a text file containing information about loading each user's file:

  • User: UserName. Not changed. The user of the file found in the information database. His data is left unchanged.
  • User: UserName. Updated data from the file. The user of the file found in the information database. Its data is updated with data from the file.
  • User: UserName. A new. The new user information database with the specified name.
