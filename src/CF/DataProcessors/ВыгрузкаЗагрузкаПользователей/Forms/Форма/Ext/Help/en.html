Upload and download user information database

To download the user should press the buttonDownload the user ...You will be prompted to select a file containing the list of users (previously downloaded from the information base), and then will load a list of users.The proposed default file name:

IMPORTANT! Comparison of users is performed by name.

There is the possibility to update user information in the database of the data from a file, if the user's name in the file and in the database of the same.To do this, check the boxWith identical usernames, update user propertiesOtherwise the same name in the user information data base will remain unchanged.

If you want to get the boot protocol, you must specify the boxTo form the boot protocol before downloading.In this case, the text file will be created that contains information about how to download files from each user:

  • User: UserName.Not changedThe user of the file found in the information database.His data is left unchanged.
  • User: UserName.Updated data from the file. The user of the file found in the information database.Its data is updated with data from the file.
  • User: UserName.A new. A new member information database with the specified name.
