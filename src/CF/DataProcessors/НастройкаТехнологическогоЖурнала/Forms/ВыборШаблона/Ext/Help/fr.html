Choisir un fichier de configuration du modèle

Dans ce formulaire, vous pouvez sélectionner un fichier de configuration du modèle, basé sur ce qui créera le magazine de la technologie de la configuration souhaitée.

Modèles fichier de configuration peut être construit (dans ce cas, ils sont marqués avec une photo en face du nom du modèle) et l'utilisateur (cette image est absent).

Il est également possible de supprimer un modèle personnalisé. Pour ce faire, sélectionnez la commande Retirer Dans les modèles de menu contextuel. Cette commande est disponible uniquement pour les modèles personnalisés.

Après avoir cliqué sur D'accord, Le modèle sélectionné sera chargé dans l'éditeur du fichier de configuration.
