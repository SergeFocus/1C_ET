Sélection du modèle de fichier de configuration

Dans ce formulaire, vous pouvez sélectionner le modèle de fichier de configuration sur la base duquel la configuration technologique souhaitée journal sera créé.

Modèles fichier de configuration peut être construit (dans ce cas, ils sont marqués avec une photo en face du nom du motif) et l'utilisateur (cette image est absent).

Il est également possible de supprimer un modèle personnalisé.Pour ce faire, sélectionnez la commandeEffacer dans la liste des modèles de menu contextuel.Cette commande est disponible uniquement pour les modèles personnalisés.

Après avoir cliqué surBien, Le modèle sélectionné sera chargé dans le fichier de configuration de l'éditeur.
