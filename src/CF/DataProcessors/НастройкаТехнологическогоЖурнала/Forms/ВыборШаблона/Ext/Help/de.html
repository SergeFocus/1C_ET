Wählen Sie eine Vorlage Konfigurationsdatei

In dieser Form können Sie eine Vorlage Konfigurationsdatei auswählen, auf deren Grundlage wird die gewünschte Konfiguration Technologie-Magazin zu erstellen.

Template-Konfigurationsdatei erstellt werden kann (in diesem Fall werden sie mit einem Bild vor dem Namen der Vorlage markiert sind) und dem Nutzer (dieses Bild fehlt).

Es ist auch möglich, ein eigenes Template zu löschen. Um dies zu tun, wählen Sie den Befehl Entfernen Im Kontext-Menü-Vorlagen. Dieser Befehl ist nur für benutzerdefinierte Vorlagen.

Nach Drücken OkAusgewählte Vorlage wird in den Editor-Konfigurationsdatei geladen werden.
