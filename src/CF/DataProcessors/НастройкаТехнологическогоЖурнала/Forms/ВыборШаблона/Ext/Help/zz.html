Выберите файл конфигурации шаблонов

В этой форме, вы можете выбрать файл конфигурации шаблона создается на основе нужной конфигурации технических журналов.

Шаблоны могут быть построены конфигурационный файл (в данном случае, они отмечены с прежним названием шаблона с изображением) и пользователь (изображение не существует).

Кроме того, вы можете удалить пользовательский шаблон. Чтобы сделать это, выберите команду Удалять В шаблонах контекстного меню. Эта команда доступна только в шаблоне.

После нажатия Хорошо, Выбранный шаблон будет загружен в файлы конфигурации редактировать.
