Select a template configuration file

In this form, you can choose a template configuration file, on the basis of which will create the desired configuration technology magazine.

Template configuration file can be built (in this case, they are marked with a picture in front of the name of the template) and the user (this picture is missing).

It is also possible to delete a custom template. To do this, select the command Remove In the context menu templates. This command is available only for user-defined templates.

After pressing OKSelected template will be loaded into the editor configuration file.
