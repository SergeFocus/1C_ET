Selecting the configuration file template

In this form you can select the configuration file template based on which the desired technological log configuration will be created.

Templates configuration file can be built (in this case they are marked with a picture in front of the pattern name) and the user (this picture is absent).

It is also possible to delete a custom template.To do this, select the commandDelete in the context menu templates list.This command is only available for custom templates.

After clickingOK, The selected template will be loaded into the editor configuration file.
