Choosing a template configuration file

In this form, you can select a template configuration file, based on which will create the desired configuration Technology magazine.

Templates configuration file can be built (in this case they are marked with a picture in front of the name of the template) and the user (this picture is absent).

It is also possible to delete a custom template. To do this, select the command Remove In the context menu templates. This command is only available for custom templates.

After clicking OK, The selected template will be loaded into the editor of the configuration file.
