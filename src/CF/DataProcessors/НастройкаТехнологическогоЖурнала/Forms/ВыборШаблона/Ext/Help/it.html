La scelta di un file di configurazione del modello

In questo modulo, è possibile selezionare un file di configurazione del modello, sulla base di che creerà la rivista Tecnologia configurazione desiderata.

Modelli file di configurazione può essere costruito (in questo caso sono contrassegnate con una foto davanti al nome del modello) e l'utente (questa immagine è assente).

E 'anche possibile eliminare un modello personalizzato. Per fare questo, selezionare il comando Rimuovere Nei modelli di menu contestuale. Questo comando è disponibile solo per i modelli personalizzati.

Dopo aver cliccato ok, Il modello selezionato verrà caricato nell'editor del file di configurazione.
