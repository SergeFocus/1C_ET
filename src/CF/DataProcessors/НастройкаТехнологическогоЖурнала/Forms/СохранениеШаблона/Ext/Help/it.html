La tecnologia di risparmio modello di file di configurazione rivista

In questa forma si sta installando opzioni generati modello di file di configurazione.

  • Nome modello - Una breve nome per il modello.Visualizza l'elenco nella finestrala selezione di un modello
  • Descrizione modello - Dato descrizione del modello (più avanzato rispetto al nome del modello).Visualizzati sul lato destro della finestrala selezione di un modello
  • Si specifica il percorso a qualsiasi directory dovrebbe indurre l'utente con una operazione di selezione del modello.

Dopo aver fatto clicBene File modello è il risparmio con il numero successivo.Salvare il file di modello non può essere impiegato sotto il numero.
