Salvare il file di configurazione rivista di tecnologia del modello

In questa forma, si sta installando le impostazioni del modello create dal file di configurazione.

  • Nome modello - Una breve nome per il modello. Consente di visualizzare nella casella di riepilogo Scelta del modello .
  • Descrizione Template - Imposta la descrizione del modello (più avanzato rispetto al nome del modello). Visualizzato nella finestra di destra Scelta del modello .
  • Si specifica il percorso qualsiasi directory dovrebbe indurre l'utente quando la selezione del modello di funzionamento.

Dopo aver cliccato ok file modello viene salvato con un altro numero. Salvare il file di modello con il numero occupato è impossibile.
