Speichern Sie die Konfigurationsdatei Template-Technologie-Magazin

In dieser Form werden Sie die Installation die Parameter, um eine Vorlage Konfigurationsdatei erstellen.

  • Name der Vorlage - Ein kurzer Name für die Vorlage. In der Liste angezeigt Template-Auswahl .
  • Beschreibung Template - Beschreibung der Muster angegeben (weiter fortgeschritten als die Template-Namen). Ist im rechten Fensterbereich angezeigt Template-Auswahl .
  • Gibt den Pfad zu dem Verzeichnis, das den Benutzer, wenn das Auswahl Vorlage auffordern sollte.

Nach Drücken Ok Template-Datei wird mit der nächsten Nummer gespeichert. Speichern Sie die Vorlage-Datei unter belegten Raum unmöglich.
