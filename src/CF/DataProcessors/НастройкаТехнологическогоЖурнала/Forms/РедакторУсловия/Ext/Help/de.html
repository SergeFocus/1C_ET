AGB Editor

Dieser Editor besteht aus drei verwandten Bereichen:

  • ein Feld, in dem Sie den Namen der Eigenschaft, die in dem Zustand beteiligt ist auszuwählen. Es wird als eine Option aus dem Menü und Schnellauswahl durch Eingabe ein Fragment von (auf Russisch) oder Kennung (in Englisch) Eigenschaften erlaubt. Beispielsweise zur Auswahl von Immobilien Dauer Sie können ein Fragment geben durUnd Fragment- lang .
  • Auswahl Feldbedingungen.
  • Auswahlfeld Wert Bedingungen. In einigen Fällen wird das Auswahlfeld Wert eine Liste von Entscheidungen.

Drücken Ok Sie führt zu einer Änderung der Bedingungen der Fixierung, der Press Absagen - Bricht die Änderungen.
