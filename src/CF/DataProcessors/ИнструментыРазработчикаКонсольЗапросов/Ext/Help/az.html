Idarə tətbiqi üçün Console sorğu

Tools "Console sorğu" 1C sorğu nəticələri yaratmaq və nəzərdən üçün nəzərdə tutulmuşdur: Enterprise.Bu müalicə əsasən həyata keçirilməsi üzrə developers konfiqurasiyaları və mütəxəssislər üçün nəzərdə tutulmuşdur.

Configurator sorğu inkişaf zaman real data tənzimlənməsi tələbi aparmaq adətən lazımdır.Bu araç sorğu dizayn imkan verir (və ya paket queries) paralel views nəticələri.Siz configurator olduğu kimi, Query Builder istifadə edə bilərsiniz qalın müştəri alət istifadə edərkən.Analiz nəticə sorğu xüsusiyyətləri bunlardır:

  • müvəqqəti data masalar tapmaq
  • ölçü sorğu vaxt və xətlərin sayı
  • sorğu nəticəsində bu hüceyrələri vurğulayaraq,
  • interaktiv iki sorğu nəticələri müqayisə (yalnız qalın müştəri)
  • yeni pəncərədə sorğu nəticəsində tapmaq.
  • bazasında yaradılan bağlanmış sorğu icra plan və SQL-sorğu mətn.Microsoft SQL Server DBMS icra planına üçün bir ağac kimi göstərilir, və verilənlər bazası istirahət üçün edilir - mətn formatında texnologiya jurnalı.Verilənlər bazası masalar və sütun və ya obyekt adları və metadata konfiqurasiya ətraflı adları: təhlili tələb iki ekran rejimi söz sorğu təmin asanlaşdırmaq üçün.

Tənzimlənməsi sonra sorğu kodu (Configurator mətn formalaşması sorğu istifadə edərək) və ya konfiqurasiya hesabat verilə bilər.Service imkanları bir neçə sorğu (sorğu paketi) ilə iş daxil bir elektron tablo sənəd və daha çox fayl, autosave, ixrac sorğu nəticələri mətn və sorğu parametrləri qorumaq üçün.
