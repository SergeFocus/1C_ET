Запрос на управляемой консоли приложения

Результаты инструмент "консоль Запрос", направленной на создание и рассмотрение запросов 1С: Предприятие.Основная цель этой терапии заключается в реализации конфигурации развития и экспертов.

При настройке запросов в области развития, как правило, требуют реальные данные, необходимые корректировки.Этот инструмент позволяет создать запрос (или пакетов запроса) параллельно просмотра результатов.Когда клиент жира может использовать инструмент, чтобы использовать конструктор запросов, как в конфигураторе.Необходимые характеристики Результаты, включающие:

  • глядя на временную таблицу данных
  • измерение времени запроса и количество строк
  • выделение этих клеток в результате запроса,
  • interactive сравнить два результата (только жир клиент)
  • теперь новое окно параллельных результатов запроса.
  • сформировать выводы в плане выполнения запросов к базе данных и SQL текста запроса.Для плана выполнения СУБД Microsoft SQL Server отображается в виде дерева, а для остальной части базы данных - в тексте технологии формат журнала.Для упрощения запроса анализ также обеспечивает два режима отображения слов запросов: имена и таблицы базы данных метаданных и столбцов или информацию об имени конфигурации объекта.

После настройки, запрос может быть в коде (с помощью текста образуют требуемую конфигурацию), или передать отчет о конфигурации.Возможности служб, в том числе несколько запросов (пакет запроса) рабочего документа сохраняются автоматически экспортировать результаты запроса в табличный документ, сохранить больше текста и параметры запроса.

Если результаты запроса в выводе должны использовать точку с запятой, вы должны избежать обратной косой черты - "
Например:ВЫБРАТЬ Nomenklatura.Naimenovanie + "\"; + С Nomenklatura.Naimenovanie Spravochnik.Nomenklatura

Требования к упаковке Быстрый текст и просит его параметры из консоли отладчика.

Sub-стандартная библиотека версии 2.3.2 и выше, имеют опцию "ObschehoNaznacheniya.ZaprosVStrokuXML ", которая позволяет быстро передавать запрос (весь текст и параметры) из отладчика для запроса консоли.Сделать это:

  • Перед настройкой и установки точек останова сразу вызывать методы "исполнительной власти", запроса в хранилище режиме отладки.
  • Когда выполнение кода останавливается Нажмите Shift + F9, чтобы остановить (или "Настройки", нажмите кнопку "Рассчитать выражение").В "выражение":
    • Введите "ObschehoNaznacheniya. В поле выраженияZaprosVStrokuXML (Request) ", где" Запрос "это имя переменной типа запроса.
    • Результаты представлены в таблице расчета формул в обращении, нажмите результат F2 строки (или контекстного меню выберите команду "Показывать значения в отдельном окне").
    • В "Show Expression" Выбрать и копировать текст в буфер обмена.
  • Нажмите кнопку "ОК", чтобы закрыть окно "выражение", чтобы отключить контрольную точку, продолжить (F5) и перейдите к запросам консоли Enterprise.
  • Запросы консоли вставить скопированный текст в текстовое поле и запрос "больше" нажать "с помощью заливки XML."
