การร้องขอสำหรับการประยุกต์ใช้คอนโซลการจัดการ

เครื่องมือ "คำขอคอนโซล" ถูกออกแบบมาเพื่อสร้างและทบทวนผลของแบบสอบถามใน 1C ที่: บริษัทการรักษานี้มีวัตถุประสงค์หลักสำหรับการกำหนดค่าพัฒนาและผู้เชี่ยวชาญในการดำเนินงาน

เมื่อมีการพัฒนาแบบสอบถามใน Configurator มักจะเป็นสิ่งที่จำเป็นในการดำเนินการคำขอปรับข้อมูลจริงเครื่องมือนี้ช่วยให้คุณสามารถออกแบบแบบสอบถาม (หรือคำสั่งแพ็คเก็ต) ผลการดูแบบคู่ขนานเมื่อใช้เครื่องมือในไคลหนาสามารถใช้สร้างแบบสอบถามเป็นคุณจะอยู่ใน Configuratorคุณสมบัติของการร้องขอผลการวิเคราะห์รวมถึง:

  • หาตารางข้อมูลชั่วคราว
  • วัดเวลาของการร้องขอและจำนวนของเส้น
  • ไฮไลต์เซลล์เหล่านี้เป็นผลมาจากการร้องขอ
  • โต้ตอบการเปรียบเทียบผลการค้นหาสอง (เฉพาะลูกค้าหนา)
  • และการหาผลแบบสอบถามในหน้าต่างใหม่
  • สรุปแผนปฏิบัติการแบบสอบถามและ SQL แบบสอบถามข้อความที่สร้างขึ้นในฐานข้อมูลสำหรับแผนปฏิบัติการ Microsoft SQL Server DBMS จะแสดงเป็นต้นไม้และสำหรับส่วนที่เหลือของฐานข้อมูล - ในนิตยสารเทคโนโลยีรูปแบบข้อความเพื่อให้ง่ายต่อการร้องขอการวิเคราะห์นอกจากนี้ยังมีคำสั่งสองโหมดการแสดงผลคำ: ชื่อของตารางฐานข้อมูลและคอลัมน์หรือชื่อวัตถุและรายละเอียดการกำหนดค่าข้อมูลเมตา

หลังการเปลี่ยนแปลงแบบสอบถามสามารถโอนในรหัส (โดยใช้การร้องขอการก่อข้อความสำหรับการปรับแต่ง) หรือการรายงานการกำหนดค่าความสามารถในการให้บริการไปยังรวมถึงการทำงานกับ (แพคเกจคำสั่ง) หลายคำสั่งรักษาข้อความและแบบสอบถามพารามิเตอร์ในแฟ้มบันทึกอัตโนมัติ, ผลแบบสอบถามส่งออกไปยังเอกสารสเปรดชีตและอื่น ๆ
