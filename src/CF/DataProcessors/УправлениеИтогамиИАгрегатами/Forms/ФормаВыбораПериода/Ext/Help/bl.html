У гэтым дыялогавым акне вы можаце ўвесці перыяд памежны разлічаных сум для назапашвання і рэгістраў бухгалтарскага ўліку.

Па змаўчанні, наступныя даты даступныя для ўстаноўкі мяжы разліковых вынікаў:

  • рэгіструе для захоўвання - канец папярэдняга месяца
  • рэгіструе для ўліку - канец месяца tekueschgo

Пасля змены, дата аўтаматычна пераўтворыцца ў дату заканчэння месяца па абранай даце.

Калі каманда ўстаноўкі колькасці абранага для вылічэнні вынікаў адным рэгістры, ён зможа змяніць толькі адзін параметр, адпаведны тыпе рэгістра.
