У гэтым дыялогавым акне можна ўвесці перыяд памежны разлічаных сум для рэгістраў назапашвання або бухгалтэрыі.

Па змаўчанні, наступныя даты даступны для ўстаноўкі мяжы разліковых вынікаў:

  • для рэгістраў назапашвання - у канцы папярэдняга месяца
  • для рэгістраў бухгалтарскага ўліку - у канцы месяца tekueschgo

Пасля змены, дата аўтаматычна пераўтворыцца ў дату заканчэння месяца па абранай даце.

Калі час ўстаноўкі каманда вылічэнні выніку, выбраную для аднаго рэгістра, ён зможа змяніць толькі адзін параметр, тып адпаведнага рэгістра.
