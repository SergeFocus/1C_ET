В даному діалозі можна ввести кордони періоду розрахованих підсумків для регістрів накопичення або бухгалтерії.

За замовчуванням пропонуються наступні дати для установки кордону розрахованих підсумків:

  • для регістрів накопичення - закінчення попереднього місяця
  • для регістрів бухгалтерії - закінчення текуещго місяці

Після зміни, дата автоматично перетвориться до дати закінчення місяця обраної дати.

Якщо команда установки періоду розрахованих підсумків обрана для одного регістра, то буде доступна зміна тільки одного параметра, відповідного типу регістра.
