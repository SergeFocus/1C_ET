The results of management includes procedures related to the outcome of the accumulation and accounting registers as well as units of current accumulation registers.The results of the management is divided into two modes:

  • Private use possible - Allow "one touch" to perform the most common actions arising from the maintenance of the system.
  • Full opportunities - It allows to perform all operations with a single register or register with an arbitrary list.

To switch between modes is a hyperlink in the lower right corner of the form.The form stores open mode.

Frequently used features

With the results of operations

  • Set between the calculated outcome - To speed up the system installs the period calculated for the results of all the accumulation and accounting registers enabled outcome.For storage registers - at the end of the previous month, for accounting registers - at the end of this month.
  • Enable the use of the results of - Involves the use of the results of all the registers, in which the mode is switched off (for example as a result of a crash of any procedure that disables the use of the results).

Operations with units

  • Rebuild and fill - All revolving accumulation registers, which included units mode and found their use, performs an operation to rebuild the use of aggregates, and performs filling units.The operation can be performed for a long time.
  • Get the best units - Retrieves a list of the best units for all circulating accumulation registers, which are set in the Configurator units.You will be prompted directory where you saved xml-files with lists of the best units.Each file will be named identifier corresponding register.So, to registerThe sale the file will be calledProdazhi.xml
