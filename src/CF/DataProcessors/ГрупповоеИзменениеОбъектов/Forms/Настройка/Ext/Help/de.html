Hinzu kommt noch, auf der Titelseite Assistent Sie können bestimmte Einstellungen mit dem Befehl SET Konfigurieren Wechsel. Das System kann Objekte in der Transaktion zu behandeln. Wenn der Aufzeichnungsvorgang mit der Markierungsfahne Ändern Sie in der Transaktion nicht auf eines der Objekte zu schreiben (beispielsweise, weil das Objekt von einem anderen Benutzer editiert) werden, wird die Änderung in alle Objekte aufgehoben werden. Standardmäßig ist diese Option deaktiviert.

Wenn die Änderungsobjekte sind nicht in einer Transaktion, ist es empfehlenswert, um es mit einem Häkchen durchführen Versehen unterbrechen. In diesem Fall, wenn das Objekt Datensatz aus irgendeinem Grund nicht möglich ist, wird das System eine Warnung aus und bieten auch weiterhin die übrigen Objekte zu ändern oder zu unterbrechen, um die Ursache des Problems zu ermitteln. Wenn das Kontrollkästchen nicht aktiviert ist, und das Objekt kann nicht aufgezeichnet werden, wird die Verarbeitung wie Objekt überspringen und wird auch weiterhin die übrigen Objekte zu ändern.

Ticken Behandeln Sie die untergeordneten Elemente (Standardeinstellung nicht aktiviert ist) können Sie die untergeordneten Elemente der ausgewählten Gruppen und übergeordneten Objekten, ohne die Flaggen aller Elemente der Gruppen und untergeordnete Objekte Installation ändern. Ist die Option deaktiviert, verarbeitet das System die Gruppe selbst, nicht seine Kinder.
