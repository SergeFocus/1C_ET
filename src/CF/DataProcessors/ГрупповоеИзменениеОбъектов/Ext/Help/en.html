To change the details (as well as additional details and information) of the selected items in the list you must enter the new values ​​in the column The new value and press Change.

To clear the value of props need to put down the check box, and left column The new value empty, press the button Change.

If you need to further customize the list of items to be handled, it is possible to use the button Next. In the list, you can adjust the composition of the elements, as well as view the current value of their properties. On the button View item or double-click it in the list, you can go to the card member.

With the command Story You can select a previously saved settings to change the elements.

Additionally, you can set special settings with the command Configure change.

Special occasions:

  • It should be borne in mind that the list of details may be details that are not available to change and are grayed out. In the case of a special need to change their changes, you can use the command Allow editing of details .
