Prețurile înregistrați

Ea determină prețul laprodus pentru preţul de tip lavalută un fel de a prețurilor la zi.Valabilă până următoarele setări de preț.

Prețurile sunt specificate manual sau cu ajutorul documentului"Tarifarea Set 'Se recomandă să se stabilească prețurile de nomenclatură a prețurilor de setare a documentelor.Dar acest lucru este opțională.Doar așa că este mai ușor să le administreze.Prețul poate fi alocate șiintra reziduuri

Prețurile dependenteSpecii Pret sunt calculate automat.

Când ștergeți o înregistrare cu privire la prețul, elimina toate prețurile corespunzătoare a prețurilor de specii dependente.

Pretul este reprezentat în cea de a doua zi, după data specificată.
