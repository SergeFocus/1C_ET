Регистр заданных произвольных печатных форм

Позволяет для каждого вида произвольных печатных форм закрепить за пользователем определенную произвольную печатную форму.

Если печатная форма не задана явно, будет использована первая подходящая по списку. Чтобы отключить ее, следует сделать неактивной.
