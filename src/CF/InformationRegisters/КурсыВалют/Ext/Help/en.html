EXCHANGE RATES

If the system uses multiple Rates, Then for each of these courses should be set for the conversion prices, Excluding currency in settlements with customers and suppliers and Cash Flows.

The course is relative to the base currency of the program and indicates how this basic unit of currency in the foreign currency.

For example, the exchange rate "dollar" = 8 for the base currency "hryvnia" means that one dollar 8 hryvnia.

The course is set to the date and is valid until the next course.
