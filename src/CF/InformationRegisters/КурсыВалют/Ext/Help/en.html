EXCHANGE RATES

If the system uses several CurrenciesThen each of them should be set rates for prices, Accounting currencies in mutual settlements with Customer and providers and Cash Flows.

The course is relative to the base currency of the program and indicates how much of the base currency per unit of the currency.

For example, the exchange rate "dollar" = 8 for the base currency 'hryvna' means that one dollar 8 hryvnia.

The course is set to date and is valid until the next course.

In the case of complex accounting prices when setting the cost by dependent rates will be recalculated. But if you delete a record on the course of will not be deleted. Such prices would have to be counted by hand through the process.
