Inscrivez informations "codes barres"

L'utilisation des codes à barres inclus dans les paramètres du programme.

Un code à barres unique qui définit la nomenclature sélectionné lorsque vous entrez le code à barres. Codes-barres des marchandises ne peuvent pas être répétées.

Les principaux produits de codes à barres stockés directement dans sa carte. Cependant, si le numéro de code à barres associé à la nomenclature, elles peuvent être stockées dans le registre d'information.

Si les paramètres du programme spécifiées pour l'utilisation de codes à barres Caractères seulement latine, le programme se traduira par des caractères incorrects code à barres (cyrillique) à leurs homologues latins (pour les cas de manière incorrecte votre disposition de clavier).
