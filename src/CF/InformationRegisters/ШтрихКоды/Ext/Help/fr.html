Utilisation Barcode commutédans les paramètres du programme

Un code-barres unique identifie cela nomenclature sélectionné lorsque vous entrez le code à barres.Les codes à barres du produit ne peut pas être répété.

Le principal code à barres des marchandises sont stockées directement dans sa carte.Toutefois, si la nomenclature Barcode associé à plusieurs, ils peuvent être écrites sur ces informations de registre.

Si spécifié dans les paramètres du programme d'utiliser le code à barres que des caractères latins, le programme se traduira par des symboles code à barres incorrect (cyrillique) à leurs homologues latins (en cas de disposition de clavier incorrect sélectionné).
