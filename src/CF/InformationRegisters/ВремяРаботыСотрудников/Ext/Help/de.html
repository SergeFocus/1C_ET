Bilanzierung von Mitarbeiterzeit zur Verfügung zu konfigurieren und nur Benutzer mit vollen Rechten zu analysieren.

In den Einstellungen enthalten:


Für einen ordnungsgemäßen Betrieb müssen Sie alle Benutzer ihrer Mitarbeiter zu fragen:


Kurz, wie es funktioniert:
Der Benutzer arbeitet mit dem Abrechnungssystem - das Fenster öffnen, geben Sie Daten in die Tabelle.
Sie können die Zeit zwischen diesen Vorgängen festgelegt, die eine akzeptable Pause betrachtet werden (mindestens 2 Minuten, empfehle ich - 15).

Während das Programm für den Anwender feststellt, wird es in dieser Zeit als Arbeitszeit gezählt.
Wenn der Benutzer im Leerlauf mehr gültig Pause ist, wird angenommen, dass er den Arbeitsplatz verlassen.
Sie können eine Frage für einige Zeit vor dem Ablauf des Benutzers aktivieren, um zu bestätigen, dass er für 30 Sekunden auf den Job war.

Wenn der Benutzer für den Arbeitsplatz zurückkehrt - ist es über die Pause und startet eine neue Arbeitseinheit.
