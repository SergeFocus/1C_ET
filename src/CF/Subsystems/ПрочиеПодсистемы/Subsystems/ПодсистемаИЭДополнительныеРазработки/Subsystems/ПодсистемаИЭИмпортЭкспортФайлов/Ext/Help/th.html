ระบบการทำงานของข้อมูลที่นำเข้าและส่งออกโดยใช้ระบบไฟล์

กำลังโหลดข้อมูลและขนไดเรกทอรีและเอกสารที่ปรับแต่งและผู้ใช้ระดับ

ตัวอย่างของการใช้:

  • - แลกเปลี่ยนสองทางกับธนาคารลูกค้าผ่านไฟล์ข้อความ
  • - อัพโหลดและดาวน์โหลดเอกสารและคำสั่งของการเคลื่อนย้ายสินค้ากับซัพพลายเออร์และผู้ซื้อผ่านไฟล์ XLS
  • - อัพโหลดและดาวน์โหลดช่วงของราคาหรือราคาที่ลูกค้า
  • - รายละเอียดการโหลดข้อมูลของ XLS
  • - ดาวน์โหลดไดเรกทอรีจากไฟล์ DBF (เช่นจากซอฟแวร์เก่า)

    เป็นต้น

ระบบช่วยเหลือ

ที่ช่วยให้ระบบย่อยนี้ติดตั้งในการกำหนดค่าผู้ใช้ UNF 1C สำหรับยูเครน

ควรจะอยู่ใน "การติดตั้งและการบริหาร" ใน "การตั้งค่า" เปิดบล็อก "การกำหนดค่าการพัฒนาเพิ่มเติม."ในหน้าต่างที่ปรากฏขึ้นให้ติ๊กรายการสำหรับ"การใช้ระบบย่อยการนำเข้าส่งออกของไฟล์"แล้วคลิก "เผา" หรือ "เผาไหม้และใกล้."แต่หลังจากที่องค์ประกอบของระบบย่อยนี้มีอยู่

การตั้งค่าอื่น ๆ ที่มีอยู่จากหน้าต่างนี้:

ไฟล์ที่อยู่ญาติ - ตัวเลือกนี้จะช่วยให้เมื่อมีการเข้าถึงไฟล์จากชื่อเซิร์ฟเวอร์จะแตกต่างจากการอ้างอิงไปยังแฟ้มในนามของผู้ใช้

ตัวอย่างเช่นหากผู้ใช้ในการกำหนดค่า 1C UNF แพลตฟอร์ม 8.2 และเซิร์ฟเวอร์การกำหนดค่าที่อยู่บน 1C เครือข่ายการเข้าถึงไฟล์สำหรับการดำเนินการเข้าและส่งออกไม่จำเป็นต้องมีการลงทะเบียนชื่อผู้ใช้ C: \ Common \ 1.txt และในนามของเซิร์ฟเวอร์ \\ UserComputer \ Common \ 1.txt

แล้วใน "ไฟล์ที่อยู่ญาติ" ควรจะทำ \\ UserComputer \ Common \ ซึ่งต่อมาไฟล์สำหรับการดำเนินการนำเข้าและการส่งออกที่จะเลือกได้อย่างรวดเร็วและถูกต้อง

จำกัด จำนวนแถวดาวน์โหลด- หากคุณได้เตรียมเพียงหมายถึงการถ่ายโอนข้อมูลที่มีขนาดใหญ่ แต่ต้องการใช้พวกเขาในเว็บไซต์

คุณชั่วคราวสามารถตั้งข้อ จำกัด นี้รายการนี​​้ไม่สามารถใช้ได้สำหรับรุ่นสาธิตของระบบข้อ จำกัด ในกรณีนี้ - 10 บรรทัดอย่างเคร่งครัด

องค์ประกอบของระบบ

รูปแบบของการดำเนินการนำเข้าส่งออกของไฟล์ ยังมีอยู่ใน "การติดตั้งและการบริหาร" ในเมนูด้านซ้ายในแม่แบบโฟลเดอร์ "ที่กำหนดไว้ล่วงหน้า" เก็บไว้แม่แบบที่กำหนดโดยโปรแกรมเมอร์

การเปลี่ยนแปลงในรูปแบบเหล่านี้สามารถทำได้โดยผู้ใช้กับระบบการอนุมัติการเขียนโปรแกรมเช่นการทำงานของโปรแกรมเมื่อการนำเข้าและการส่งออกที่ไม่ได้มาตรฐานสำหรับพวกเขา

จากหน้าต่างของรายการจากเทมเพลตที่มีลักษณะของระบบที่ไม่ซ้ำกันนี้:

รายชื่อทีมและ-E (Import-Export) หลังจากที่ระบบจะปรากฏในรายการของการ์ดและคู่มือมาตรฐานและเอกสารระบบรายการของคำสั่งที่ช่วยให้คุณเริ่มต้นอัปโหลดหรือดาวน์โหลดไปยังรายชื่อของรายการหรือองค์ประกอบฐานข้อมูลถ้าแม่แบบที่เหมาะสมเพียงคนเดียวที่จะถูกเลือกโดยอัตโนมัติโดยระบบ

แต่ในรายการไดเรกทอรี "แฟ้มแม่แบบการนำเข้าและการส่งออก" ในรายการของคำสั่งและสามารถใช้ได้พิเศษอี:

การดำเนินงานที่กำหนดไว้ล่วงหน้าของการนำเข้าและการส่งออก - การเริ่มต้นการแลกเปลี่ยนของอัลกอริทึมที่ไม่ซ้ำกันระบบที่วางไว้โดยนักพัฒนา

การจัดการกับปัญหาดังกล่าวจำเป็นต้องมีการฝึกอบรมส่วนบุคคลและไม่ได้อธิบายไว้ภายในเอกสารอ้างอิงนี้

แลกเปลี่ยนแพ็คเก็ตสำหรับชุดของแฟ้ม- เมื่อคุณจำเป็นต้องโหลดหรือขนวัตถุฐานข้อมูลโดยใช้การรวมกันของสองคนหรือมากกว่าไฟล์

ตัวอย่างหนึ่งสำหรับส่วนหัวเอกสารและอีกสอง - สำหรับชิ้นส่วนของตารางเอกสาร

การประมวลผลนี้เป็นสิ่งจำเป็นเพื่อระบุชนิดของการแลกเปลี่ยนวัตถุหรือสถานที่แลกเปลี่ยนของตัวเองระบุแม่แบบไฟล์นำเข้าส่งออกสำหรับวัตถุส่วนหัวที่อยู่ของไฟล์เพื่อเขาคุณสามารถระบุระยะเวลา (ในกรณีของเอกสาร)

แล้วกรอกข้อมูลในตารางที่ระบุชื่อตารางของวัตถุในขณะที่มันตั้งอยู่ใน Configurator คำอธิบายแฟ้มแม่แบบตารางนี้สอดคล้องกับที่อยู่ของไฟล์

รายการของงานที่ใช้งานร่วมกัน - เมื่อต้องการเรียกใช้รายการการดำเนินงานนำเข้าส่งออกในลำดับ

การรักษานี้สามารถกรอกตารางที่มีรายชื่อของชุดของแฟ้มแม่แบบและที่อยู่ของพวกเขาระบุทิศทางของ "นำเข้า - ใช่ส่งออก - no."

ระบุชนิดของการแลกเปลี่ยนและชื่อตารางของวัตถุในกรณีที่จำเป็นแล้วเริ่มต้นรายการของการดำเนินการแลกเปลี่ยนคุณสามารถระบุระยะเวลาที่เลือกสำหรับการดำเนินการเกี่ยวกับเอกสาร

การรักษาเอกสารเงินสดแลกเปลี่ยน (ในขณะที่ "เงินสด") มีความสามารถในการดำเนินการในรายชื่อนำเข้า () และส่งออก () สำหรับรายชื่อของงานในพิธีเปิดการประมวลผลเลือกรายการของแม่แบบที่เหมาะสม (สำหรับการแลกเปลี่ยนเอกสารเงินสด)

ไฟล์คำสั่งแม่แบบรายการอื่น ๆ :

วารสารของการดำเนินงานนำเข้าส่งออกบันทึกนี้ประกอบด้วยข้อมูลเกี่ยวกับระบบการผลิตของการดำเนินงานเข้าและส่งออกหากพวกเขา "นิตยสารข่าว" ที่ระบุไว้ในแฟ้มแม่แบบ

ยกตัวอย่างเช่นการดำเนินงานนำเข้าทั้งหมดที่มีธงของ "ดาวน์โหลดไฟล์ทั้งหมดจากโฟลเดอร์นี้" นิตยสารเพื่อหลีกเลี่ยงการดำเนินการอีกครั้งไฟล์ดาวน์โหลด

ดาวน์โหลดแม่แบบ / แม่แบบบันทึกการดำเนินงานเหล่านี้ช่วยให้การแลกเปลี่ยนแม่แบบนำเข้าและส่งไฟล์กับผู้ใช้อื่น ๆ ของระบบหรือระบบนักพัฒนา

รายละเอียดของไฟล์ที่ถูกเก็บไว้ในรูปแบบพิเศษ "ieUNFSH" ซิงค์และนำเข้าโดยใช้ชื่อแม่แบบไม่ได้ตัวเลือกแม่แบบทั้งหมดสามารถโอนโดยการดาวน์โหลดแม่แบบพารามิเตอร์เหล่านั้นที่มีเฉพาะในฐานข้อมูลที่คุณจะต้องตั้งค่าด้วยตนเอง

บัตรนำเข้าส่งออกแฟ้มแม่แบบ

นี้เป็นวัตถุที่ระบบที่ซับซ้อนมากที่สุดเพื่อความสะดวกสบายก็ควรจะเต็มไปจากบนลงล่างจากขวาไปซ้าย

ยกตัวอย่างเช่นเห็บในหมวก "เพื่อการส่งออกเท่านั้น" หรือ "เท่านั้นสำหรับการนำเข้าของ" ช่วยซ่อนพารามิเตอร์แม่แบบที่ไม่ได้ใช้ในกรณีนี้

ดูแม่แบบ มันกำหนดงานหลักของรูปแบบนี้ยกเว้น "รุ่นใด" พารามิเตอร์เราขอแนะนำให้เลือกแม่แบบฟอร์มที่เหมาะสมกับงานของคุณ

ในการเชื่อมต่อกับทางเลือกของประเภทเฉพาะของแม่จะสามารถใช้ได้กับเหล่านั้นหรือพารามิเตอร์อื่น ๆ ของไฟล์แม่แบบพารามิเตอร์แต่ละครั้งจะถูกอธิบายไว้ด้านล่าง

แม่แบบประเภทมีดังนี้

    • หากต้องการจองหรือบัตรกระดาษ - แม่แบบอธิบายวัตถุเองหรือสเปรดชีตหรือองค์ประกอบฐานข้อมูลใด ๆ ไดเรกทอรีหรือเอกสาร

แม่แบบนี้จะช่วยให้ตามลำดับการโหลดหรือยกเลิกการโหลดวัตถุหรือตารางหนึ่งบรรทัดของการนำเข้าไฟล์และการส่งออก - หมวกอุปกรณ์ประกอบฉากหนึ่งหรือหนึ่งบรรทัดของตาราง

    • สำหรับรายชื่อของรายการหรือเอกสารไดเรกทอรี - แม่แบบอธิบายฟิลด์บนพื้นฐานของสายแฟ้มแม่แบบ - องค์ประกอบหนึ่งคู่มือหรือเอกสารฉบับเดียว

"การตั้งค่าแม่" พารามิเตอร์บล็อกจะเป็น "กฎระเบียบสำหรับรายการของการทำงาน."

    • สำหรับตารางของผลิตภัณฑ์- สำหรับส่วนตาราง "สินค้า" เอกสารเมื่อคุณนำเข้าข้อมูลแม่แบบโปรแกรมจะพยายาม dozapolnit สายข้อมูลสำหรับสินค้า

"การตั้งค่าแม่" จะมีการตั้งค่าพิเศษตัวอย่างเช่นถ้าเป็นธงตั้ง "สร้างวัตถุใหม่บล็อกจะสามารถมองเห็นได้จากค่าพารามิเตอร์เริ่มต้นสำหรับการตั้งชื่อใหม่ของ" กฎระเบียบสำหรับการทำงานกับเงินสำรอง "

    • ในการแลกเปลี่ยนเอกสารเงินสด - ชนิดพิเศษของแม่แบบสำหรับการดำเนินงานของพฤติกรรม (กรอกข้อมูลลงในการนำเข้า) เครื่องมือรวมทั้งเงินสดเพื่อใช้ร่วมกับระบบธนาคารไคลเอ็นต์

บล็อก "ข้อตกลงและเงื่อนไขของการทำธุรกรรมกับเครื่องมือเงินสด" ช่วยให้เราสามารถอธิบายแง่ของการแลกเปลี่ยนในรายละเอียด

    • ผลจากการร้องขอใด ๆ - ดูแม่แบบที่ช่วยให้คุณสามารถจัดระเบียบการส่งออกของผลการค้นหาใด ๆ ในภาษา 1C 8

สำหรับประเภทของการร้องขอที่ "ตั้งค่าแม่" นี่คุณสามารถระบุข้อความของแบบสอบถามใด ๆ ที่เขียนในภาษา 1C 8, เช่นเดียวกับการตั้งค่าของพารามิเตอร์แบบสอบถามในตารางด้านล่าง

ในเวลาเดียวกันที่ "สาขาลำดับ" ใน "ชื่อฟิลด์" ควรจะระบุชื่อของเขตข้อมูล - ผลการค้นหา

ปุ่ม "พารามิเตอร์ดาวน์โหลดจากข้อความแบบสอบถาม", "ฟิลด์สร้างไฟล์เกี่ยวกับผลของแบบสอบถาม", "ตรวจสอบคำขอ" ช่วยให้ตารางเพื่อเติมเต็มทุ่งนาและพารามิเตอร์โดยอัตโนมัติทดสอบการทำงานของการร้องขอปุ่ม "เรียกใช้ขน" ยกเลิกการโหลดข้อมูลโดยตรงจากแม่แบบบัตร

ชื่อตาราง - เลือกที่จะระบุตารางสำหรับไดเรกทอรีหรือเอกสาร (เช่นมันเป็นชื่อในการปรับแต่ง) ซึ่งจะอยู่ภายใต้การดำเนินงานนำเข้าส่งออก

ต่อไปนี้จะอธิบายถึงพารามิเตอร์แม่แบบซึ่งจะพบในแท็บของบัตรของเขาขึ้นอยู่กับประเภทของ:

ที่คั่นหนังสือ"ตัวเลือก File" มันถูกใช้เพื่อระบุพารามิเตอร์ file (s) เพื่อนำเข้าหรือส่งออก

ดาวน์โหลดไฟล์ทั้งหมดจากโฟล์เดอร์นี้ - มันชี้ให้เห็นว่ามันเป็นสิ่งที่นำเข้าสำหรับอาร์เรย์ของไฟล์ที่นี่คุณควรระบุหน้ากาก Directory และไฟล์จะตรวจสอบการเลือกของแฟ้มที่เหมาะสม
