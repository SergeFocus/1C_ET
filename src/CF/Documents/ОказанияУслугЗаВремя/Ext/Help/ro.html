Documentul "Servicii timpului"

Documentul este utilizat numai atunci cândcontorizare așezări cu clienții.

Documentul este destinat pentru a reflecta furnizarea unor astfel de servicii pentru clienții care sunt înregistrate în acest timp.
Logica documentului este
Operatorul stabilește la începutul furnizării de servicii (client, data, poziția (număr) și (sau) înseamnă).
Apoi, la sfârșitul serviciului este fixat data de sfârșit și a calculat valoarea serviciilor prestate, se formează prețul.
