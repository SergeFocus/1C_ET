Документ "УСТАНОВКА ЦЕН"

Документ используется, чтобы зарегистрировать установку или изменение цен для большого списка номенклатуры.

Расчет и установку цен для наборов товаров удобно производить специальной кнопкой "Заполнить таблицу наборами и расчитать им цены". В этом случае цена единицы наборов будет рассчитана из сумм всех компонентов. В общем случае установки цены для "набора", и при расчете зависимых цен, программа будет отталкиваться уже от сформированной цены набора, не изучая цену его компонент.

 

Данные "О строке" -заметки, информация пользоателя. На печать не выводятся, но хранятся в программе.

По кнопке F10 доступен диалог быстрого ввода товара с количеством. При его использовании достаточно нажать F10, затем ввести сканером или вручную штрих код (если он используется) или выбрать номенклатуру, указать Количество и F10 - завершить ввод данных.

По кнопке F11 ввод по Штрих-коду вручную (если он используется), например для случаев когда сканер кода отключен или  не читает код.
