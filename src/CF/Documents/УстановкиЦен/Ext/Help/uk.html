Документ "ВСТАНОВЛЕННЯ ЦІН"

Документ використовується, щоб зареєструвати установку або зміна цін для великого спискуноменклатури

Розрахунок і установку цін для наборів товарів зручно робити спеціальною кнопкою "Заповнити таблицю наборами і розрахувати їм ціни".У цьому випадку ціна одиниці наборів буде розрахована з сум всіх компонентів.У загальному випадку установки ціни для "набору", і при розрахунку залежних цін, програма буде відштовхуватися вже від сформованої ціни набору, не вивчаючи ціну його компонент.

Якщо використовується складний облік цін, то одним документом встановлюються ціни для певного види цінТак само по кнопці "Вивести інформацію про ціни в іншому вигляді цін" можна готувати ціни одного виду, спираючись на значення в іншому вигляді цін.

Реєструються тільки змінені ціни.Якщо різниця ціни - 0, ціна і залежні від неї не реєструються.

Дані "Про рядку" -заметкі, інформація користувача.На друк не виводяться, але зберігаються в програмі.

За кнопці F9 доступний діалог швидкого введення товару з кількістю.При його використанні достатньо натиснути F9, потім ввести сканером або вручну штрих код (якщовін використовується) Або вибрати номенклатуру, вказати Кількість і F9 - завершити введення даних.

За кнопці F11 введення по Штрих-коду вручну (якщовін використовується), Наприклад для випадків коли сканер коду відключений або не читає код.
