Documentul "prețul stabilit"

Documentul este utilizat pentru a înregistra sau modifica setarea prețurilor pentru o listă mare nomenclatură.

Calculul și stabilirea prețurilor pentru produse specifice este convenabil de a transporta un buton special "completează seturile de masă și se calculează prețurile lor." În acest caz, prețul unitar este stabilit va fi calculată din sumele tuturor componentelor. În general, stabilirea prețurilor pentru "set" și calcularea prețurilor dependente, programul se va baza pe deja format un set de preturi, fără a studia prețul componentelor sale.

Dacă utilizați prețuri contabile complexe, același document stabilit prețurile pentru anumite preț de tip. De asemenea, pe "informații de afișare cu privire la prețurile din alte forme de prețuri" pot fi preparate pentru prețul de un singur tip, pe baza valorilor într-o altă formă de prețuri.

Înregistrat doar modificarea prețului. În cazul în care diferența dintre prețurile de - 0, prețul și dependent de ea fiind înregistrate.

Acestea -zametki "On line", informații de utilizator. Ea nu se imprimă, dar sunt stocate în program.

După ce apăsați F9 este disponibil intra rapid un dialog cu cantitatea de mărfuri. Atunci când se utilizează o, pur și simplu apăsați F9, apoi introduceți codul de bare sau manual (în cazul în care acesta este folosit) Sau selectați o gamă, specificați numărul și F9 - pentru a finaliza introducerea datelor.

Pe tasta F11 bare de intrare manual (dacă acesta este folosit), De exemplu, în cazurile în care scanerul de cod este dezactivat sau nu poate citi codul.
