Le document "SET PRIX"

Le document est utilisé pour enregistrer ou modifier la fixation des prix pour une grande liste la nomenclature.

Le calcul et la fixation du prix des produits spécifiques est pratique à transporter un bouton spécial "Remplissez les sets de table et de calculer leurs prix." Dans ce cas, le prix unitaire est fixé sera calculée à partir des quantités de tous les composants. En général, la fixation des prix pour le "set" et le calcul des prix à charge, le programme va construire sur le déjà formé un ensemble de prix, sans étudier le prix de ses composants.

Si vous utilisez des prix comptables complexes, le même document fixer les prix pour certains Prix ​​de. Egalement sur le "Afficher des informations sur les prix dans les autres formes de prix" peuvent être préparés pour le prix d'un type, sur la base des valeurs d'une autre forme de prix.

Enregistré seul changement de prix. Si la différence de prix - 0, prix et dépendant de lui enregistrées.

Ces -zametki "en ligne", les informations utilisateur. Il ne imprime pas, mais ils sont stockés dans le programme.

Après avoir appuyé sur F9 est disponible rapidement entrer dans un dialogue avec la quantité de marchandises. Lors de son utilisation, il suffit d'appuyer sur F9, puis entrez le code à barres du scanner ou manuellement (si c'est utilisé) Ou sélectionnez une plage, spécifier le nombre et F9 - pour terminer la saisie de données.

Sur l'entrée de touche F11 code-barres manuellement (si c'est utilisé), Par exemple dans les cas où le scanner de code est désactivé ou ne peut pas lire le code.
