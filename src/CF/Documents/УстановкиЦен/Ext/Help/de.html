Document "Neubewertung"

Dokument wird verwendet, um die Installation oder die Veränderung der Preise für eine große Liste registrieren Nomenklatur.

Berechnung und Festsetzung der Preise für bestimmte Produkte ist bequem zu tragen eine spezielle Taste "Fill Tischsets und berechnen ihre Preise." In diesem Fall setzt der Stückpreis von den Mengen der Komponenten berechnet werden. Im Allgemeinen ist die Preissetzung für "Set", und die Berechnung der Preise abhängig, wird das Programm auf dem bereits gebildeten einer Reihe von Preisen zu bauen, ohne den Preis für seine Komponenten zu lernen.

Wenn Sie eine komplexe Verrechnungspreise sind, das gleiche Dokument legen die Preise für bestimmte Arten von Preisen. Einfach auf die "Display Informationen über Preise in anderen Formen der Preise" kann zum Preis von einer Spezies, basierend auf dem Wert einer anderen Form von Preis vorbereitet werden.

Diese "On line" Notizen, Informationen Polzoatel. Nicht gedruckt, sondern im Programm gespeichert.

F10-Taste zur Verfügung Dialog schnell in der Menge des Produktes. Wenn Sie es verwenden, drücken Sie einfach F10 und geben Sie dann den Scanner oder Barcode manuell (wenn es verwendet wird,) Oder wählen Sie einen Bereich, die Menge eingeben und F10 - komplette Dateneingabe.

F11-Taste, um den Barcode manuell eingeben (wenn es verwendet wird,), Beispielsweise in Fällen, in denen Code-Scanner ist deaktiviert, oder können den Code nicht lesen.
