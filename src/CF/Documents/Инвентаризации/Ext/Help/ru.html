Документ "ИНВЕНТАРИЗАЦИЯ"

  • При инвентаризации номенклатуры необходимо сверить учетное и фактическое количество. 
  • Произвести до-оприходование и списание товаров при несоответствии фактическому учету. 
  • Зарегистрировать связанное с этим движение денег или отразить разницу в заработной плате сотрудников. 
  • Сформировать связанный документ реализации товаров по результатам инвентаризации. 

    Не может быть инвентаризации товаров типа "наборы". Инвентаризации подлежат их составные части.

    Не могут быть инвентаризированы товары с признаком "без количественного учета" и услуги.

    Данные "О строке" -заметки, информация пользоателя. На печать не выводятся, но хранятся в программе.

    При отражении инвентаризации в зарплате сотрудников в первую очередь учитывается ответственный по складу. Если он не установлен - производится распределение на всех сотрудников.

    В настройках конфигурации может быть включена возможность регистрации реализации по результатам инвентаризации. Такой документ инвентаризации не производит изменений, но связан с документом реализации. Для этого в документе инвентаризации не должно быть строк с "оприходованием" товаров (увеличением количества), так как нет возможности это отразить в виде реализации.

    По кнопке F10 доступен диалог быстрого ввода товара с количеством. При его использовании достаточно нажать F10, затем ввести сканером или вручную штрих код (если он используется) или выбрать номенклатуру, указать Количество и F10 - завершить ввод данных.

    По кнопке F11 ввод по Штрих-коду вручную (если он используется), например для случаев когда сканер кода отключен или  не читает код.
