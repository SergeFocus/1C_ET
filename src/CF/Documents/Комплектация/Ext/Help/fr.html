Le document "PACKAGE"

Le document est utilisé pour enregistrer la fabrication (emballage) ou par razukomplektatsiyu Caractéristiques produits ou manuellement.

Sur le "Résultat" - produits complémentaires.

Sur les «Matériaux» - radiés lorsque les matériaux complets (nomenclature).

Sur les «œuvres» - Travaux remplies Personnel lors de l'assemblage.

Sur les "Outils" - propre installations les entreprises impliquées dans l'ensemble.

Sur les "dépenses supplémentaires" - dépenses de trésorerie supportées par notre société dans le processus de regroupement / razukomplektatsii.

Razukomplektatsiya - l'équipement de processus inverse. En ce sens les résultats sont radiés, et les matériaux reviennent.
