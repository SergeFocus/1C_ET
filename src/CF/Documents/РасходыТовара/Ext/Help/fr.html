Le document «consommation de biens et services SALES", "retourner au fournisseur», «produit de CHARGE"

Si Set Client - Le document fera la vente de biens et services. Si, au lieu inconnu fournisseur - Le document fera le retour des marchandises au fournisseur. Si la contrepartie est pas spécifié, le document sera radier les marchandises.

Le document permet également d'enregistrer immédiatement le paiement de l'acheteur en une somme forfaitaire ou une table des paiements dans des devises différentes.

Si la mise en œuvre de l'acheteur privé a été donné un rabais, vous devez changer la quantité de marchandises au détriment des biens. Si vous modifiez le montant de la réception, qui est la vente d'un prêt, et non une réduction!

Si les engins utilisés des prix préférentiels, le programme examine le nombre de bar la nomenclature et établit un prix préférentiel, si le nombre est supérieur ou égal à cette barre. Ensuite, si l'option "problèmes dans la mise en œuvre d'un prix réduit comme un rabais sur le produit", le prix ne changera pas, et la colonne Prix réduits.

Dans les calculs client ou fournisseur directement dans le flux de documents de la marchandise (utilisant sous-système Y compris de l'argent), la forme de paiement est prélevé sur les paramètres du programme. Si le paiement est effectué dans un autre mode de paiement ou le paiement complexe (par exemple, le mouvement de plusieurs devise), Vous devez remplir le tableau «paiement»

Le tableau doit être fait pour payer l'argent reçu du client au cours de la mise en œuvre. Autres colonies signer des documents du mouvement d'argent (revenus) du client.

Le cours du document peut être corrigée si les règlements avec les clients ou les fournisseurs sont faites dans un rythme différent de celui totale installée taux de change du jour.

Radiation de biens est effectuée sans tenir compte du cours du document. Si Voir les prix les dépenses «monnaie», les marchandises sont consommées avec le taux de change courant total.

Les documents de cours sont également prises en compte dans l'analyse de leurs propres soldes de trésorerie libellés dans la monnaie de base.

Dans la mise en œuvre du "set" dans les "ventes" va obtenir des informations sur la vente d'un ensemble de positions, mais sera déduit de l'entrepôt de toutes ses composantes.

Ces «Sur la ligne" informations -zametki de polzoatelya. Il ne imprime pas, mais ils sont stockés dans le programme.
