Документ "Внутрішнє замовлення"

Документ використовується тільки при включеному обліку по складах.

Документ призначений для відображення переліку внутрішнього замовлення для складу одержувача від складу-донора.
Документ формує "вимоги".
У момент надходження товарів на склад-одержувача, товари замовлення списуються з "вимоги".
Якщо документ позначений, як "виконаний", то товари так само списуються з вимоги.
Спостерігати за сформованими вимогами можна у звіті за внутрішніми замовленнями.
