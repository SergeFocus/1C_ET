Документ внутрішнього замовлення

Документ використовується тільки привключеному обліку по складах.

Документ призначений для відображення переліку внутрішнього замовлення для складу одержувача від складу-донора.
При проведенні документ формує "вимоги".
У момент надходження товару на склад-одержувача, товари замовлення списуються з "вимоги".
Якщо документ позначений, як "виконаний", то товари так само списуються з вимоги.
Спостерігати за сформованими вимогами можна в звіті за внутрішніми замовленнями.
