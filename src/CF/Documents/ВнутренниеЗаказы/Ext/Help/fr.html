Le document «Ordre interne"

Le document est utilisé uniquement lorsque Activation de la comptabilité Entrepôt.

Le document est destiné à refléter l'ordre interne de la liste pour le magasin-entrepôt sur le destinataire du donneur.
Pendant le document constitue une "exigence."
Au moment de la réception des marchandises à l'entrepôt du destinataire, les produits Ordre sont débités des «exigences».
Si le document est étiqueté comme «fait», le produit a également déduit de l'obligation.
Suivre les exigences peuvent être établies dans le rapport sur les commandes internes.
