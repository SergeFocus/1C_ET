Papierverkaufsplan

Es ermöglicht Ihnen, einen Plan Verkaufs Nomenklatur Nomenklatur Gruppen oder Gruppen von Nomenklatur zu zeichnen und dann eine vergleichende Plan-Ist-Analyse zu machen.

Die Planung kann für eigene Nomenklatur, Nomenklatur Gruppen oder Gruppen von Klassifizierung (Verzeichnis) durchgeführt werden.

Die Planung kann als Verkäufe auf die Anzahl und Menge der Realisierung sein.

Diese "On line" -zametki, Benutzerinformationen.Nicht gedruckt, sondern werden im Programm gespeichert.

Mit einem Klick auf die F9 verfügbar ist, schnell einen Dialog mit der Menge der Waren vor.Wenn Sie es, drücken Sie einfach F9, dann geben Sie Scanner oder Barcode manuell (fallses wird verwendet,) Oder einen Bereich auswählen, die Anzahl und die F9 angeben - die Dateneingabe zu vervollständigen.

Mit einem Klick auf die F11-Eingabe Barcode manuell (fallses wird verwendet,), Zum Beispiel in den Fällen, wenn der Code-Scanner ist deaktiviert oder kann den Code nicht lesen.
