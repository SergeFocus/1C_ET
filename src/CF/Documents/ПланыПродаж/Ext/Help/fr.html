Le "plan de vente" de documents

Il vous permet d'émettre un plan de vente des groupes de la nomenclature nomenclature ou les groupes de la nomenclature, puis faites un plan comparatif vs analyse proprement dite.

Scheduling peut être effectuée pour la nomenclature séparée, les groupes de la nomenclature ou des groupes désignés (de référence séparée).

La planification peut être aussi la quantité des ventes, et le montant de la réalisation.

Ces «Sur la ligne" informations -zametki de polzoatelya. Il ne imprime pas, mais ils sont stockés dans le programme.

Après avoir appuyé sur F9 est disponible rapidement entrer dans un dialogue avec la quantité de marchandises. Lors de son utilisation, il suffit d'appuyer sur F9, puis entrez le code à barres du scanner ou manuellement (si c'est utilisé) Ou sélectionnez une plage, spécifier le nombre et F9 - pour terminer la saisie de données.

Sur l'entrée de touche F11 code-barres manuellement (si c'est utilisé), Par exemple dans les cas où le scanner de code est désactivé ou ne peut pas lire le code.
