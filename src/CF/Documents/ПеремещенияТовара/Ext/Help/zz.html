При использовании более одного СкладС мобильного программы и количественного зрения продукт должен быть со склада на склад.

Мобильные и складские "товаров в пути" необходимо, когда это необходимо принимать во внимание, что продукт временно недоступна из-за отсутствия такого склада, из которого могут быть реализованы.

При перемещении тип продукта "набор" на самом деле состоит из движущихся в ее состав всех элементов. Тем не менее, для четкого учета фактического остатка на, мы предлагаем вам этот "пакет" для каждого отдельного элемента. Если "Настройки" Объем учет его компонентов рассчитываются автоматически для каждого элемента. Этот документ использует таблицу игнорируется.

Когда цена указанного документа смотреть сложные бухгалтерские механизмы и цена, из источника хранилища и склада приходится oprimhodovaniya получателей для отмены включают, учитывая цену денег (валюты бухгалтерского учета, если это применимо) производится в соответствии с установленными ценами зрения.

Динамики цена / объем, цена сорняков реквизит для отображения только в документе

Программа использует внутренние цены на семена.
