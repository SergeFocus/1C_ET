Das Dokument Waren-

Bei der Verwendung von MehrfachlagerhallenProdukte mit einer quantitativen Blick auf das Programm, sollte auch aus dem Lager zu Lager bewegt werden.

Fahren Sie mit dem Lager "Waren auf der Durchreise" können Sie ist notwendig für die Fälle, in denen es notwendig ist, zu berücksichtigen, dass das Produkt aufgrund eines Mangels in solchen Lagern ist vorübergehend nicht verfügbar, aus denen es durchgeführt werden kann.

Bei den Produkttyp "set" zu bewegen, in der Tat, machte die Bewegung aller Elemente ihrer Zusammensetzung.Aber für eine klare Abrechnung der tatsächlichen Rückstände wird empfohlen, dass Sie jedes Element von "Set" auseinander bewegen.Wenn Sie die "set" der Buchführung über die Menge ihrer Bestandteile bewegen berechnet für jedes Element automatisch.In der Tabelle des Dokuments wird ignoriert.

Wenn ein komplexer Mechanismus zur Feststellung der Preise und die in der Dokumenttyp angegeben Preise, die Rechnung von der Quelle Lager abgebucht und dem Lager Entsendung Empfänger wird mit Bezug auf die festgelegten Preis gemacht einschließlichunter Berücksichtigung der Währung Preis (falls zutreffend Rechnungswährung).

Lautsprecher Preis / Menge, Preis Requisiten Art nur verwendet, um das Dokument anzuzeigen

Das Programm verwendet die interne Arten von Preisen.

Die Maßeinheit (In Rechnungseinheiten) im Falle anzugeben, dass dieses Mal ist es anders als die grundlegende Einheit der Messung Nomenklatur.Diese Regel gilt für alle Konfigurationsdokumente.

Diese "On line" -zametki, Benutzerinformationen.Es wird nicht gedruckt, sondern werden im Programm hinterlegt.

Mit einem Klick auf die F9 verfügbar ist, schnell einen Dialog mit der Menge der Waren vor.Wenn Sie es, drücken Sie einfach F9, geben Sie Scanner oder Barcode manuell (fallses ist benutzt) Oder einen Bereich auswählen, die Anzahl und F9 angeben - die Dateneingabe zu vervollständigen.

Mit einem Klick auf die F11-Eingang Barcode manuell (fallses ist benutzt), Beispielsweise in Fällen, in denen der Code Scanner ist deaktiviert oder kann das nicht lesen.
