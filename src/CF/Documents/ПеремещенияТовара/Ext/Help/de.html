Das Dokument Warenverkehr

Bei der Verwendung von mehrerenlagerhallenProdukte mit einer quantitativen Sicht des Programms sollte auch aus dem Lager zu Lager bewegt werden.

Bewegen Sie die Lager "Waren im Transit" können ist notwendig für die Fälle, in denen es notwendig ist, zu berücksichtigen, dass das Produkt aufgrund eines Mangels in einem solchen Lager vorübergehend nicht verfügbar ist, von dem es umgesetzt werden kann.

Bei den Produkttyp "set" zu bewegen, in der Tat, machte die Bewegung aller Elemente seiner Zusammensetzung.Aber für eine klare Abrechnung der tatsächlichen Rückstände wird empfohlen, dass Sie jedes Element von "Set" auseinander bewegen.Wenn Sie die "Set" der Buchführung über die Menge ihrer Bestandteile bewegen berechnet für jedes Element automatisch.In der Tabelle des Dokuments wird ignoriert.

Wenn ein komplexer Mechanismus für die Aufnahme Preise und die in dem Dokumenttyp angegebenen Preise, die Rechnung von der Quelle Lager belastet und das Lager Posting Empfänger wird unter Bezugnahme auf den festgelegten Preis gemacht, einschließlichunter Berücksichtigung der Währung Preis (ggf. Rechnungswährung).

Lautsprecher Preis / Menge, Preis Requisiten Art nur verwendet, um das Dokument anzuzeigen

Das Programm verwendet die interne Arten von Preisen.

Die Maßeinheit (In Rechnungseinheiten) sollten nur im Fall angegeben werden, dass dieses Mal ist es anders als die Grundeinheit der Mess Nomenklatur.Diese Regel gilt für alle Konfigurationsdokumente.

Diese "On line" -zametki, Benutzerinformationen.Nicht gedruckt, sondern werden im Programm gespeichert.

Mit einem Klick auf die F9 verfügbar ist, schnell einen Dialog mit der Menge der Waren vor.Wenn Sie es, drücken Sie einfach F9, dann geben Sie Scanner oder Barcode manuell (fallses wird verwendet,) Oder einen Bereich auswählen, die Anzahl und die F9 angeben - die Dateneingabe zu vervollständigen.

Mit einem Klick auf die F11-Eingabe Barcode manuell (fallses wird verwendet,), Zum Beispiel in den Fällen, wenn der Code-Scanner ist deaktiviert oder kann den Code nicht lesen.
