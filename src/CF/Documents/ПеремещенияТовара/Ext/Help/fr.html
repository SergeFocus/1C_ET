Le document "circulation des marchandises"

Lors de l'utilisation multiple entrepôts, Les produits avec une vue quantitative du programme et doivent être déplacés de l'entrepôt à l'entrepôt.

Déménager avec l'entrepôt «marchandises en transit» est nécessaire pour les cas où il est nécessaire de prendre en compte que le produit est temporairement indisponible en raison d'un manque dans ces entrepôts, à partir de laquelle il peut être mis en œuvre.

Lors du déplacement du type de produit "set" est réellement fait pour déplacer tous les éléments de sa composition. Mais pour une comptabilité claire de résidus réelles il est recommandé que vous vous déplacez chaque élément de ce «paquet» seul. Si vous déplacez le "set" montant de la comptabilité pour ses composants est calculé automatiquement pour chaque élément. Ce document utilise la table est ignoré.
