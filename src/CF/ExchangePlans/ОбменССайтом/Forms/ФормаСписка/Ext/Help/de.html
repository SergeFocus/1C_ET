Austausch mit Websites

Hier können Sie den Datenaustausch zwischen den Rechnungslegungsinformationssystem und Online-Shop, unterstützt die Konfiguration Protokoll zwischen dem System "1C: Enterprise" und Website.

Hauptmerkmale:

  • Entlassung aus der Programm-Website Katalog von Produkten, einschließlich aller Eigenschaften der Ware, die Preise und den Überresten der Waren in Lagerhallen, sowie die automatische Pflege der Daten auf dem neuesten Stand.
  • Laden Sie das Programm der im Laden ausgestellt Ordnung, sowie die Aktualisierung der Daten auf der Website der Status der Zahlung und Versand von Bestellungen.

Die Ausgangsdaten sind in der Liste
  • Produktname - Hostname
  • Datum Download - Das Datum der letzten Download Bestellungen von der Website
  • Datum erfolgreichen Download - Das Datum der letzten erfolgreichen Download-Aufträge aus der Website
  • Datum der Entlastung - Das Datum der endgültigen Entladen von Gütern auf der Website, oder aktualisieren Informationen zu Bestellungen
  • Datum erfolgreiche Landung - Das Datum der letzten erfolgreichen Entladen von Gütern auf der Website, oder aktualisieren Informationen zu Bestellungen
  • Der Austausch von Waren - Flagge auf der Website Produktkatalog hochladen
  • Umtauschanträge - Anmelden Download auf Kundenaufträge

Verfügbare Befehle
  • Führen Sie Exchange - Startet eine Sitzung der Datenaustausch mit der Website
  • Führen Sie einen Austausch - In diesem Modus wird der Austausch von Produktkatalogdaten wird das vorhandene Verzeichnis zu überschreiben (wenn es bereits auf der Website hochgeladen)
  • Veränderungen registriert anzeigen - Öffnet ein Formular Ansicht von Waren und Aufträge, für die Daten werden auf die Website hochgeladen werden
