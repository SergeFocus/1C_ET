주요 특징들:

  • 모든 상품의 특성, 가격과 창고에서 상품의 잔재뿐만 아니라 최신 상태로 데이터를 자동으로 유지 보수를 포함하여 제품의 프로그램의 웹 사이트 카탈로그에서 방전.
  • 상점에서 발행 순서의 프로그램을 다운로드뿐만 아니라, 지불 및 주문 배송 상태의 웹 사이트에 데이터를 업데이트.

출력 데이터는 목록에
  • 상품명 - 호스트 이름
  • 다운로드 날짜 - 사이트에서 마지막으로 다운로드 주문 날짜
  • 날짜 성공적으로 다운로드 - 사이트에서 마지막으로 성공한 다운로드 주문 날짜

  • 날짜 성공적으로 착륙 - 주문에 마지막으로 성공한 사이트에서 제품의 하역, 또는 업데이트 정보의 날짜
  • 상품의 교환 - 플래그는 사이트의 제품 카탈로그에 업로드
  • 교환 주문 - 고객의 주문에서 로그인 다운로드
