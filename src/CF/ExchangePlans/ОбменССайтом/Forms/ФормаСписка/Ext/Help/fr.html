Échange avec des sites

Permet de configurer l'échange de données entre le système d'information comptable et de boutique en ligne qui prend en charge Protocole entre le système "1C: Enterprise" et le site Web.

Principales caractéristiques:

  • Décharge du site le catalogue de produits, y compris toutes les propriétés des produits, les prix et les restes des marchandises dans les entrepôts, ainsi que la maintenance automatique des données à jour du programme.
  • Télécharger le programme de l'ordonnance rendue dans la boutique, ainsi que la mise à jour des données sur le site Web de l'état de paiement et expédition des commandes.

Il affiche une liste des échanges nœuds configurés avec le site

Les commandes disponibles

Les données de sortie est dans la liste
  • Nom du produit - Nom d'hôte
  • date de téléchargement - La date des dernières commandes de téléchargement à partir du site
  • Date de téléchargement réussi - La date des derniers succès commandes de téléchargement à partir du site
  • Date de sortie - La date de la finale de déchargement des marchandises sur le site, ou des informations de mise à jour sur les commandes
  • Date de atterrissage réussi - La date de la dernière déchargement réussie de produits sur le site, ou des informations de mise à jour sur les commandes
  • L'échange de marchandises - Drapeau pour télécharger le catalogue de produits du site
  • Les ordres d'échange - Signe téléchargement à partir des commandes des clients

Les commandes disponibles
  • Exécutez échange - Démarre une session de l'échange de données avec le site
  • Exécutez un échange complet - Dans ce mode, l'échange de données de catalogue de produits va écraser le répertoire existant (si elle a déjà été téléchargé sur le site)
  • Voir enregistré changements - Ouvre une vue sous forme de biens et de commandes pour lesquelles les données seront téléchargées sur le site Web
