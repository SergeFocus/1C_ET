Scambio con i siti

Consente di configurare lo scambio di dati tra il sistema informativo contabile e negozio online che supporta Protocollo tra il sistema "1C: Enterprise" e il sito web.

Caratteristiche principali:

  • Scarico dal sito web del programma catalogo di prodotti, tra cui tutte le proprietà dei beni, i prezzi e i resti delle merci nei magazzini, così come la manutenzione automatica dei dati aggiornati.
  • Scarica il programma del provvedimento emesso nel negozio, così come l'aggiornamento dei dati sul sito web dello stato di pagamento e la spedizione degli ordini.

I comandi disponibili

I dati di uscita è nella lista
  • nome del prodotto - Nome host
  • Data di scaricare - La data delle ultime ordinazioni scaricare dal sito
  • Date scaricare successo - La data dell'ultimo successo ordini scaricare dal sito
  • Data di scarico - La data dello sbarco finale dei beni sul sito, o aggiornare le informazioni sugli ordini
  • Data atterraggio di successo - La data dell'ultimo scarico di successo di beni sul sito, o aggiornare le informazioni sugli ordini
  • Lo scambio di merci - Bandiera per caricare il catalogo prodotti del sito
  • Gli ordini di conversione - Segno scaricare da ordini dei clienti

I comandi disponibili
  • Scambio Run - Avvia una sessione di scambio di dati con il sito
  • Eseguire un ampio scambio - In questa modalità lo scambio di dati del catalogo prodotti sovrascriverà la directory esistente (se è già stato caricato sul sito)
  • Mostra registrato cambiamenti - Apre la vista sotto forma di merci e gli ordini per i quali i dati saranno caricati sul sito web
