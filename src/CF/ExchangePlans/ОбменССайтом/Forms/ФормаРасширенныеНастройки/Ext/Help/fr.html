Paramètres avancés

Vous pouvez définir les paramètres du répertoire de données de téléchargement "batch"

  • Le nombre d'éléments dans une partie - Définit la quantité maximale de marchandises, d'informations sur qui peuvent être placés dans le partage de fichiers.
  • Le nombre de fois pour répéter les données d'exportation - Le nombre de tentatives d'envoi d'un fichier à un site Web avec des informations sur les produits. Si le nombre de tentatives infructueuses a prouvé échange est interrompu.
