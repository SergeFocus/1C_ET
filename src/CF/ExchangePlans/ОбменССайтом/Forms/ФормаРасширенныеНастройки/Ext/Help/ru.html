Расширенные настройки

Позволяет задать параметы "порционной" выгрузки данных каталога

  • Количество товаров в одной порции - устанавливает максимальное количество товаров, информация о которых может быть помещена в файл обмена.
  • Количество попыток повторить выгрузку данных - количество попыток отправить на сайт файл с данными о товарах. Если указанное количество попыток оказалось неудачным обмен прерывается.
