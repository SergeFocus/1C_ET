Erweiterte Einstellungen

Sie können die Parameter des "Batch" Download Datenverzeichnis festgelegt

  • Die Anzahl der Elemente in einer Portion - Legt die maximale Menge an Gütern, Informationen, auf die Sie in der Dateifreigabe platziert werden.
  • Die Anzahl der Male, um den Export von Daten wiederholen - Die Anzahl der Versuche, eine Datei zu einer Webseite mit Informationen über die Produkte zu senden. Wenn die angegebene Zahl der Versuche erfolglos geblieben Austausch unterbrochen ist.
