Impostazioni avanzate

È possibile impostare i parametri del directory di download dati "partita"

  • Il numero di elementi in una porzione - Imposta la quantità massima di beni, informazioni su cui possono essere messi in condivisione di file.
  • Il numero di volte per ripetere i dati di esportazione - Il numero di tentativi di inviare un file a un sito web con informazioni sui prodotti. Se il numero specificato di tentativi si è rivelato infruttuoso scambio è interrotto.
