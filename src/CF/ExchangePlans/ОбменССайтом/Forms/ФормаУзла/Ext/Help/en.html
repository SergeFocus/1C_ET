The communication node with the website

Allows you to configure data exchange between the accounting information system and online store that supports protocol between the system "1C: Enterprise" and website.

Key features:

  • Discharge from the program's website catalog of products, including all the properties of the goods, the prices and the remnants of the goods in warehouses, as well as automatic maintenance of the data up to date.
  • Download the program of the order issued in the shop, as well as updating the data on the website of the status of payment and shipment of orders.

basic settings
  • Communication Mode - It determines which data sets will participate in the exchange (unloading of goods and / or exchange information on request).
  • The method of data exchange - Direct communication with the site or through a directory on disk, in which files will be placed exchange.
  • Automatic exchange - The ability to turn on the automatic exchange and adjust the frequency of its execution.
  • Advanced Setup - Set the portion size uploaded to the site data and the number of retries of exchange in case of error.

Setting method of communication:
  • In a direct exchange with the site you must specify the address of the site the user name and password to access the site.
  • When using the directory sharing is necessary to specify the directory upload and file, which contains information on orders placed on the site.

Flag Exchange orders will be available only if the option is enabled, the functional Customer Orders In chapter Administration - CRM and sales.
