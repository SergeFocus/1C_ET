saytına rabitə node

Siz dəstəkləyir mühasibat informasiya sistemi və online mağaza arasında məlumat mübadiləsi konfiqurasiya imkan verir "1C: Enterprise" və internet sistemi arasında protokol.

Açar funksiyalar:

  • Bütün malların xassələri, qiymətləri və anbarlarda malların qalıqları, eləcə də bu günə qədər məlumatların avtomatik saxlanılması, o cümlədən məhsullarının, proqramın veb kataloq axıdılması.
  • Mağaza verilmiş sərəncamın proqram download, eləcə də ödəniş və sifariş daşınması statusu saytında məlumatların yenilənməsi.

əsas parametrləri
  • Communication Mode - Bu data dəstləri (xahişi ilə mal və / və ya məlumat mübadiləsi boşaldılması) müqabilində iştirak edəcək olan müəyyən edir.
  • məlumat mübadiləsi üsulu - Site və ya faylları mübadiləsi yerləşdiriləcək olan disk bir qovluq vasitəsilə birbaşa rabitə.
  • Avtomatik mübadiləsi - Qabiliyyəti avtomatik mübadiləsi yandırmaq və onun icrası tezliyi tənzimləmək üçün.
  • Advanced Setup - Site məlumat və səhv halda mübadiləsi retries sayı daşıyanlar hissəsi ölçüsü seçin.

Rabitə metodu qurulması:
  • Site birbaşa mübadiləsi Saytda site daxil olmaq üçün istifadəçi adı və parol ünvanını daxil olmalıdır.
  • Directory paylaşma istifadə edərkən saytda yerləşdirilmiş sifarişlər haqqında məlumat ehtiva edir directory yüklemek və fayl müəyyən etmək lazımdır.

Bayraq Exchange sifariş seçimi effektiv yalnız funksional mümkün olacaq Müştəri Sifarişlər Fəsildə Administration - CRM və satış.

Boşaltma mal qurulması
  • təşkilat - məhsul kataloq sahibi - Onun adından kataloq saytda dərc on təşkilat.
  • Uploaded data - Malların və \ və ya qiymətləri və qalıqları Catalog. Bu saytda alıcı tərəfindən şəkillər və seçilməsi üçün mövcud olan digər fayllar anbarların boşaldılması üçün ehtiyac göstərir. Maddə növü ilə - Bu variant təsnifat nomenklaturasının yüklemek üçün mümkündür. təsnifatın default versiyası - nomenklaturasının qrupları tərəfindən.
  • Mode upload data - Tam axıdılması, və ya son upload ildən yalnız dəyişib data.
  • Kataloqlar parametrləri malları - Siz saytda bir və ya daha çox qovluq müəyyən və malların və xidmətlərin Axıdıla seçim qurmaq lazımdır.

Kataloq artıq site daşıyanlar olsaydı məhsul kataloq olmadan yalnız qiymət müqavilələr və anbar qalıqları yüklə əhəmiyyət kəsb edir.
