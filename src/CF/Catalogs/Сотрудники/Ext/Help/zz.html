Должность, телефон, электронная почта, адрес, номер паспорта, о рождении, дата ИНН - Дополнительные сведения.

Платежная ведомость - Таблица должна быть заполнена с «заряда» сотрудников, положения в силе, если не bezsrochnye, размер и Валюта Тарифы, количество перемещенных лиц в зарплате. Цена --1 изменений. Изменение может быть один час / день / любой другой будет рассчитываться umnazhatsya "размер" период Платежная ведомость,

Тип заработной платы:

  • Пари
  • Для изменения
  • Процент от общего выполнения (предприятия)
  • Индивидуальные продажи (стоимость товаров в соответствии с документами) на сумму пропорционально
  • В процентах (корпоративные) прибыль. Он использует консолидированный валовой прибыли
  • Процент затрат на выполнения работ (исполнительный файл)
  • Для комплектации. Для его работы в актовом / razukomplektatsii
  • Услуги по сервисному номеру (), предоставляемые
  • Процент от суммы предоставленных услуг во время
  • Другие (например, файлы компенсации расходов на процесс работы или инвентаризации)

Чтобы проиллюстрировать товары продаются свой персонал использует специальную группу Клиент "Собственные сотрудники". То же самое Установка Вы можете установить Проверить цену По умолчанию продавать свои собственные сотрудники. Продажи заработной платы сотрудников в своих расчетах. Вы должны держать персонал карта Каталог пункт "клиент", который автоматически генерируется контакты.

Для учета приобретения товаров с использованием специальной группы из собственной программы работника Поставщики "Собственные сотрудники." Сотрудники для покупки товаров из расчета заработной платы, вы должны держать в контакте с сотрудниками карты входа Каталог "поставщик", который автоматически генерируется.

Склад работа в работника, Такие Склад, Исполнение или процент прибыли, из которой будут приняты во внимание при расчете заработной платы работника. Если он не имеет таких обвинений, ukazyvaat их желанию.
