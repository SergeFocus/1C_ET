Довідник Співробітники

Картотека співробітників підприємства для розрахунку заробітної плати.

Посада, Телефон, Електронна пошта, Адреса, Дані паспорта, Дата народження, ІПН - Необов'язкові реквізити.

Розрахунок зарплати - Таблиця, яку слід наповнити "нарахуваннями" даного співробітника, вказати період дії, якщо вони не безстрокова, розмір і валюту нарахувань, кількість змін при розрахунку зарплати. Ставка - 1 зміна. Зміною може бути година / доба / будь-який інший період, на який буде умнажаться "розмір" нарахування при розрахунку заробітної плати.

Види нарахувань заробітної плати:

  • Ставка
  • За зміну
  • Відсоток від суми реалізації (всього по підприємству)
  • Відсоток від суми особистих продажів (за документами Витрати Товарів)
  • Відсоток від прибутку (всього по підприємству). Використовується зведений показник валового прибутку
  • Відсоток від вартості виконаних робіт (за документами Виконання робіт)
  • За Комплектацію. За роботи при комплектації / розукомплектації
  • За надані послуги за час (за кількістю наданих послуг)
  • Відсоток від суми наданих послуг за час
  • Інше (наприклад, відшкодування за документами інвентаризації або за витрати при виконанні робіт)

Для обліку продажів товарів власним співробітникам програма використовує спеціальну групу клієнтів "Власні Співробітники". Так само в налаштуваннях можна встановити вид цін по-замовчуванню для продажу власним працівникам. Реалізація товарів співробітникам враховується при розрахунку їх заробітної плати. Для цього потрібно зберігати зв'язок картки співробітника із записом довідника "Клієнти", яка формується автоматично.

Для обліку придбання товарів у власних співробітників програма використовує спеціальну групу постачальників "Власні Співробітники". Придбання товарів у співробітників враховується при розрахунку їх заробітної плати. Для цього потрібно зберігати зв'язок картки співробітника із записом довідника "Постачальники", яка формується автоматично.

Склади, на яких працює співробітник. Такі склади, Відсоток реалізації або прибутку з яких буде врахований при розрахунку заробітної плати співробітника. Якщо у нього немає такого виду нарахувань, указиваать їх необов'язково.
