Les fabricants

Catalogue des fabricants des biens (marques, usines). Si son utilisation est incluse dans les paramètres vous permet d'analyser le chiffre d'affaires des produits et vente de divers proivzoditeley.

À ne pas confondre avec les fournisseurs de référence. Fournisseur - celui qui vend la question pour nous, et il crée un Fabricant.

Le fabricant doit être notée dans la gamme et est un groupe supplémentaire dans l'analyse de la question du trafic de données et les ventes.

Pour le répertoire des fournisseurs référence predumotrena la nomenclature d'un fabricant particulier, dont il nous vend. Il est également utile pour les cas où le fabricant est un fournisseur.

Requis:

Région (si y compris la comptabilité Régions) détermine l'emplacement du fabricant.

Adresse.Téléphone - Des informations supplémentaires sur le fabricant.

Sa formule nomenclature de code à barres. Avec Impression d'étiquettes nomenclature pour ce fournisseur peut être utilisée par son formule la formation du code à barres. Pour ce faire dans les paramètres doit être activé pour utiliser des formules.
