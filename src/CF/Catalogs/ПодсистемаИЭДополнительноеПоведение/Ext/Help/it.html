Ecco un esempio - Si supponga di dover quando si importano i dati se il valore di oggetti di scena nel "Commento", la parola "altri" oggetti di scena impostare la "Sezione" alla voce "Altre spese" delle voci di directory del flusso di cassa.

Poi nel tappo "di condotta" puntelli nome - "Commento", la condizione - "Contiene", che significa - "" altro "" e nelle tabelle di questo: puntelli tavolo - "no" nome - "articolo", Condizione - non riempiti, valore - "Altre spese" delle voci di directory del flusso di cassa.

Quando si esporta una condizione è soddisfatta nella direzione opposta - a condizione che tutte le condizioni della sezione tabellare, il campo (dal cappello) assume il valore (del tappo).
