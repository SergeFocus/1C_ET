Əlavə məlumat arayış, əsas mühasibat nomenklaturası

elementləri və arayış sənədlərin əlavə məlumat əlavə etmək üçün nəzərdə tutulmuşdur.

Əlavə rekvizit yazın

  • Line - Əlavə rekvizit string Mesaja dolu olacaq;
  • Date - Siz tarixi və vaxtı doldurmaq bilər;
  • Number - Siz bir rəqəmli dəyər etmək olacaq;
  • Boolean - dəyər YES və ya NO;
  • Guide element bağlantılar - dəyər hər hansı bir mövcud directory konfiqurasiya hər hansı bir element üçün ixtiyari istinad ola bilər;
  • Bazasında hər hansı bir sənəd ixtiyari istinad ola bilər dəyər sənədləşdirilməsi üçün Link;
  • Guide dəyəri - dəyərinin öz siyahısı seçiləcək.Birbaşa doldurulması dəyərlər olacaq siyahısında dəyər əlavə edin.

obyektlərin hər hansı "əlavə məlumat" istifadə və orada əlavə rekvizit yaratmalıdır doldurun.
