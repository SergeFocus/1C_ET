Designed to attach additional details the elements of directories and documents.

  • Boolean - the value YES or NO;
  • Reference to the directory - the value can be any reference to any element of any available reference configuration;
  • A link to the document is the value can be any reference to any document in the database;
  • The value of the directory - the value will be selected from a list of its own. Add the values ​​in the list can be filled with direct values.

For the filling should use the "additional details" for any of the objects and create an extra props there.
