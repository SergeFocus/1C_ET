Справочник "Дополнительные реквизиты", "Ключи Учета Номенклатуры"

Предназначен для прикрепления дополнительных реквизитов к элементам справочников и документам.

Тип дополнительного реквизита:

  • Строка - дополнительный реквизит будет строчным текстовым полем для заполнения;
  • Дата - можно будет заполнять датой и временем;
  • Число - Можно будет вносить числовое значение;
  • Булево - значение ДА или НЕТ;
  • Ссылка на элемент справочника - значение может быть произвольной ссылкой на любой элемент любого доступного справочника в конфигурации;
  • Ссылка на документ -значение может быть произвольной ссылкой на любой документ в базе данных;
  • Значение из справочника - значение будет выбираться из собственного списка. Добавить значения в список можно будет при непосредственном заполнении значений.

 

Для заполнения следует воспользоваться кнопкой "дополнительные реквизиты" для любого из объектов и там создать дополнительный реквизит.
