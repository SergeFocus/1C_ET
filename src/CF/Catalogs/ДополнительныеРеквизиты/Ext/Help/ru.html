Справочник "Дополнительные реквизиты"

Предназначен для прикрепления дополнительных реквизитов к элементам справочников и документам.

Для заполнения следует воспользоваться кнопкой "дополнительные реквизиты" для любого из объектов и там создать дополнительный реквизит.
