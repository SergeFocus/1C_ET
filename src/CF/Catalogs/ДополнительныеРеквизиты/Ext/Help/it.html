Directory "ulteriori dettagli", "Riferimento chiavi alla nomenclatura"

Progettato per collegare ulteriori dettagli gli elementi di elenchi e documenti.

Tipo ulteriori dettagli:

  • Line - props supplementari campo di testo linea sarà riempito;
  • Data - è possibile inserire la data e l'ora;
  • Il numero - Sarete in grado di fare un valore numerico;
  • Boolean - il valore SI o NO;
  • Riferimento alla directory - il valore può essere qualsiasi riferimento a qualsiasi elemento di una configurazione di riferimento disponibili;
  • Un link al documento è il valore può essere alcun riferimento a qualsiasi documento nel database;
  • Il valore della directory - il valore sarà selezionato da una lista propria. Aggiungere i valori nella lista può essere riempito con i valori diretti.

Per il ripieno deve utilizzare le "ulteriori dettagli" per qualsiasi oggetto e creare un puntelli in più lì.
