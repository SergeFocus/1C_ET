Riferimento ulteriori dettagli, nomenclatura contabilità chiave

Progettato per collegare ulteriori dettagli agli elementi e documenti di riferimento.

Tipo puntelli supplementari

  • Line - puntelli supplementari casella di testo stringa verrà riempito;
  • Data - è possibile inserire la data e l'ora;
  • Numero - Sarete in grado di fare un valore numerico;
  • Boolean - il valore SI o NO;
  • Collegamenti a l'elemento di guida - il valore di riferimento può essere arbitrario a qualsiasi elemento di qualsiasi configurazione della directory disponibili;
  • Collegamento a documentare il valore di riferimento può essere arbitrario a qualsiasi documento nel database;
  • Il valore della guida - valore verrà scelto proprio elenco.Aggiungere valore nell'elenco saranno i valori di riempimento diretti.

Per riempire dovrebbero usare il "ulteriori informazioni" per qualsiasi oggetto e creare oggetti di scena ulteriori lì.
