Directory "Ulteriori dettagli", "chiavi contabilità nomenclatura"

Progettato per collegare ulteriori dettagli gli elementi di libri di riferimento e documenti.

Tipo puntelli supplementari

  • Line - campo di testo puntelli supplementari si allineeranno per riempire;
  • Data - è possibile inserire la data e l'ora;
  • Numero - Sarà possibile effettuare un valore numerico;
  • Booleano - il valore di SI o NO;
  • Collegamento sull'elemento directory - valore può essere qualsiasi riferimento a qualsiasi elemento di una configurazione di riferimento disponibili;
  • Un link al documento è il valore può essere alcun riferimento a qualsiasi documento nel database;
  • Il valore dalla directory - il valore verrà selezionato da una lista di tua scelta.Aggiungere i valori nell'elenco possono essere riempiti con i valori diretti.

Per riempire dovrebbero utilizzare le "ulteriori dettagli" per qualsiasi oggetto e creare un ulteriore puntelli là.
