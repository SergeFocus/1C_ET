Verzeichnis "Zusatzinformationen", "Keys Bilanzierung Nomenklatur"

Entwickelt, um weitere Details zu den Elementen der Referenz-Bücher und Dokumente anhängen.

Geben Sie zusätzliche Requisiten

  • Linie - zusätzliche Requisiten werden Textfeld Linie zu füllen;
  • Datum - Sie können in der Datum und Uhrzeit füllen;
  • Number - Es ist möglich, einen numerischen Wert zu machen;
  • Boolean - der Wert JA oder NEIN;
  • Link auf das Verzeichnis Element - Wert kann eine beliebige Referenz auf ein beliebiges Element von jedem verfügbaren Referenzkonfiguration;
  • Ein Link zum Dokument ist der Wert jeder Verweis auf jedes Dokument in der Datenbank sein kann;
  • Der Wert aus dem Verzeichnis - der Wert wird aus einer Liste von Ihrem eigenen gewählt werden.Fügen Sie die Werte in der Liste kann mit direkten Werten gefüllt werden.

So füllen sollte es die "weitere Details" für eines der Objekte und schaffen zusätzliche Requisiten.
