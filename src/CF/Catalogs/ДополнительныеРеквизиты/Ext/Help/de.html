Verzeichnis "Mehr Details", "Keys Bezugnahmen auf die Nomenklatur"

Entwickelt, um zusätzliche Details der Elemente der Verzeichnisse und Dokumente anhängen.

Geben Sie zusätzliche Details:

  • Line - zusätzliche Requisiten Zeilentext Feld gefüllt wird;
  • Datum - Sie können in der Datum und Uhrzeit zu füllen;
  • Die Zahl - Sie werden in der Lage, um einen numerischen Wert zu machen;
  • Boolean - Der Wert YES oder NO;
  • Verweis auf das Verzeichnis - der Wert kann jede Bezugnahme auf jedem Element jeder verfügbaren Referenzkonfiguration ist;
  • Ein Link zu dem Dokument ist der Wert kann jede Bezugnahme auf jedes Dokument in der Datenbank sein;
  • Der Wert des Verzeichnis - der Wert wird aus einer Liste von seinen eigenen gewählt werden. Fügen Sie die Werte in der Liste kann mit direktem Werten gefüllt werden.

Für die Füllung sollte die "weitere Details" für eines der Objekte zu verwenden, und erstellen Sie eine zusätzliche Requisiten gibt.
