Дополнительные сведения Конструктивные содержание других каталогов и файлов.

Введите дополнительные сведения:

  • Линия - дополнительные опоры текстовое поле будет заполнено,
  • Дата - Вы можете заполнить дату и время;
  • Количество - вы можете сделать значение;
  • Boolean - Значение да или нет;
  • Каталог ссылок - значение любого компонента может быть любая ссылка на любой доступной справочной конфигурации;
  • Ссылка на файл в том, что значение может быть любая ссылка на любой файл в базе данных;
  • Значение Каталог - значение будет поступать из собственного списка вариантов. Добавление списка значений могут быть заполнены непосредственного значения.

Заполнение следует использовать для "других деталей" любого объекта и создать появляется дополнительная опора.
