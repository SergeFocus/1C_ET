Введите другие реквизиты

  • Линия - дополнительные подпорки строка текстовое поле будет заполнено;
  • Дата - вы можете заполнить дату и время;
  • Номер - вы сможете сделать значение;
  • Boolean - Значение ДА или НЕТ;
  • В сочетании с направляющим элементом - значение любого компонента может быть любым произвольным ссылка каталог доступных конфигураций;
  • Привязанные к значениям указанного файла может быть любой базой данных любого документа;
  • Значение Guide - значение будет выбрано из списка.Добавить значение в список будет заполнен непосредственно значение.

Она должна быть использована для того, чтобы заполнить "Дополнительная информация" любого объекта, и создать больше реквизита там.
