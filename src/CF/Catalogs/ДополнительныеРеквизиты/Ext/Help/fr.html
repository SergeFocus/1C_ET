Répertoire "plus de détails", "référence clés pour la nomenclature"

Conçu pour fixer les détails supplémentaires, les éléments de répertoires et de documents.

Tapez des détails supplémentaires:

  • Line - accessoires supplémentaires champ de texte en ligne sera remplie;
  • Date - vous pouvez remplir la date et l'heure;
  • Le nombre - Vous serez en mesure de faire une valeur numérique;
  • Boolean - la valeur OUI ou NON;
  • Référence au répertoire - la valeur peut être toute référence à un élément de toute configuration de référence est disponible;
  • Un lien vers le document est la valeur peut être toute référence à un document dans la base de données;
  • La valeur du répertoire - la valeur sera choisi parmi une liste de ses propres. Ajouter les valeurs de la liste peut être rempli avec des valeurs directes.

Pour la garniture devraient utiliser les "détails additionnels" pour l'un des objets et de créer un des accessoires supplémentaires là.
