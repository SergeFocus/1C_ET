Répertoire "des détails supplémentaires", "Keys nomenclature comptable"

Conçu pour fixer des détails supplémentaires les éléments d'ouvrages de référence et des documents.

Tapez accessoires supplémentaires

  • Line - accessoires supplémentaires seront alignées champ de texte à remplir;
  • Date - vous pouvez remplir la date et l'heure;
  • Nombre - Il sera possible de faire une valeur numérique;
  • Boolean - la valeur de OUI ou NON;
  • Lien vers l'élément de répertoire - valeur peut être toute référence à un élément d'une configuration de référence disponibles;
  • Un lien vers le document est la valeur peut être toute référence à un document dans la base de données;
  • La valeur du répertoire - la valeur sera choisi parmi une liste de vos propres.Ajouter les valeurs de la liste peut être rempli avec des valeurs directes.

Pour remplir doivent utiliser les «détails supplémentaires» pour l'un des objets et de créer un des accessoires supplémentaires là-bas.
