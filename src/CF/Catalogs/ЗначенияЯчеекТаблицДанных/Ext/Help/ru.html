Справочник значений ячеек таблиц данных

В конфигурации предусмотрен механизм хранения и анализа произвольных таблиц данных.

Это специальные документы, которые могут быть использованы по своему усмотрению пользователями для хранения чего либо.

Документ заполняется в том числе за счет значений произвольного характера. Которые хранятся в этом справочнике.

Тип значения задается (Строка, Дата, Число, Булево (да/нет), Ссылка на элемент справочника базы данных, Ссылка на документ базы данных, Значение из справочника).

Значение из справочника - значение из "личного" виртуального справочника под элемент, реализованного за счет дополнительных реквизитов.
