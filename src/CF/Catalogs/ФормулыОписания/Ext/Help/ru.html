Справочник формул описания и формирования

 Предназначен для хранения формул формирования:

  • Формулы авто-наименования номенклатуры для номенклатурных групп
  • Формулы преобразования цен (напр. закупочной цены в цену реализации)
  • Формулы формирования текста этикетки
  • Формулы формирования штрих-кода номенклатуры
  • Формулы для произвольных надписей на формах

Справочник является дополнительным (его можно найти в разделе "дополнительные справочники окна настройки учета, программы).

Формула

Пример формулы текста этикетки:

СокрЛП(Цел(Номенклатура.Цена) + 1) + " " + Лев(Номенклатура.НоменклатурнаяГруппа.Код, 2) + Лев(Номенклатура.Производитель.Код, 2) + "x" + Формат(Цена, "ЧЦ=9; ЧДЦ=0")

ЛьготнаяЦена - дополнительный параметр при использовании механизма льготных цен продаже с количеством равным или выше планки.

Формулы должны быть написаны языком программирования 1С.
