Directorul "Descrierea"

Proiectat pentru formule de formare de stocare:

  • Formula auto-nume Nomenclatură pentru Nomenclatura de grup
  • Prețurile formula de transformare (de exemplu, prețul de achiziție în prețul de vânzare)
  • Textul etichete formula de formare
  • Formarea Formula Bar Code Nomenclatură

De referință este opțională (poate fi găsit în secțiunea "Reference suplimentare Fereastra "Setări de contabilitate, progammy").

Formulă

Textul etichetei formulă exemplu:

SokrLP (Int (Nomenklatura.Tsena) + 1) + "" + Leo (Номенклатура.НоменклатурнаяГруппа.Код, 2) + stânga (Номенклатура.Производитель.Код, 2) + "x" + Format (preț, "CC = 9; CHDTS = '0 ')

LgotnayaTsena - Un parametru suplimentar în folosind mecanismul de prețuri preferențiale vânzare pentru suma egală sau mai mare ornamente.

Formulele trebuie scrise 1C limbaj de programare.
