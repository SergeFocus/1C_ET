참조 수식 설명

식의 형성의 저장을위한 설계 :

  • 포뮬러 원 자동차 이름 명명법 에 명명법 그룹
  • 공식 전환 가격 (예. 판매 가격에 구입 가격)
  • 텍스트 레이블의 공식 형성
  • 화학식 형성 바코드 명명법
  • 양식에 임의의 레이블에 대한 공식

참조는 선택 사항입니다 (이것은 "추가 참조에서 찾을 수 있습니다 창 "설정 회계, proscale").

공식

예 수식 텍스트 레이블 :

SokrLP (INT (Nomenklatura.Tsena) + 1) + ""+ 레오 (Nomenklatura.NomenklaturnayaGruppa.Kod, 2) + 레오 (Nomenklatura.Proizvoditel.Kod, 2) + "X"+ 형식 (VAT, "CC = 9; CHDTS = 0 ")

LgotnayaTsena -에 추가 파라미터 우선 가격의 메커니즘을 사용하여 또는 판 위의 동일한 양에 판매.

수식 작성해야 1C 프로그래밍 언어.
