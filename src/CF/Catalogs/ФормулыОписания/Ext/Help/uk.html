Довідник Формули Описи

Призначений для зберігання формул формування:

  • Формули авто-найменування Номенклатури для Номенклатурних Груп
  • Формули перетворення цін (напр. Закупівельної ціни в ціну реалізації)
  • Формули формування Тексту етикетки
  • Формули формування Штрих-коду номенклатури
  • Формули для довільних написів на формах

Довідник є додатковим (його можна знайти в розділі "додаткові довідники вікна "Налаштування Обліку, прогамм").

Формула

Приклад формули тексту етикетки:

СокрЛП (Цел (Номенклатура.Цена) + 1) + "" + Лев (Номенклатура.НоменклатурнаяГруппа.Код, 2) + Лев (Номенклатура.Проізводітель.Код, 2) + "x" + Формат (Ціна, "ЧЦ = 9; ЧДЦ = 0 ")

ЛьготнаяЦена - Додатковий параметр при використанні механізму пільгових цін продажу з кількістю рівним або вище планки.

Формули повинні бути написані мовою програмування 1С.
