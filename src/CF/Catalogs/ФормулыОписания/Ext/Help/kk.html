Анықтамалық Формула Сипаттама

Формулалардың қалыптастыру сақтау үшiн жобаланған:

  • Formula One авто-атаулар Номенклатурасы қарай Номенклатурасы топтар
  • Формула түрлендіру бағасы (мысалы. Сату бағасы сатып алу бағасы)
  • Мәтін белгілерінің Формула қалыптастыру
  • Формула қалыптастыру Barcode номенклатурасы
  • нысандары бойынша ерікті жапсырмалар үшін формулалар

Анықтамалық міндетті емес (ол «Қосымша Reference табуға болады терезе «Параметрлер Есеп, proscale»).

Формула

Мысал формуласы мәтін жапсырмалар:

SokrLP (Int (Nomenklatura.Tsena) +1) + «» + Лео (Nomenklatura.NomenklaturnayaGruppa.Kod, 2) + Leo (Nomenklatura.Proizvoditel.Kod, 2) + «х» + Format (ҚҚС, «CC = 9; CHDTS = 0 «)

LgotnayaTsena - Қосымша параметр жеңілдікті баға тетігін пайдалана отырып, табаққа тең немесе одан жоғары сомаға сату.

Формулалар жазылған болуы тиіс 1С бағдарламалау тілі.
