Reference Formula Description

Designed for storage of formation of formulas:

  • Formula One auto-names Nomenclature to Nomenclature groups
  • Formula conversion price (eg. The purchase price in the selling price)
  • Formula formation of the text labels
  • Formula formation Barcode nomenclature
  • The formulas for the arbitrary labels on forms

Reference is optional (it can be found in the "Additional Reference window "Settings Accounting, proscale").

Formula

Example formula text labels:

SokrLP (Int (Nomenklatura.Tsena) +1) + "" + Leo (Nomenklatura.NomenklaturnayaGruppa.Kod, 2) + Leo (Nomenklatura.Proizvoditel.Kod, 2) + "x" + Format (VAT, "CC = 9; CHDTS = 0 ")

LgotnayaTsena - An additional parameter in the using the mechanism of preferential prices sale to the amount equal to or above the plate.

Formulas should be written 1C programming language.
