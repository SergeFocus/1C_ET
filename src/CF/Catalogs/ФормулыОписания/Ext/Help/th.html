สูตรอ้างอิงที่กล่าวถึงและการสร้าง

สูตรที่ออกแบบมาเพื่อเก็บของ:

  • สูตรหนึ่งชื่ออัตโนมัติศัพท์เฉพาะ สำหรับกลุ่มศัพท์
  • ราคาแปลงสูตร (เช่น. ราคาซื้อในการก่อให้ราคา)
  • ป้ายข้อความการสร้างสูตร
  • บาร์โค้ดก่อสูตรศัพท์เฉพาะ
  • สูตรสำหรับป้ายชื่อโดยพลการในรูปแบบ

อ้างอิงเป็นตัวเลือก (มันสามารถพบได้ใน "คำแนะนำเพิ่มเติมหน้าต่างการตั้งค่าโปรแกรมบัญชี

SokrLP (A (Nomenklatura.Tsena) + 1) + "" + สิงห์ (Nomenklatura.NomenklaturnayaHruppa.Kod 2) + สิงห์ (Nomenklatura.Proizvoditel.Kod 2) + 'X' + รูปแบบ (VAT "CC = 9; CHDTS = 0 ")

LhotnayaTsena - เลือกที่ใช้กลไกของราคาพิเศษ ได้จากการขายจำนวนเท่ากันหรือมากกว่าของแผ่นเปลือกโลก
