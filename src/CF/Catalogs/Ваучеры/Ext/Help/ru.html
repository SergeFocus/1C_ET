Справочник Ваучеры

 Включается и регулируются в настройках конфигурации.

Справочник позволяет добавить в учет ваучеры нескольких типов:

  • Идентификаторы контрагентов (личные карты)
  • Карта с заданным процентом скидки клиента
  • Карта с заданной суммой скидки клиент
  • Карта накопительных "баллов" клиента для операций реализации
  • Карта, погашающая сумму при реализации ("подарочная") с определенным номиналом

 

Для большинства организационных операций с ваучерами предусмотрена специальная обработка.

Ваучеры учитываются в виде отдельных элементов справочника.

Однако могут быть связаны с элементами номенклатуры, когда следует учитывать фактические операции по приобретению, перемещению, выдачи и приёму ваучеров (в качестве документов строгой отчетности). Для такой номенклатуры желательно выделить отдельную папку. В какой то степени ваучеры являются товаром при приобретении от эммитента (производителя, поставщика карт) или при реализации клиенту (например, "подарочные карты").

Этот физический учет ваучеров можно отключить в настройках.

Ваучеры могут быть ограничены периодом действия, применяться только к отобранным группам номенклатуры или номенклатурным группам, ограничены для складов реализации.

Ваучеры могут определяться собственным штрих-кодом.

Для изучения истории ваучера служит отдельный отчет.
