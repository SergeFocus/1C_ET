Каталог ваучеров

В конфигурации установка оказывается Правила,

Каталог позволяет добавлять несколько типов учетных документов:

  • Идентификатор контрагента (личная карточка)
  • Скидка Заказчик заранее доля карт
  • Заданное количество карты клиента и скидки
  • Хранения Карта "точка" для выполнения операций клиента
  • Карты, подлежащие погашению в объеме продаж ("подарок") с определенной номинальной стоимости

Для большинства организаций работают оснащены купонов Специальная обработка,

Ваучеры записываются в отдельный каталог проекта.

Тем не менее, могут быть связаны с элементом НоменклатураКогда он должен принимать во внимание приобретения, передачи, выдачи и принял верительные грамоты фактической работы (такие, как документов строгой отчетности). По этой номенклатуре, желательно, чтобы выбрать папку. Насколько купоны для покупки товаров из emmitenta (производителя, поставщика карты) или реализовать клиента (например, "Подарочная карта"), когда.

Это физическое бухгалтерский документ может быть отключена в настройках.

Документы могут быть ограничены периодом эксплуатации, применяются только к выбранной группе или номенклатуры Названы группыОграниченное Склад Реализация.

Купоны могут быть определены по своей штрих-кода.
