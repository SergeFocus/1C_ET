Referenz Arten von Veranstaltungen

Es muss in den Einstellungen enthalten Buchhaltung Ereignisse (CRM) werden.

Spravonik können Sie Vorlagen für Dokumente zu erstellenevent-RegistrierungDieses Muster beschreibt im Detail die Veranstaltung und bittet ihn, die ursprünglichen Werte.

Directory können Sie ihre Arten von Veranstaltungen mit einer einzigartigen Reihe von Bedingungen zu spezifizieren.

Zum Beispiel zeigt die Art der Veranstaltung ", ein Telefonanruf an den Kunden", dass:

  • Bei einem Thema (Call-Thema)
  • Verwenden Telefon (Telefonnummer des Kunden)
  • beteiligtkunde (Ein Verweis auf das Element der Kundenreferenz)
  • beteiligtmitarbeiter (Calling einen Mitarbeiter unseres Unternehmens)
  • Geben Sie das Datum der (die Uhrzeit des Anrufs)
  • Die Dauer (Gesprächsdauer)
  • Fasst man die Ergebnisse (zum Beispiel der schriftlichen Kunden Reaktion auf ein bestimmtes Thema)

Usw.und dergleichen.

Auf der "neuen Event-Vorlage" und "Template-Tabellen" können sofort bereit sein, um Werte für diese Art von Veranstaltung.

Bei der Registrierung neuer Ereignisse mit dieser Art werden die vordefinierten Daten (für den Benutzer sichtbar) blockiert und füllt beteiligt sein.

So zum Beispiel alle Arten von Veranstaltungen "einen Telefonanruf von einem Client Ivanov" mit etablierten Mitarbeiter Ivanov, wird sofort standardmäßig auf sie registriert werden.
