Ссылка Тип события

Он должен быть включен бухгалтерские события (CRM) в настройках.

Spravonik позволяет создавать шаблоны документовзаписьЭтот шаблон подробно описан в случае и установить начальное значение.

Это руководство позволяет использовать уникальный набор обстоятельств, при которых указанный тип события.

Например, тип события, "вызов клиента", он сказал:

  • Места тема (Тема вызовов)
  • Использование телефона (т.е. номер телефона клиента)
  • принимать участиеклиент (Ссылка на клиентов членов руководство)
  • принимать участиенаемный рабочий (Наши сотрудники звонящего по телефону)
  • ПОКАЗАНИЯ (CALL) Дата
  • Длительность (вызов времени)
  • Результаты ФРС (например, письменный ответ клиенту на ту же тему)

И многое другоеи тому подобное.

Шаблон этикетки "событие" и "шаблон таблицы", готовые немедленно привести это ценное событие.

При регистрации нового развития этого типа будет активирована (пользователи могут видеть,) модуль и заполнение предопределенных данных.

Например, данный тип персонала Сергея Иванова, "Иванова от клиента по телефону" всех событий он будет немедленно зарегистрированы по умолчанию.
