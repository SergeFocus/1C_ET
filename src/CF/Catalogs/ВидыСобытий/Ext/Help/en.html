Reference types of events

It must be included accounting events (CRM) in the settings.

Spravonik allows you to create templates for documentsevent registrationThis pattern describes in detail the event and asks him the original values.

Directory allows you to specify their types of events with a unique set of conditions.

For example, the type of event, "a telephone call to the client" indicates that:

  • Given a topic (call topic)
  • Use phone (phone number of the customer)
  • Participatesclient (A reference to the element of customer reference)
  • Participatesemployee (Calling an employee of our company)
  • Specify the date of the (time of the call)
  • The duration (call duration)
  • Summing up the results (for example, written customer response to a given topic)

Etc.etc.

On the "new event template" and "template tables" can be immediately ready to make values ​​for this type of event.

When registering new events with this kind will be involved (visible to the user) blocks and fills the predefined data.

So, for example, all kinds of events "a phone call from a client Ivanov" with established employee Ivanov, will be immediately registered by default on it.
