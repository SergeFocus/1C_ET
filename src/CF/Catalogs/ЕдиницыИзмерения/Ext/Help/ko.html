단위의 디렉토리

추가 유닛이 설명을 사용하여 명명법.

단위 명명법 만약 기본 단위에 대하여 계수를 통해 서로 접속 될 수있는 두 개 이상의 상이한 단위의 범위를 고려할 필요가있을 때 우리는 단지 필요하다. 당신은 당신의 계정의 필요에 따라, 기본 단위 무엇을 취할 것입니다.

명칭 (예 : "일"등) 측정의 하나의 단위를 사용하는 경우, 측정 단위를 생성하지 않는 것이 최선이며, 명명법에 인쇄 카드의 이름을 쓸 "이름, 조각." 양호한 판면을 형성한다.

측정 단위의 목록을 인쇄, 당신은 카드 제품을 인쇄 할 수 있습니다.
