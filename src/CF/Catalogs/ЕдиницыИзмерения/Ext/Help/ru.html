Справочник Единицы измерения

 Служит описанием для дополнительных единиц измерения номенклатуры.

Единицы измерения номенклатуры нужны только для случаев, когда номенклатуру нужно учитывать в двух и более различных единицах измерения, которые можно связать между собой посредством коэффициента, относительно основных единиц. Что Вы будете брать за основную единицу, зависит от потребностей Вашего учета.

Если для номенклатуры используется только одна единица измерения (например "штука"), то лучше не создавать единицу измерения, а в наименовании для печати в карточке номенклатуры написать "Наименование, шт." для удовлетворительного вида печатных форм.

Для печати перечня единиц измерения можно распечатать карточку товара.
