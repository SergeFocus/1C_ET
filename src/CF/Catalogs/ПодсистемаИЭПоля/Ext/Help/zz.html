Обратитесь к разделу "шаблона сайта файл"

Цель этого каталога будет использоваться для хранения списка файлов может быть поля ШаблонИмпорт-экспорт бизнес для обработки документов.

Существует большая группа предопределенных полей для их поведения во время погрузки и разгрузки процедур. В частности, загрузки и разгрузки таблицы "Продукт" и "клиентов банка" биржах.

Имя - Авторизация база строгие опоры, которая соответствует имени опорной конструкции 1С. Если вы используете имя с точки (Nomenklatura.Kod), программа будет пытаться удалить Uninstall контента subrekvizita (номер товара). Если это поле используется в шаблоне, в качестве ключа, данные загружаются в программу будет пытаться определить subrekvizitu (код) значения объекта (номенклатура).

Примечание Основным описание поля шаблона.

Поле Тип (String s, число N, дата D) в формате DBF, используется для определения типа файлов баз данных столбца. Длина ИТочность - Определите длину поля DBF. Точность показали особенно сильное поле - номер.

Гольф Некоторые важные Определяет разгрузку и погрузку фиксированной стоимости, независимо от данных содержимого.

По умолчанию, первое поле Поле указывает имя файла базы данных. В обмен DBF будет обрезано до 10 символов, и переведены на латынь.

Флаг Владелец сайта принадлежит таблице объектов Это позволяет указать файлы или удаления поля заголовка разрешения гид нагрузку и часть таблицы.

Каталог шаблонов в области импорта и экспорта иностранной валюты генерируется. Когда шаблон переключения, исполнял те же поля.
