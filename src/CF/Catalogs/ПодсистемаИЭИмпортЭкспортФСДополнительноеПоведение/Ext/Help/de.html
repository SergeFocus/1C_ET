Verzeichnis "Zusätzliche Verhaltensregeln für den Import-und Export-Vorlagen"

Verwenden Sie die Verhaltensregeln - Hier können Sie die Festabhängige Regeln zu organisieren, um eine Import füllen abhängig von den Daten oder Code versehen auszuführen.

Verhaltensregeln in der Tabelle der Bestellung angegeben und in einem separaten Verzeichnis gespeichert. Sie können in jeder Anzahl von Mustern verwendet werden.

Die Karte muss die Verhaltensregeln festlegen Nennen Requisiten (Für Importe) oder der Name des Feldes / Spalte (für den Export) Bedingung Verhalten (Importe) und Wert.

In der Tabelle Teil - die tatsächliche Übertragungsverhalten zu importieren oder zu exportieren Bedingungen.

Hier ist ein Beispiel - Angenommen, Sie wollen, wenn Sie Daten importieren, wenn der Wert der Requisiten in dem "Kommentar" ist das Wort "andere" gesetzt Requisiten "Artikel" auf "Sonstige Aufwendungen" aus dem Verzeichnis der Artikel Cashflow.

Dann hat "Verhalten" Namen Requisiten - "Kommentar"-Bedingung - "Enthält", was bedeutet, - "" andere "" und im Tabellenteil: Diese Tabelle Requisiten - "nein" Namen - "Artikel", Zustand - nicht gefüllt, Bedeutung - "Sonstige Aufwendungen" aus dem Verzeichnis der Artikel Cashflow.
