Abbonamenti riferimento a fatti

La guida permette di condizioni avanzate di oggetti del database degli eventi di sottoscrizione:

  • registrazione elemento della directory o del documento;
  • del documento;
  • contrassegno elemento rimozione della directory o del documento;
  • scoperta (o tentare di creare una nuova conoscenza con directory esistente o elemento del documento);
  • chiusura.

Condizioni di Sottoscrizione possono applicarsi a tutti gli oggetti dello stesso tipo, o oggetti specifici nel database.

Il programma può impostare condizioni come 1C lingua condizionale, che chiarirà la necessità di risposta all'evento.È possibile fare riferimento all'oggetto attraverso. "SsylkaNaOb'ekt

Per esempio condizioniSsylkaNaOb'ekt.Proveden = TRUE

Fornito i seguenti metodi di risposta all'evento:

  • Un messaggio
  • Scrivendo nel file Journal
  • L'invio di e-mail
  • eseguire il programma

Possono essere date al destinatario del numero di utenti.
