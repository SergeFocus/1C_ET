Directory Subscriptions to events

Directory allows you to specify complex conditions subscribe to events with database objects:

  • recording element directory or document;
  • holding a document;
  • mark on the removal of the member handbook or document;
  • discovery (or attempt to create a new acquaintance with an existing directory or a document element);
  • closing.

Terms of subscription can apply to all objects of the same type, or specific database objects.

In the program, you can set conditions such conditional language 1C, which will clarify the need to react to an event. You can refer to the object through. "SsylkaNaObekt".

For example the condition, SsylkaNaObekt.Proveden = TRUE

The following methods of the reaction to the event:

  • Report user
  • Record the log file
  • Sending email
  • Run the program
