Directory Előfizetés események

Directory lehetővé teszi, hogy meghatározza az összetett feltételek feliratkozás események adatbázis-objektumok:

  • címtárelem irat vagy dokumentum;
  • kezében egy dokumentumot;
  • mark eltávolítása könyv vagy dokumentum elem;
  • felfedezés (vagy kísérel meg létrehozni egy új ismeretség egy meglévő könyvtár vagy dokumentum elem);
  • záró.

Előfizetési feltételek kérheti minden tárgy azonos típusú, vagy meghatározott adatbázis-objektumokat.

Ami a program, akkor adjon meg egy feltételes nyelv 1C, amely tisztázza, hogy szükség van, hogy reagáljon az eseményekre.Elérheti a tárgy keresztül. "SsylkaNaObekt

Például a feltétellel,SsylkaNaObekt.Proveden = TRUE

Az alábbi módon válaszoljon az eseményt:

  • Az üzenet a felhasználó
  • Az írás a lapban fájl
  • E-mail küldése
  • A program futtatásához

részesülők üzeneteket a felhasználók száma is beállítható.
