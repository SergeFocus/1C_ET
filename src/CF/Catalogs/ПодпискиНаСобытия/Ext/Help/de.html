Verzeichnis der Ereignisabonnements

Verzeichnis können Sie komplexe Bedingungen zu definieren, um Ereignisse mit Datenbankobjekten abonnieren:

  • verzeichnis Artikeldatensatz oder ein Dokument;
  • Halten eines Dokuments;
  • Mark Entfernung Buch oder Dokument-Element;
  • Entdeckung (oder Versuch, eine neue Bekanntschaft mit einem bestehenden Verzeichnis oder Dokument-Element zu erstellen);
  • Schließen.

Abonnementbedingungen kann auf alle Objekte des gleichen Typs oder bestimmte Datenbankobjekte anwenden.

In Bezug auf das Programm können Sie eine bedingte Sprache 1C angeben, die die Notwendigkeit, auf Ereignisse zu reagieren klären.Sie können das Objekt durch zugreifen zu können. "SsylkaNaObekt

Beispielsweise die Bedingung,SsylkaNaObekt.Proveden = TRUE

Die folgenden Möglichkeiten, auf das Ereignis zu reagieren:

  • Die Nachricht an den Benutzer
  • Schreiben in der Journal-Datei
  • Senden E-Mail
  • Führen Sie das Programm

Empfänger von Nachrichten von der Anzahl der Benutzer festgelegt werden.
