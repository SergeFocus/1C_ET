Справочник Хранилища денег

 Предназначен для учета денег в различных хранилищах (кассах, расчетных счетах)

Для каждого хранилища следует указать: Форму оплаты, Валюту (если ведется учет по валютам) и другие реквизиты.

Рекомендуется в наименовании хранилища денег так же упоминать валюту и форму оплаты.

Хранилище пополнения позволяет автоматически списать деньги из другого хранилища при отсутствии необходимой суммы в этом хранилище. При этом цепочка из хранилищ пополнения НЕ работает. Хранилище пополнения может быть только с такой же валютой. Механизм пополнения работает при расходовании в документе Движения денег и выбытии в документе поступления товара.
