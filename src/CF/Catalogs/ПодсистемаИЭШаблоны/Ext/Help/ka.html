Reference "ფაილი თარგები"

ნიმუშები იმპორტ-ექსპორტის ოპერაციების ფაილი ასევე ხელმისაწვდომია "Setup და ადმინისტრირება" მენიუში მარცხენა.თარგები ფოლდერი "წინასწარგანსაზღვრული" ინახება თარგები, დადგენილი პროგრამისტი.

ცვლილებები ამ თარგების შეუძლია მხოლოდ მიერ მომხმარებლების პროგრამირების ავტორიზაციის სისტემის, როგორც პროგრამის საქციელი, როდესაც იმპორტიორი და ექსპორტიორი არასტანდარტული მათთვის.

ფანჯრიდან სიაში თარგები ხელმისაწვდომი უნიკალური სისტემის მახასიათებლები:

ჩამონათვალი გუნდებიდა-E (Import-Export) შემდეგ სისტემა გამოჩნდება სიაში ბარათები და ყველა სტანდარტული სახელმძღვანელოები და სისტემის დოკუმენტაცია.სია ბრძანებები საშუალებას გაძლევთ ატვირთვა და ჩამოტვირთვის ნუსხა ან მონაცემთა ბაზის ელემენტს.თუ შესაფერისი template, მხოლოდ ერთი შეირჩევა ავტომატურად სისტემის მიერ.

მაგრამ დირექტორია სია "ფაილი თარგები იმპორტის და ექსპორტის" სიაში ბრძანებები და ხელმისაწვდომია დამატებითი E:

განსაზღვრული ოპერაციების იმპორტისა და ექსპორტის - დაიწყოს გაცვლა უნიკალური ალგორითმები, სისტემა ჩაუყარა დეველოპერები.

საქმე ასეთი პრობლემა მოითხოვს პირადი სასწავლო და არ არის აღწერილი ამ მითითება

პაკეტის სანაცვლოდ კომპლექტი ფაილი- როდესაც თქვენ უნდა გადაწეროთ და განიტვირთოს მონაცემთა ბაზაში ობიექტების გამოყენებით კომბინაცია ორი ან მეტი ფაილი.

მაგალითად, ერთ-ერთი იმ დოკუმენტის header და სხვა ორი - იმ ნაწილების დოკუმენტი მაგიდასთან.

ეს გადამუშავება აუცილებელია მიუთითოთ ტიპის ობიექტი გაცვლა ან საკუთარი გაცვლა დაწესებულებაში, მიუთითოთ იმპორტ-ექსპორტის ფაილი თარგი header ობიექტი, მისამართი ფაილი მას.თქვენ შეგიძლიათ მიუთითოთ პერიოდში (იმ შემთხვევაში, თუ დოკუმენტები).

მაშინ შევსება მაგიდა, რაც მიუთითებს, მაგიდის სახელი ობიექტი, როგორც არის მითითებული configurator, template აღწერა ფაილი ამ მაგიდასთან შესაბამისი მისამართი ფაილი.

სია გაზიარება ამოცანების - იმისათვის, რომ აწარმოებს იმპორტ-ექსპორტის ოპერაციების სია თანმიმდევრობით.

ეს მკურნალობა შეიძლება შეავსოს ცხრილი სია კომპლექტი template ფაილი და მათი მისამართები, მითითებული მიმართულებით "იმპორტი - დიახ, ექსპორტი - არა."

მიუთითებს გაცვლა და მაგიდის სახელი ობიექტი, საჭიროების შემთხვევაში.ამის შემდეგ დაიწყება სია სავალუტო ოპერაციები.თქვენ შეგიძლიათ მიუთითოთ შერჩევა ვადა ოპერაციების დოკუმენტები.

მკურნალობაგაცვლითი ფულადი დოკუმენტები (როგორც "ნაღდი") უზრუნველყოფს უნარი ოპერაციების იმპორტი სია () და ექსპორტის () სია ამოცანები.გახსნაზე დამუშავება ირჩევს სიაში შესაფერისი თარგები (გაცვლის ფულადი დოკუმენტები).

სხვა ბრძანება ფაილი template სია

ჟურნალი იმპორტი ექსპორტიეს ჟურნალი შეიცავს ინფორმაციას წარმოებული სისტემის საექსპორტო და საიმპორტო ოპერაციები, თუ ისინი "ჟურნალის" განსაზღვრული თარგის ფაილი.

მაგალითად, ყველა იმპორტის ოპერაციების დროშა "ჩამოტვირთოთ ყველა ფაილი ამ საქაღალდეში" ჟურნალი ჩაატარა, რათა თავიდან ავიცილოთ ხელახალი ჩამოტვირთვის ფაილი.

ჩამოტვირთვა თარგები / Templates Saveეს ოპერაცია საშუალებას გაცვლა template იმპორტისა და ექსპორტის ფაილი სხვა მომხმარებლებს სისტემაში ან სისტემის დეველოპერები.

აღწერილობა ფაილი ინახება სპეციალურ ფორმატში "ieUNFSH" და იმპორტიორი დასინქრონებულია სახელი template.არა ყველა თარგი ვარიანტი შეიძლება გადაეცეს მიერ ჩამოტვირთვის თარგები.იმ პარამეტრებს, რომლებიც კონკრეტულ მონაცემთა ბაზაში, თქვენ უნდა იყოს ხელით მითითებული.

ბარათის იმპორტი ექსპორტი ფაილი template.

ეს არის ყველაზე რთული სისტემა, ობიექტი.მოხერხებულობის, ეს უნდა იყოს შევსებული ზემოდან მარჯვნიდან მარცხნივ.

მაგალითად, tick ამ cap "საექსპორტო მხოლოდ" ან "მხოლოდ იმპორტი" დაეხმარება დამალვა თარგის პარამეტრები, რომლებიც არ გამოიყენება ამ შემთხვევაში.

კალენდარი თარგი იგი განსაზღვრავს ძირითადი ამოცანა ამ ნიმუში, გარდა "ნებისმიერი ვერსია" პარამეტრი.ჩვენ გირჩევთ აირჩიოს ფორმის შაბლონი რომელიც საუკეთესო შეესაბამება თქვენი ამოცანა.

ამასთან დაკავშირებით, არჩევანი კონკრეტული ტიპის template ხელმისაწვდომი იქნება იმ ან სხვა პარამეტრების შაბლონის.ყოველი პარამეტრი აღწერილია ქვემოთ.

მაგიდა სახელი - ვარიანტი დაზუსტება მაგიდები დირექტორიაში ან დოკუმენტი (მაგალითად, ის დასახელებული configurator), რომელიც ექვემდებარება იმპორტ-ექსპორტის ოპერაციები.
