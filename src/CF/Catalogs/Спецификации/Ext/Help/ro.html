Format descriu formarea de produse pentru utilizare în configurație de date auto-umplere.

Reprezintă o oportunitate de a umple rezultatelor de configurare (produse), care sunt materiale de pensionari efectuate atunci când finalizarea lucrărilor, costurile suplimentare de bani de către gruparea implicat cu fonduri proprii.

Spre deosebire de un set de configurație: La punerea în aplicare un set de scris de pe intreaga gama de compoziția sa. Atunci când sunt echipate cu materiale scrise și de pe format o nouă instanță a produs stoc. Care este depozitat separat și este listată separat în aplicare, dar pot fi demontate.
