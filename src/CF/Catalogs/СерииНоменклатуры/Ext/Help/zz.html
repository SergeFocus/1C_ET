Название Артикул серии

Каталог, каждый пакет Номенклатура Вы можете перечислить все его серии.

Мы можем использовать серийный счета номенклатура количественный и серии - Сфера применения и производителе 1 единицу элемент уникального серийного номера / кода /Штрих,

Есть "полный" и "Простой" режим записи серии названы.

Прием, передача, сбыт (Обратно) название - должны в полной мере учитывать ряд докладов о всех событиях. Таким образом, можно узнать полный путь к определенному продукту, принимая во внимание сериал.

В серии "Простой режим" достаточно указать только имя приобретения (и использовать поиск). Таким образом, вы можете найти только где и как, чтобы выиграть эту серию. В этом режиме, информация не записывается, который заключается в реализации несколько серий.

Номенклатура, потому что это может установить "строгий" сериал, т.е., тот, который должен быть указан для серии.

Если сам серийный номер или код продукта выступает в качестве штрих-кода. Они могут быть прочитаны с помощью сканера, чтобы создать новую серию карт.

В настройках можно задать поведение, где найти штрих-код будет определять серию и серийный номер / код продукта.

После использования файл не может быть переименован ряд.

Есть некоторые ограничения:

Для продукции сериала считают не запускается автоматически рассчитать порог скидка сумму.

Товары с диапазоне от 1 общее число записей. Таким образом, программа проверяет остатки называя в целом.

Если серийный считается элементом группы, выберите группу документов, таких номенклатурных, все его оборудование будут загружены в файлы таблиц, чтобы иметь возможность задать все стандартные компоненты серии.
