Справочник СЕРИИ Номенклатуры

Справочник, в котором для каждой серийной номенклатуры можно перечислить все ее серии.

У серийной номенклатуры может вестись учет как количественный, так и по серии - на 1 единицу номенклатуры один элемент с уникальным серийным номером/кодом производителя/штрих-кодом.

Номенклатуру так же можно установить "строго серийной", то есть такой, для которой серии указывать обязательно.

Если серийный номер или код продукта сами выступают, как штрих-код. Их можно прочитать со сканера в карточку при создании новой серии.

В настройках программы можно установить поведение, при котором поиск по штрих-код будет определять серию и по серийному номеру/коду продукта.

После использовании в документах, имя серии измененить нельзя.

Существуют ограничения:

Для товаров с серийным учетом не работает автоматический расчет скидки по порогу количества.

Товары с серией всегда регистрируются с количеством 1. Поэтому остатки программа проверяет по номенклатуре вцелом.

Если номенклатура с серийным учетом является элементом набора, то при выборе такого набора в документах, вся его комплектация будет загружена в таблицу документа, чтобы была возможность указать серии для всех серийных компонент.
