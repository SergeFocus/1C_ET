Directory "Der Bereich für Arbeitsplatz Fonds des Betreibers"

Bestimmt Registrierungsposition der Nomenklatur in einer bestimmten Reihenfolge zu den Bedingungen ihrer Durchführung für Kassen des Bedieners.

Wenn durchgeführt, Buchhaltung, können Sie mehrere Bereiche für die verschiedenen Unternehmensbereiche angeben. In diesem Fall muss der Betreiber Fondseinheit gehören.

Für eine Reihe von Produkten, können Sie die Bedingungen für die Umsetzung festzulegen, abhängig von den Konfigurationseinstellungen für Material, aus dem das Produkt implementiert ist, eine eigene Rechtspersönlichkeit (Organisation), in deren Namen die Ware verkauft, die Verkaufs angegebene Preisklasse, registrieren Empfang bei einem bestimmten Geschäft Geld.
