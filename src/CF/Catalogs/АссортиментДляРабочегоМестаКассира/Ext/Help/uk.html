Довідник "Асортимент для Робочого місця оператора каси"

Призначений реєстрації позицій номенклатури в певному порядку з умовами її реалізації для роботи оператора каси.

Якщо ведеться облік по підрозділах, то можна задавати декілька асортиментів для різних підрозділів. У цьому випадку оператор каси повинен належати підрозділу.

Для позицій асортименту можна вказати умови реалізації в залежності від налаштувань конфігурації: склад з якого реалізується товар, власне юріліческое особа (організація) від імені якого реалізується товар, заданий вид ціни реалізації, реєструвати надходження коштів у визначений сховище грошей.
