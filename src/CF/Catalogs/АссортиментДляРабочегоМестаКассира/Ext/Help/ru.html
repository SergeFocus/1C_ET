Справочник "Ассортимент для Рабочего места оператора кассы"

Предназначен регистрации позиций номенклатуры в определенном порядке с условиями её реализации для работы оператора кассы.

Если ведётся учет по подразделениям, то можно задавать несколько ассортиментов для различных подразделений. В этом случае оператор кассы должен принадлежать подразделению.

Для позиций ассортимента можно указать условия реализации в зависимости от настроек конфигурации: склад с которого реализуется товар, собственное юридическое лицо (организация) от имени которого реализуется товар, заданный вид цены реализации, регистрировать поступления денег в определенное хранилище денег.
