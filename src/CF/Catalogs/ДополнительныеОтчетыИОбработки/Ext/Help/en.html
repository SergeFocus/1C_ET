Directory of additional reports and treatments

It intended for storage of reports and additional treatments that are compatible with the configuration without the need for its modification

Reports or processing here will be loaded into the database as a separate file that is unpacked before running the program.

  • Symptom "Run modal" will cause the system to run the process (report) in burst mode (if enabled).
  • Sign "Perform the service menu", along with the installation configuration interface "Enable menu service call processing" take out the launch of this treatment in the service menu users.Item "Custom command" (Ctrl + Shift + F10).
