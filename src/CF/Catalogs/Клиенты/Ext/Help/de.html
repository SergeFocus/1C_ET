Voraussetzungen:

Typ Preis (ob eingeschlossenen posistema Komplexen Mechanismus angesichts des Preises) bestimmt die Art der Standardpreise an Durchführung Artikel-Client. Preis Typ kann das gleiche für alle Kundengruppen zugeordnet werden.

Bereich (ob einschließlich Buchhaltung Regions) bestimmt die Position des Auftraggebers. Auf dieser Basis können Sie Gruppe Kunden in den Berichten.

Die prozentuale Rabatte - (falls vorhanden Accounting Rabatte) legt einen persönlichen Ermäßigung für den Client. Wenn den gleichen Prozentsatz, wird der Verkauf NomenklaturIn-Karte, die zeigt auch an, den Diskontsatz, wählt das Programm die höchste prozentuale Rabatte. Die prozentualen Rabatt kann auf Kundengruppen zugeordnet werden.

Adresse.Telefon.OKPO.Rechtsstatus.E-mail, die Details des bargeldlosen Zahlungs - Weitere Informationen über die Kunden.

Lager Klient Wenn das Subsystem "Bilanzierung des Aktien", können Sie den Standardspeicher für den Client festlegen. In diesem Fall wird die Auswahl der Dokumente des Kunden, das Lager wird automatisch installiert.

Formel-Beschriftungen (ob verwenden Sie die Formel) Definiert eine einzigartige Formel für die Bildung des Textes auf dem Etikett, wenn Sie Etiketten für den Kunden zu drucken. Solche Formeln gespeichert sind in Referenz Formeln. Wenn beispielsweise Dokument "Warenfluss" Set-Client mit seiner Formel von Textbeschriftungen, wenn Sie anrufen Unterprogramm-Etikettendruck der Drucker mit der durch diese Formel erzeugten Text beschriften. Wenn die Formel Beschriftungen der Client nicht gefüllt, aber Bilanzierungs- Lagerhallen und eine Formel für die volle Lager Umsetzung, wird diese verwendet. Die letzte Sache ist, die grundlegende Formel für die Bildung der Text des Etiketts zu verwenden Programm-Einstellungen. Wenn er nicht gefüllt ist, und dann wird die Beschriftung entspricht dem Text sein Barcode Nomenklatur.

Beispiel Formel Beschriftungen:

SokrLP (Int (Preis) + 1) + "" + Leo (Nomenklatura.NomenklaturnayaGruppa.Kod, 2) + Leo (Nomenklatura.Proizvoditel.Kod, 2) + "x" + Format (Mehrwertsteuer, "CC = 9; CHDTS = 0 ")

Formeln sollten in einer Programmiersprache 1C beschrieben werden.

Name Drucken - Der Text, der in der "Kunde" beim Drucken von Dokumenten aus dem Programm angezeigt. Dieser Text kann von den üblichen Namen des Kunden abweichen. Zum Beispiel der Name des Clients in das Verzeichnis - Sail, Ltd., und sein Name für den Druck - LLC "Sail", Odessa, Malaya Arnautskaya 9.

Zuhause Währung (falls vorhanden Bilanzierung Währung) ist für die Dokumente in der Auswahl den Client automatisch eine bestimmte Währung.

Eigene Mitarbeiter (falls vorhanden Lohnbuchhaltung) kann ein Client sein. Solche Clients werden automatisch generiert, wenn Sie Informationen eingeben Personal. Dies ist notwendig, um sich auf den Kauf von Waren mit dem Kunden bei den Mitarbeitern Rabatte und berücksichtigen die Siedlungen Gehaltsliste Mitarbeiter.

Für eine Gruppe von Clients können Sie Prozentsatz der Provisionen gesetzt. Er wird funktionieren, wenn Sie einen Prozentsatz verwenden Vereinbarung mit dem Auftraggeber.
