Références clients

Conçu pour le dépôt des clients. Utilisez si y compris l'intégration des clients.

Fournit prédéfini client «personne privée» pour l'enregistrement de données non personnelles sur les ventes. Autrement dit, ceux pour lesquels il n'y a pas besoin de garder une trace de la dette. Il peut être renommé.

Hiérarchie Directory. Voilà, vous pouvez regrouper les clients.

Requis:

Type Prix (si posistema inclus Mécanisme complexe vu le prix) détermine le type de prix par défaut à Exécution Client d'objet. Type de prix peut être attribué le même pour tous les groupes de clients.

Région (si y compris la comptabilité Régions) détermine l'emplacement du client. Sur cette base, vous pouvez regrouper les clients dans les rapports.

Les réductions de pourcentage - (si inclus Réductions comptables) établit une réduction de personnel pour le client. Si le même pourcentage, puis la vente la nomenclatureDans la carte qui indique également le taux d'actualisation, le programme sélectionne les réductions de pourcentage plus élevés. Le pourcentage de réduction peut être attribuée à des groupes de clients.

Adresse.Téléphone.Okpo.Statut légal.E-mail, les détails de paiement sans numéraire - Informations complémentaires sur le client.

Stock client Lorsque le sous-système «Comptabilisation du stock," vous permet de spécifier le stockage par défaut pour le client. Dans ce cas, le choix des documents du client, l'entrepôt sera installé automatiquement.

Étiquettes de texte de formule (si utiliser la formule) Définit une formule unique pour la formation du texte sur l'étiquette lorsque vous imprimez des étiquettes pour le client. De telles formules sont stockés dans Formules de référence. Par exemple, si document «flux de marchandises" Réglez client avec sa formule d'étiquettes de texte, lorsque vous appelez impression d'étiquettes sous-programme l'imprimante étiquette avec le texte généré par cette formule. Si le texte de la formule étiquettes le client est pas rempli, mais entrepôts de comptabilité utilisé et une formule pour le plein Entrepôt mise en œuvre, il sera utilisé. La dernière chose est d'utiliser la formule de base pour la formation du texte de l'étiquette les paramètres du programme. Dans le cas contraire rempli et ensuite l'étiquette de texte sera égal au texte code à barre la nomenclature.

Exemple Formule étiquettes de texte:

SokrLP (Int (Prix) + 1) + "" + Leo (Nomenklatura.NomenklaturnayaGruppa.Kod, 2) + Leo (Nomenklatura.Proizvoditel.Kod, 2) + "x" + Format (TVA, "CC = 9; CHDTS = 0 ")

Formules doivent être rédigés dans un 1C du langage de programmation.

Nom Imprimer - Le texte qui sera affiché dans le «Client» lors de l'impression des documents du programme. Ce texte peut être différent du nom habituel du client. Par exemple, le nom du client dans le répertoire - Voile, Ltd., et son nom pour l'impression - LLC "Voile", Odessa, Malaya Arnautskaya 9.

Accueil Devise (si inclus Comptabilisation des devises) est fixé pour les documents dans la sélection du client établit automatiquement une certaine monnaie.

Propre personnel (si inclus comptabilité de la paie) peut être un client. Ces clients sont générés automatiquement lorsque vous entrez les informations Personnel. Ceci est nécessaire afin de donner aux employés les réductions sur l'achat de biens et de prendre en compte les colonies avec le client au paie employé.

Pour un groupe de clients, vous pouvez définir le pourcentage de frais de commission. Il travaillera sauf si vous utilisez un pourcentage de Accord avec le client.

Notez que si le programme de tenir des registres de clients, mais ne conduira pas compte de l'argent, Dans l'analyse rapport, «Débiteurs» l'image sera unilatérale. Comme si les marchandises été mis en œuvre, mais pas payé par le client, car il n'y a pas de possibilité d'inscrire dans le programme Rentrées de fonds des clients avec de l'argent enregistré déconnecté.

Quand il est nécessaire de compenser la dette entre le client et / ou Fournisseurs en tenant compte de la réception / sorties d'argent, vous devez utiliser un formulaire spécial des ajustements "Compensation des contreparties de la dette».

Entrepreneurs - nom général pour les clients et les fournisseurs.
