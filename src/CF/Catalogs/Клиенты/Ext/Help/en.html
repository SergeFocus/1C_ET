Type Price (if included posistema Complex mechanism considering the price) determines the type of default prices at implementation Item client. Price type can be assigned the same for all customer groups.

Area (if including accounting Regions) determines the location of the Client. On this basis, you can group clients in the reports.

The percentage discounts - (if included Accounting discounts) establishes a personal discount for the Client. If the same percentage, then the sale nomenclatureIn card which also indicates the discount rate, the program selects the highest percentage discounts. Address.Phone.OKPO.Legal status.E-mail, details of cashless payment - Additional information on the Customer.

Stock client When the subsystem "Accounting for the stock," allows you to specify the default storage for the client. In this case, the choice of the Client's documents, the warehouse will be installed automatically.

Formula text labels (if use the formula) Defines a unique formula for the formation of the text on the label when you print labels for the customer. Such formulas are stored in Reference formulas. For example, if document "flow of goods" Set client with his formula of text labels, when you call subroutine label printing the printer will label with the text generated by this formula. If the formula text labels the client is not filled, but used accounting warehouses and a formula for the full Warehouse implementation, it will be used. The last thing is to use the basic formula for the formation of the text of the label program settings. If it is not filled and then the text label will be equal to the text Barcode nomenclature.

Example formula text labels:

SokrLP (Int (Price) + 1) + "" + Leo (Nomenklatura.NomenklaturnayaGruppa.Kod, 2) + Leo (Nomenklatura.Proizvoditel.Kod, 2) + "x" + Format (VAT, "CC = 9; CHDTS = 0 ")

Formulas should be written in a programming language 1C.

Name Print - The text to be displayed in the "Client" when printing documents from the program. This text may differ from the usual name of the client. For example, the name of the Client in the directory - Sail, Ltd., and its name for printing - LLC "Sail", Odessa, Malaya Arnautskaya 9.

Home Currency (if included Accounting for currency) is set for the documents in the selection of the client automatically establishes a certain currency.
