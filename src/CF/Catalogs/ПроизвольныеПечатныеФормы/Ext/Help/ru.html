Справочник Произвольные Печатные Формы

 Справочник для регистрации и хранения собственных произвольных печатных форм.

Произвольные печатные формы - альтернативный механизм печати элементов справочников и документов. Благодаря ему, пользователь может создавать собственные печатные формы, не производя изменений в конфигурации, обмениваться ими, изменять и использовать по своему усмотрению.

Произвольные печатные формы задаются для Вида печатной формы.

Если актуальной произвольной формы для вида печатной формы, согласно условиям не задано, то будет произведена печать обычной встроенной печатной формы.

Произвольные печатные формы можно так же задать строго пользователю в регистре "Заданных печатных форм".

Макет произвольной печатной формы может оставаться во внешнем файле формата mxl или быть загруженным в базу данных.

Структура такого макета должна содержать области печати с параметрами печати, описанными на закладке "Описание". Они различные для различных объектов печати.

Можно задать сложные программируемые параметры и производить промежуточные расчеты при формировании печатной формы.

Программируемые параметры и общие параметры промежуточных расчетов можно так же расположить в макете внешней печатной формы.

В карточке печатной формы можно задать строгие параметры печати.

Использование печатной формы можно ограничить периодом и списком пользователей.

Программа условия для автоматического подбора позволяет задать такое условие, чтобы для документов одного и того же вида в один период времени для одного пользователя печатать различные печатные формы.

В такой программе на языке 1С можно задать условие к реквизиту "СсылкаНаЭлемент", который указывает на документ или элемент справочника, подлежащий произвольной печати.

Так, например, программа:

 ТипЗнч(СсылкаНаЭлемент.КлиентПоставщик) = Тип("СправочникСсылка.Клиенты")

- задает условие для печати документа Поступление Товаров только для случаев, когда контрагент - клиент. То есть это ви документа возврат товаров.
