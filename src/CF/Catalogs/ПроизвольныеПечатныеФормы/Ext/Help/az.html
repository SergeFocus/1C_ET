Keyfi çap forması - alternativ çap mexanizmi elementləri və arayış sənədlər.Vasitəsilə, istifadəçilər öz istədiyi konfiqurasiya dəyişikliklər payı, redaktə və istifadə etmədən öz çap plitələr yarada bilərsiniz.

Random çap plitələr Vida çap forması üçün müəyyən edilir.

Şərtlərlə insan gözü çap, faktiki pulsuz form müəyyən deyilsə, çap adi daxili çap şəklində həyata keçiriləcək.

Random çap lövhələr də halda ciddi istifadəçi müəyyən edilə bilər "çap formalarının edir."

Layout ixtiyari forma xarici fayl format və ya MXL bazasında instantly çap edilə bilər.

Layout strukturu "Açıklaması" təsvir printer ayarları çap sahəsi olmalıdır.Onlar müxtəlif obyektlərin çap üçün fərqlidir.

Siz kompleks proqramlaşdırıla parametrləri daxil və çap formalarının yaranmasında aralıq ödənişlər edə bilərsiniz.

Programlanabilir parametrləri və aralıq ödənişlər kimi qlobal variantları layout və xarici çap şəklində yerləşdirilə bilər.

Kart çap şəklində ciddi çap parametrləri müəyyən edilə bilər.

Çap formaları istifadə müddəti və istifadəçilərin siyahısı məhdudlaşdıra bilər.

Avtomatik seçilməsi proqramı üçün Conditions bir istifadəçi üçün bir müddət eyni cür sənəd çap formaları bir sıra çap şərt müəyyən etməyə imkan verir.
