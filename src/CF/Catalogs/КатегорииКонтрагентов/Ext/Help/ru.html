Справочник категорий контрагентов

 Учет по категориям должен быть включен в настройках.

Справочник позволяет учитывать контрагентов (поставщиков и клиентов) по категориям, заданным пользователем.

Сотрудников можно прикреплять к категориям контрагентов.

  • Так же одного сотрудника можно выбрать в качестве куратора категории.

 

Анализ можно группировать по категориям контрагентов помимо обычной иерархии контрагентов по группам.
