Elenco Immagini

Progettato per collegare gli elementi Una immagine di directory e documenti.

Le immagini possono essere un collegamento a un file su Internet, a un file su un computer o caricati nel database.

Nelle impostazioni è possibile specificare una formula per i nomi di file auto-immagine quando si scaricano.

Nelle proprietà di un gruppo di immagini, è possibile specificare la dimensione delle direzioni orizzontale e verticale, che saranno fornite immagine.

Quando si scaricano dei file a una dimensione di database del database aumenterà proportsinalno immagini scaricate. Pertanto, un gran numero di immagini di grandi dimensioni sarebbe ragionevole organizzare una cartella esterna con le immagini e specificare gli indirizzi dei file di immagine senza scaricare le immagini direttamente nel database.
