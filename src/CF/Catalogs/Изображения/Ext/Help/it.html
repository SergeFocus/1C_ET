Elenco immagine

Progettato per allegare immagini a elementi e documenti di riferimento.

Le immagini possono essere un collegamento ad un file su Internet, facendo riferimento a un file sul computer o caricati nel database.

Nelle impostazioni è possibile specificare una formula per i nomi di file auto-immagine quando si scaricano.

In un gruppo di immagini, è possibile specificare le proprietà delle dimensioni orizzontale e verticale, che sarà data immagine.

Quando si scaricano file a una dimensione del database del database aumenterà in proporzione l'immagine caricata.Pertanto, un gran numero grande immagine di corretta per organizzare una cartella esterna con le immagini e specificare gli indirizzi di immagini file senza caricare le immagini direttamente al database.
