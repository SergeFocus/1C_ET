Verzeichnis Images

Entwickelt, um auf die Elemente Ein Bild der Verzeichnisse und Dokumente anhängen.

Die Bilder können einen Link zu einer Datei im Internet, auf eine Datei auf einem Computer oder in die Datenbank geladen werden.

In den Einstellungen können Sie eine Formel für die automatische Bilddateinamen beim Download angeben.

In den Eigenschaften einer Gruppe von Bildern, können Sie die Größe der horizontalen und vertikalen Richtung, das Bild gegeben werden soll.

Wenn Sie Dateien in einer Datenbank Größe der Datenbank wird proportsinalno heruntergeladenen Bilder zu erhöhen. Daher ist eine große Anzahl von großen Bildern wäre es sinnvoll, einen externen Ordner mit Bildern organisieren und geben Sie die Adressen der Bilddateien, ohne den Download der Bilder direkt in die Datenbank.
