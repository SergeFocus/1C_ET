Verzeichnis "Bilder"

Ein Bild entwickelt, um auf die Elemente der Verzeichnisse und Unterlagen zu befestigen.

Bilder können ein Verweis auf eine Datei auf dem Internet sein, link auf die Datei auf Ihrem Computer oder in die Datenbank geladen.

Wenn Sie Dateien in der Datenbank die Größe der Datenbank erhöht proportsinalno Bilder. Daher ist für eine große Anzahl von großen Bildern wäre richtig zu externen Ordner mit Bildern und Hyperlinks Bilddateien, ohne dass die Bilder direkt in der Datenbank abrufen zu organisieren.
