Bezugsbilder

Geeignet für die Bilder, die auf Elemente der Referenz-Bücher und Dokumente zu befestigen.

Die Bilder können in die Datenbank auf einem Computer oder geladen ein Link zu einer Datei auf dem Internet, auf eine Datei sein.

In den Einstellungen können Sie eine Formel für die automatische Bilddateinamen angeben, wenn Sie das Herunterladen.

In einer Gruppe von Bildern können Sie die Eigenschaften der Dimensionen horizontal und vertikal angegeben, das Bild gegeben werden.

Wenn Sie Dateien in einer Datenbank Größe der Datenbank Herunterladen wird im Verhältnis zu dem hochgeladenen Bild zu erhöhen.Daher ist eine große Anzahl von großen Bildern wäre es sinnvoll sein, einen externen Ordner mit Bildern zu organisieren und die Adressen von Bilddateien angeben, ohne das Bild direkt in die Datenbank herunterladen zu müssen.
