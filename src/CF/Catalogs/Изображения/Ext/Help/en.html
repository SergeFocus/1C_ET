Reference Images

Suitable for attaching images to elements of reference books and documents.

Images may be a link to a file on the Internet, to a file on a computer or loaded into the database.

In the settings you can specify a formula for auto-image file names when downloading.

In a group of images, you can specify the properties of the dimensions horizontally and vertically, which will be given image.

When downloading files to a database size of the database will increase in proportion to the uploaded image.Therefore, a large number of large images it would be reasonable to organize an external folder with images and specify the addresses of image files without having to download the image directly into the database.
