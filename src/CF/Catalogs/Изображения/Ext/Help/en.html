Directory Images

Designed to attach to the elements An image of directories and documents.

The images may be a link to a file on the Internet, to a file on a computer or loaded into the database.

In the settings you can specify a formula for auto-image file names when downloading.

In the properties of a group of images, you can specify the size of the horizontal and vertical directions, which will be given image.

When downloading files to a database size of the database will increase proportsinalno downloaded images. Therefore, a large number of large images it would be reasonable to organize an external folder with images and specify the addresses of image files without downloading the images directly into the database.
