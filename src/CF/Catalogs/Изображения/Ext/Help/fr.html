Annuaire Images

Conçu pour fixer les éléments Une image de répertoires et de documents.

Les images peuvent être un lien vers un fichier sur Internet, à un fichier sur un ordinateur ou chargé dans la base de données.

Dans les paramètres, vous pouvez spécifier une formule pour les noms de fichiers auto-images lors du téléchargement.

Dans les propriétés d'un groupe d'images, vous pouvez spécifier la taille des directions horizontale et verticale, qui seront donnés image.

Lorsque le téléchargement de fichiers à une taille de la base de données de base de données va augmenter proportsinalno images téléchargées. Par conséquent, un grand nombre de grandes images, il serait raisonnable d'organiser un dossier externe avec des images et spécifier les adresses des fichiers d'image sans avoir à télécharger les images directement dans la base de données.
