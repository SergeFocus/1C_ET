Справочник Изображения

 Предназначен для прикрепления изображений к элементам справочников и документам.

Изображения могут быть ссылкой на файл в сети интернет, ссылкой на файл на компьютере или загружены в базу данных.

В настройках можно указать формулу для авто-наименования файла изображения при загрузке.

В свойствах группы изображений можно задать размеры по горизонтали и по вертикали, к которым будет приведено изображение.

При загрузке файлов в базу данных размер базы данных будет увеличиваться пропорционально загруженным изображениям. Поэтому для большого числа больших изображений правильно было бы организовать внешнюю папку с изображениями и указывать адреса файлов картинок, не загружая изображения непосредственно в базу данных.
