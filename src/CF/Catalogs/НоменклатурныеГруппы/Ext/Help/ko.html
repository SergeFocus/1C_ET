명명법 그룹 디렉토리

와 회계 에이블 명명법 그룹의 맥락에서 제품.

명칭의 그룹 (상품)와 혼동하지 않습니다. 그룹 명명법 그룹 또는 명칭과 연관 될 수있는 독립적 인 가이드 또는 방해 -이 참조 명칭과 명칭 그룹 "폴더". 주요 목적은 - 특정 항목의 등록 및 회계의 규칙을 지정합니다.

매개 변수 :이 그룹의 공식 자동 이름 명명법 (대한 사용에 전환 방식)는 명칭 그룹의 제품 이름에 대한 규칙을 설정할 수 있습니다. 프로그램의 자동 이름은 계정에 기본 공식을 소요하는 경우 가격보기 명명법 (경우에 이름의 경우 가격).

매개 변수 : 같은 이름의 명칭의 그룹을 지정 - 명명법 기는 하나의 그룹 디렉토리에 결합되는 경우이 경우에 필요 명명법. 예를 들어, 경우에 따라서는 자동 기의 명명법 위치의 수단으로 작용한다. 따라서,시 선택 공식 자동 이름 및 그룹 기준 범위와의 연결이 핸드북 요소는,이 자동 이름을 적용 그룹에 종속된다.

옵션 :이 그룹의 범위, 계정에 시리얼을 복용 - 전체 세트 들어 회계의 명칭에 종속됩니다 시리즈.

이것은 항목 요구 고유 명칭 그룹에 대해 설정 될 수있다.

테이블 : 명명법의 추가 세부 사항 - 기회는 설치를위한 명명법 그룹에서 전체 명칭에 자세한 내용의 그것의 자신의 세트를 추가합니다. 주의 : 이미 그룹의 명명법의 명칭을 사용하는 경우 자세한 내용의 구성을 변경하지 마십시오!

명명법 기는 가격 형성 방법의 고유 설치 가격의 종류라고 (가격 중 한 종류 만, 다른 그룹의 형성에 따라 다른 방식으로 nomenklaturynh) 될 수있다.
