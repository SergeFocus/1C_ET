Справочник номенклатурных групп

 При включенном учете товара в разрезе номенклатурных групп.

Не следует путать с Группами номенклатуры (товара и услуг). Группы номенклатуры это "папки" в справочнике номенклатуры, а Номенклатурные группы - самостоятельный справочник, который может быть или связан с группами номенклатуры или пересекаться с ним. Главное его предназначение - обозначить правила регистрации и учета определенной номенклатуры.

параметр: Формула авто-наименования номенклатуры данной группы (при включенном использовании формул) позволяет задать правила формирования наименования товара данной Номенклатурной группы. При авто-наименовании по формуле программа учитывает базовый Вид цены номенклатуры (для случаев, когда цена часть наименования).

параметр: Связать одноименную группу номенклатуры - необходимо для случаев, когда номенклатурная группа связана с одной группой справочника номенклатуры. Например, в тех случаях, когда она служит лишь средством автоматического наименования номенклатуры данной группы. Так, когда выбрана формула авто-наименования и указана связь с группой справочника номенклатура, то к элементам этого справочника, подчиненных это группе будет применено это авто-наименование.

параметр: Номенклатура этой группы с серийным учетом - для всей подчиненной номенклатуры будет задан учет по сериям.

Может быть установлено требование уникальности артикулов номенклатуры группы.

таблица: Дополнительные реквизиты номенклатуры - возможность добавить ко всей номенклатуре в рамках этой номенклатурной группы свой набор дополнительных реквизитов для их установки. Внимание: Не вносите изменения в состав дополнительных реквизитов, если уже используете номенклатуру данной номенклатурной группы!

Номенклатурная группа может  упоминаться в видах цен для установки уникального способа формирования цены (один Вид цен, но разные способы формирования в зависимости для различных номенклатурных групп).
