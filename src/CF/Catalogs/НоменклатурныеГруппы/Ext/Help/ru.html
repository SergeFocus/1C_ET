Справочник "Номенклатурные Группы"

При включенном учете товаров в разрезе номенклатурных групп.

Не следует путать с Группами номенклатуры (товаров). Группы номенклатуры это "папки" в справочнике номенклатуры, а Номенклатурные группы - самостоятельный справочник, который может быть или связан с группами номенклатуры или пересекаться с ним.

Формула авто-наименования номенклатуры данной группы (при включенном использовании формул) позволяет задать правила формирования наименования товаров данной Номенклатурной группы. При авто-наименовании по формуле программа учитывает базовый вид цены номенклатуры (для случаев, когда цена часть наименования).

Связать одноименную группу номенклатуры - необходимо для случаев, когда номенклатурная группа связана с одной группой справочника номенклатуры. Для тех случаев, когда она служит лишь средством автоматического наименования номенклатуры данной группы. Таким образом, когда выбрана формула авто-наименования и указана связь с группой справочника номенклатура, то к элементам этого справочника, подчиненных это группе будет применено это авто-наименование.

Номенклатура этой группы с серийным учетом - для всей подчиненной номенклатуры будет задан учет по сериям.
