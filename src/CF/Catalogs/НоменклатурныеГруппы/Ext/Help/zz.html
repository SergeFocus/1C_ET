Названный Группа Каталог

С Включить учет Продукт контекста именования правовая группа.

Не следует путать с группой под названием (товаров). Группа Номенклатура "Папка" в справочной номенклатуре и названных групп - групп или независимой направляющей терминологии, связанной или вмешательства ИТ. Его основная цель - назначенные регистрации и учета правил конкретного проекта.

Параметры: Эта группа формулы автоматически присваивается (The С помощью переключателя Формула) Позволяет установить правила для имени название продукта группы. Если схема автоматического именования рассмотреть основную формулу Проверить цену Номенклатура (в случае, когда цена на имя).

Параметры: назначить группу имени тот же - Это следующий необходимый назвал группа связана с набором каталогов Номенклатура, Например, в случае, он автоматически назван позиция группы означает закон. Таким образом, если выбранный Формула Автоматическая имя и связь с контрольной группой диапазоне, это руководство элемент, он находится в подчинении группы будет применять эту автоматическое имя.

Опции: диапазон этой группы, принимая во внимание сериал - Для всей совокупности подлежат учетной терминологии Серия,

Это может быть только настройки номенклатура группы требование.

Таблица: Детали Названы - Иметь возможность присоединиться полное имя именованного группы в своем собственном наборе дополнительных деталей для их установки. Примечание: Если вы использовали имя группы по имени друга деталь компоненты не изменять!

Цены видов названные группы могут быть упомянуты для конкретного способа формирования цены на установки (в цене, но в иной форме по-разному в зависимости от групп nomenklaturynh).
