Language Reference

A list of interface languages. The default language "Russian".

Also can add a list to specify the languages ​​to Regions and contractors - language of communication and language documentation.

The program includes some of the world's languages, but you can add your own.

It should be mentioned, which will be used by the language:

  • For documents (language can be set as such, which will print the document to counterparties).
  • For the name of the object (to ask in the names of the translation).
  • For communication (as the information in the card company or an employee of the language to communicate with them).
