Directory Languages

A list of interface languages.The default language "Russian".

In addition the list may be complemented for specifying languages ​​forregions and counterparties - language of communication and language documentation.

The program incorporated some of the world's languages, but you can add your own.

It is necessary to specify what language will be used:

  • For documents (language can be set as such, which will be printed counterparts documents).
  • For the names of objects (in order to define translation in names).
  • For communication (as the information in the card company or an employee of the language to communicate with them).
