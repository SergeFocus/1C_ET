Справочник языков

Перечень языков для интерфейса и данных. По-умолчанию язык "русский".

Кроме того перечень может дополнятся для указания языков для регионов и контрагентов - языка общения и языка документации.

В программу заложены некоторые языки мира, но Вы можете добавить собственные.

Следует указать, для чего будет использован язык:

  • Для документов (язык можно будет задать в качестве такого, на котором будут печататься документы контрагентам).
  • Для наименования объектов (для того, чтобы в наименованиях задавать перевод).
  • Для общения (в качестве информации в карточке контрагента или сотрудника о языке общения с ним).
