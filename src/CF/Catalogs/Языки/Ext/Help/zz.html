Язык Справочник

Список языков для интерфейса и данных.Язык по умолчанию является "русский".

Вы также можете перечислить в качестве руководства для dopovnyatsyaплощадка и контрагенты - язык общения и языка документации.

Программа сочетает в себе язык мира, но вы можете добавить свои собственные.

Необходимо указать, какой язык будет использоваться:

  • Для получения документов (язык может быть установлен таким образом, что он будет публиковать подрядчика).
  • Для этого объекта (указать название перевода).
  • Связь (в том числе информационная карта, служащий или подрядчик языка для общения с ними).
