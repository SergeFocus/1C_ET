Справочник Объекты Клиентов

 Предназначен для учета выполнения работ над объектами клиентов, которые не вписываются в логику справочника Номенклатура. Например, для строительных объектов.

Объекты Клиента это дополнительны справочник при выполнении работ над такими объектами, которые не приходуются и не расходуются и не учитываются на складе. То есть не являются номенклатурой. Например, при выполнении строительных работ объектом клиента может выступить "крыльцо", "крыша и т.п. При сервисных услугах и, например, ремонтных работах над товарами этот справочник только запутает Вас - отключите его.
