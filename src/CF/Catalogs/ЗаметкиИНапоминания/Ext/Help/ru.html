Справочник "Заметки, задачи и напоминания"

Система позволяет:

  • Создавать заметки. Свободные или связанные с объектами базы данных
  • Создавать напоминания. Свободные или связанные с объектами базы данных. С возможностью повторять их через определенное время
  • Создавать Задачи с возможностью повторять их через определенное время

Предусмотрены такие типы задач:

    • Резервирование каталога базы данных  
    • Резервирование каталога базы данных в архив
    • Запись отчета руководителю в файл Excel
    • Отправка отчета руководителю на емейл
    • Выполнение программного кода на языке 1С

 

Программа проверяет наличие задач и напоминаний раз в минуту.

Задачи рекомендуется использовать вместе с включеной настройкой журналирования событий, чтобы в журнале событий было видно как выполнялись задачи.
