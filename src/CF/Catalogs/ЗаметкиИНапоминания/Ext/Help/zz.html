Ссылаясь на заметки, задачи и заметки

Система может:

  • Создание примечаний. Бесплатные или объектов, связанных с базами данных
  • Создать напоминания. Бесплатные или объектов, связанных с базами данных. Их способность при повторном через некоторое время
  • Возможность создания задачи повторить их использование по истечении определенного времени

Есть эти типы вопросов:

    • Резервное копирование базы данных каталога
    • Резервное копирование базы данных Архивация Каталог
    • Написать отчет, глава Высокая прочность
    • Будет ли отправить отчет в заголовке электронной почты
    • Выполнение Код языка 1С

Программа проверяет задачу и напомнить каждую минуту.
