Référence notes, tâches et mémos

Le système permet:

  • Créer des notes. Objets de base de données gratuites ou connexes
  • Créer des rappels. Objets de base de données gratuites ou connexes. Avec la possibilité de les répéter après un certain temps
  • Créer des tâches avec la possibilité de les répéter après un certain temps

Il ya ces types de problèmes:

    • Répertoire de sauvegarde de base de données
    • Sauvegarde base de données de catalogue à l'archive
    • Rédaction de déposer un rapport à la tête Exceller
    • Envoi du rapport à la tête d'un courriel
    • Exécution Code de la langue 1C

Le programme vérifie pour des tâches et des rappels à chaque minute.

Tâches recommandées pour une utilisation avec ce paramètre est activé, journalisation des événementsPour journal des événements a été vu comment effectuer les tâches.
