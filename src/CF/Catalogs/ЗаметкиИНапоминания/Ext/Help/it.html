Note di riferimento, attività e memo

Il sistema consente di:

  • Creare note.Gratuito o associati a oggetti di database
  • Creare promemoria.Gli oggetti liberi o relativi database.Con la capacità di ripeterli dopo un certo tempo
  • Creare attività con la capacità di ripeterli dopo un certo tempo

Ci sono questi tipi di compiti:

    • Directory del database di backup
    • database del catalogo di backup all'archivio
    • rapporto record nel file testa
    • L'invio il rapporto al capo di una e-mail
    • Codice lingua esecuzione1C

Il programma controlla per le attività e promemoria una volta al minuto.

Attività consigliate per l'uso con l'impostazione inclusola registrazione degli eventiPer il registro eventifu visto come eseguire le attività.
