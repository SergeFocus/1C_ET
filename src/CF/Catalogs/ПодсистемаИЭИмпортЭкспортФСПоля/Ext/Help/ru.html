Справочник "Поля файлов шаблонов"

 

Справочник предназначен для хранения перечня возможных полей файлов для шаблонов, используемых для импортно-экспортных операций с файлами.

Существует большой набор предопределенных полей, для которых запрограммировано поведение при загрузке и выгрузке. В частности ля загрузки и выгрузки таблицы "Товары", для обмена "Банк-клиент".

Наименование - строгое наменование реквизита базы данных, которое соответствует реквизиту конфигурации 1С. Если используется наименование с точкой (Номенклатура.Код), то программа будет при выгрузке пытаться выгрузить содержание субреквизита (код номенклатуры). Если данное поле будет использовано в шаблоне, как ключевое, то при загрузке данных программа попытается определить значение объекта (номенклатуры) по субреквизиту (коду).

Комментарий это основное описание поля шаблона.

Тип поля (Строка S, Число N, Дата D) используются в формате DBF для определения типа колонки файла базы данных. Длина и Точность - определяют длину поля DBF. Точность кроме прочего строгу указывает на то, что поле - число.

Поле Определенное значение позволяет указать постоянное значение для выгрузки и загрузки, независимо от содержания данных.

Заголовок поля по умолчанию определяет имя поле файла базы данных. При обмене с DBF будет обрезан до 10 символов и переведен в латиницу.

Флаг, что Поле принадлежит объекту хозяину таблицы позволяет указать поля шапки документов или карточки справочников при выгрузке и загрузке их табличных частей.

Обмен полями производится в справочнике Шаблонов импорт-экспорта. Когда производится обмен шаблонами, выгружаются так же используемые поля.
