Довідник "Поля файлів шаблонів"

Довідник призначений для зберігання переліку можливих полів файлів для шаблонів, Використовуваних для імпортно-експортних операцій з файлами.

Існує великий набір визначених полів, для яких запрограмовано поведінку при завантаженні і вивантаженні. Зокрема ля завантаження і вивантаження таблиці "Товари", для обміну "Банк-клієнт".

Найменування - суворе Наменованіе реквізиту бази даних, яке відповідає реквізиту конфігурації 1С. Якщо використовується найменування з точкою (Номенклатура.Код), то програма буде при вивантаженні намагатися вивантажити вміст субреквізіта (код номенклатури). Якщо дане поле буде використано в шаблоні, як ключове, то при завантаженні даних програма спробує визначити значення об'єкта (номенклатури) по субреквізіту (коду).

Коментар це основний опис поля шаблону.

Тип поля (Рядок S, Число N, Дата D) використовуються в форматі DBF для визначення типу колонки файлу бази даних. Довжина іТочність - Визначають довжину поля DBF. Точність крім іншого строго вказує на те, що поле - число.

Поле Певне значення дозволяє вказати постійне значення для розвантаження і завантаження, незалежно від змісту даних.

Тема поля за замовчуванням визначає ім'я поле файлу бази даних. При обміні з DBF буде обрізаний до 10 символів і переведений в латиницю.

Прапор, що Поле належить об'єкту господареві таблиці дозволяє вказати поля шапки документів або картки довідників при вивантаженні і завантаженні їх табличних частин.

Обмін полями проводиться в довіднику Шаблонів імпорт-експорту. Коли проводиться обмін шаблонами, вивантажуються так само використовувані поля.
