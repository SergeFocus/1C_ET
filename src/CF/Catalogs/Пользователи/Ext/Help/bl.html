Калі база дадзеных на працоўным месцы выйграюць ад некалькіх супрацоўнікаў, не лішнім, каб вызначыць іх ролю.Каб карыстацца гэтай канфігурацыяй, карыстальнікаў, іх паролі і іх ролі вызначаюцца ў рэжыме канфігуратар 1С

Абавязкова быць карыстальнікам з роляй "Поўныя правы"Для таго, каб наладзіць праграму.Ўсталюеце такой пароль для гэтага карыстальніка.Усе далейшыя асноўныя налады праграмы зробленыя ад яго імя.

Ролю "Рэалізацыя"Гэтая роля толькі з абмежаванымі правамі для рэгістрацыі продажаў.
Ролю "Рэалізацыя і прыём тавару, ўстаноўка цэн і курсаў"Гэта мае больш пашыраныя правы.Такі карыстач афармляе куплю, рэгулюе кошты.
Ролю "Казначэй"Для аперацый з грашыма і заработнай платы.
Ролю "Аператар абсталяванне"- Для аператараў у канфігурацыі сайта (мантаж).
Ролю "Аператар касы"- Аператар каса.

Ролю "Кліент прадпрыемства"- Для карыстальнікаў прадстаўнікоў самай справе ў канчатковым карпаратыўных кліентаў.
Ролю "Пастаўшчык прадпрыемствы"- Для карыстальнікаў прадстаўнікоў пастаўшчыкоў тавараў і паслуг для прадпрыемства.

Ролю "Адміністратар сістэмы"- Для сістэмнага адміністратара (адміністратара базы дадзеных для прадпрыемства для выканання работ па тэхнічным абслугоўванні з базай дадзеных.
Ролю "Аператар функцыі"- Для карыстальнікаў з пэўнай функцыяй, такіх як з інтэрфейсам паказаным дадатковай апрацоўкі злучэння.
