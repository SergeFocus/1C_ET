Довідник Користувачі

Довідник не доступний для редакції.Користувачів і їх ролі слід визначати в режимі Конфігуратора 1С.

Якщо базою даних на даному робочому місці будуть користуватися кілька співробітників, то не зайвим буде визначити їх Роль.Для роботи з даною конфігурацією користувачі, їх паролі і їх ролі задаються в режимі конфігуратора 1С

Обов'язково повинен бути користувач з роллю "Повні права"Для настройки програми.Встановіть обов'язково пароль для такого користувача.Всі подальші основні настройки програми робите від його імені.

Роль "Реалізація"Це роль з обмеженими правами виключно для реєстрації продажів.
Роль "Реалізація та прийом товару, установка цін і курсів"Володіє більш розширеними правами.Такий користувач оформляє придбання, регулює ціни.
Роль "Скарбник"Для операцій з грошима і заробітною платою.
Роль "Оператор комплектації"- Для операторів на ділянці комплектації (збірки).
Роль "Оператор каси"- Для оператора касового робочого місця.

Роль "Клієнт підприємства"- Для користувачів представників власне кінцевих клієнтів підприємства.
Роль "Постачальник підприємства"- Для користувачів представників постачальників товару і послуг для підприємства.

Роль "Адміністратор системи"- Для системного адміністратора (адміністратора бази даних на підприємстві для виконання регламентних робіт з базою даних.
Роль "Оператор функції"- Для користувачів з однією певною функцією, наприклад з інтерфейсом, заданим за рахунок додаткової підключеної обробки.
