Répertoire des fournisseurs

Conçu pour les fournisseurs de dépôt. Utilisez si y compris l'intégration des fournisseurs.

Fournit prédéfini Fournisseur «personne privée» pour l'enregistrement de données non personnelles sur les acquisitions. Autrement dit, ceux pour lesquels il n'y a pas besoin de garder une trace de la dette. Ce fournisseur peut être renommé.

Hiérarchie Directory. Autrement dit, les fournisseurs peuvent être regroupés.

Requis:

Type Prix (si posistema inclus Mécanisme complexe vu le prix) détermine le prix de défaut pour l'affichage des produits du fournisseur. Type de prix peut être attribué le même pour l'ensemble du groupe des fournisseurs.

Région (si y compris la comptabilité Régions) détermine l'emplacement du fournisseur. Sur cette base, vous pouvez rapports groupe de fournisseurs.

Adresse.Téléphone.Okpo.Statut légal.E-mail, les détails de paiement sans numéraire - Des informations supplémentaires sur les fournisseurs.

Stock fournisseur (Lorsque le sous-système), «Comptabilisation du stock," vous permet de spécifier le stockage par défaut pour les fournisseurs. Ensuite, le choix des fournisseurs dans les documents, un entrepôt sera installé automatiquement.

Nom Imprimer - Le texte qui sera affiché dans le «Fournisseur» lors de l'impression des documents du programme. Ce texte peut être différent du nom habituel du fournisseur. Par exemple, le nom du fournisseur - Voile, Ltd, et d'imprimer le nom - LLC "Voile", Odessa, Malaya Arnautskaya 9.

Accueil Devise (si inclus Comptabilisation des devises) est fixé pour les documents à la sélection du fournisseur définit automatiquement une monnaie particulière.

Propre personnel (si inclus comptabilité de la paie) peut agir Fournisseur. Ces fournisseurs sont créés automatiquement lorsque vous entrez les informations Personnel. Il est nécessaire de prendre en compte les colonies avec le fournisseur au paie employé.

Fabricant - (si le compte est inclus) Une référence à l'ensemble de la gamme du fabricant, acheté chez le fournisseur. La même chose devrait être le cas quand il est la nomenclature fabricant de fournisseur. Puis, dans la déclaration "Des biens. Les soldes et opérations"Il est possible dans la version du rapport" par les producteurs "pour étudier le mouvement de son produit. Toujours en Compte rendu de vente fourni l'option "nomenclature vente par les producteurs."

Notez que si le programme de tenir des registres des fournisseurs, mais pas conduire compte de l'argent, Dans l'analyse Rapport "Selon les calculs avec les fournisseurs" l'image sera unilatérale. Comme si les produits ont été achetés mais non payés, comme il n'y a pas de possibilité d'inscrire dans le programme prestataire de paiement avec de l'argent enregistré déconnecté.

Quand il est nécessaire de compenser la dette entre le fournisseur et / ou Service client en tenant compte de la réception / sorties d'argent, vous devez utiliser un formulaire spécial des ajustements "Compensation des contreparties de la dette».

Entrepreneurs - nom général pour les clients et les fournisseurs.

A la commande "Imprimer" vous pouvez obtenir une carte avec le fournisseur et ses instruments de dette actuels.
