Справочник "ПОСТАВЩИКИ"

Предназначен для картотеки Поставщиков. Используется, если включен учет по поставщикам.

Предусмотрен предопределенный Поставщик "Частное лицо" для регистрации данных о неперсонифицированных приобретениях. То есть таких, для которых нет необходимости отслеживать задолженность. Этого поставщика можно переименовать.

Справочник иерархический. То есть Поставщиков можно группировать.

Реквизиты:

Вид Цен (если включена посистема Сложного Механизма учета Цен) определяет цену по-умолчанию для оприходования товара от поставщика. Вид Цен можно назначать так же для всей группы поставщиков.

Регион (если включен учет регионов) определяет расположение Поставщика. По этому признаку можно группировать Поставщиков в отчетах.

Адрес, Телефон, ОКПО, Правовой статус, Электронная почта, Реквизиты безналичного расчета - дополнительные данные о Поставщике.

Склад поставщика (при включенной подсистеме) "Учета по Складам" позволяет указать склад по умолчанию для Поставщика. Тогда, при выборе Поставщика в документах, такой склад будет установлен автоматически.

Наименование для печати - текст, который будет выводиться в поле "Поставщик" при печати документов из программы. Этот текст может отличаться от обыкновенного наименования Поставщика. Например, наименование поставщика - Парус, ООО , а наименование для печати - ООО "Парус", Одесса, ул.Малая Арнаутская, 9.

Основная Валюта (если включен Учет валют) устанавливается для того, чтобы в документах при выборе этого поставщика автоматически устанавливалась определенная валюта.

Собственный сотрудник (если включен учет заработной платы) может выступать Поставщиком. Такие Поставщики создаются автоматически при вводе информации по сотрудникам. Это необходимо для того, чтобы учесть взаиморасчеты с таким поставщиком при расчете заработной платы сотрудника.

Производитель - (если включен учет) ссылка на Производителя всей номенклатуры, купленной у данного поставщика. Так же нужно в случае, когда сам Поставщик это Производитель номенклатуры. Затем в отчете "Товары. Остатки и обороты" можно в варианте отчета "по производителям" изучить движения его товара. Так же в отчете по Продажам предусмотрен вариант "продажи номенклатуры по производителям".

 

Учтите, что если в программе вести учет по поставщикам, но не вести учет денег, то при анализе отчета "По расчетам с поставщиками" картина будет выглядеть односторонне. Как будто товар приобретался, но не оплачивался, так как нет возможности зарегистрировать в программе оплату поставщику при отключенном учете денег.

Когда нужно произвести взаимозачет долга между Поставщиками и/или Клиентами с учетом поступления/выбытия денег, следует воспользоваться специальной формой корректировки "Взаимозачет долга контрагентов".

Контрагенты - общее наименование для Клиентов и Поставщиков.

По команде "Печать" можно получить карточку Поставщика с текущим долгом и его документами.
