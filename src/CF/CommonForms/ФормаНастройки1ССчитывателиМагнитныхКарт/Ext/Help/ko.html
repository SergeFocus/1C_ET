설정 양식 처리기 드라이버 "1C : 마그네틱 카드 리더"

폼은, 외부 구성 요소와 접속되는 자기 카드 판독기의 파라미터를 설정하기위한 것이다 "1C :. 바코드 스캐너" 형태는 다음과 같은 장치를 구성 할 수 있습니다 :

  • 포트
    장치에 연결되어있는 직렬 포트 (물리적 또는 에뮬레이트).
  • 속도
    장치가 접속 된 포트를 설정하는 단계; 통신 속도.
  • 비트 데이터
    장치가 접속 된 포트를 설정하는 단계; 전송 된 바이트의 데이터 비트의 수.
  • 정지 비트
    장치가 접속 된 포트를 설정하는 단계; 정지 비트의 수.
  • 접두사
    접두사는 카드를 계산하는 장치의 값에 추가. 읽기 또는 새로운 트랙의 시작 부분의 표시입니다.
  • 접미사
    접미사 카드 카운트 장치의 값에 추가. 이는 트랙 또는 판독의 완료를 나타낸다.
  • 트랙 번호
    데이터가 지정된 접두사와 접미사 트랙 사이에 입력되는 트랙을 지정하는 데 사용됩니다. 데이터는 트랙의 앞의 확인란을 선택합니다 구성에 전송 된 추적합니다.

설정을 변경 한 후, 자신의 보존을 포기 버튼을 변경 한 내용을 저장하거나 "취소"할 수있는 "확인"버튼을 누릅니다.
