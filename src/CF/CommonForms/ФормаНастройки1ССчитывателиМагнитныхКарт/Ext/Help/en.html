Setting form handler driver "1C: Magnetic-card reader"

The form is intended to set the parameters of the magnetic card readers to be connected with an external component, "1C: barcode scanner." The form allows you to configure the following devices:

  • Port
    Serial port (physical or emulated) that is connected to the device.
  • Speed
    Setting the port to which the device is connected; data rate.
  • Bit data
    Setting the port to which the device is connected; number of data bits in a transmitted byte.
  • Stop bit
    Setting the port to which the device is connected; number of stop bits.
  • Prefixes
    Prefix added to the value of a device to count cards. Is a sign of the beginning of a read or a new track.
  • Suffixes
    Suffix appended to the value of the device to count cards. It is indicative of the completion of the track or reading.
  • The track numbers
    Used to specify the track to which the data is entered into between the specified prefix and suffix track. To track data were transferred to the configuration you must check the box in front of the track.

After changing the settings, press the "OK" button to save the changes made or "Cancel" button to abandon their conservation.
