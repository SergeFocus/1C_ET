양식 처리기 드라이버 설정 "오리온 : 재정 등록 FR01K"

양식은, 드라이버를 지원하는 회계 기록의 매개 변수를 설정하기위한 것입니다 "오리온 :. 재정 등록 FR01K" 형태는 다음과 같은 장치를 구성 할 수 있습니다 :

  • 포트
    회계 레지스터에 연결된 시리얼 포트.
  • 속도
    재정 등록과 데이터 환율.
  • 시간 제한
    회계 기록에서 최대 응답 시간.
  • 사용자 암호
    사용자 암호 FR. 그것은 DF에 수표를 인쇄 할 때 사용된다.
  • 관리자 암호
    관리자 암호 FR. 관리자 권한을 필요로하는 DF을 수행 할 때 사용된다.
  • Oplata1
    텍스트 인쇄 이름 무 현금 결제 №1. 예를 들어, "카드 결제"
  • Oplata2
    텍스트 인쇄 이름 무 현금 결제 №2. 예를 들어, "신용".

설정을 변경 한 후, 자신의 보존을 포기 버튼을 변경 한 내용을 저장하거나 "취소"할 수있는 "확인"버튼을 누릅니다.
