Réglage forme pilote de gestionnaire "ORION: registraire fiscale FR01K"

Le formulaire est destiné à définir les paramètres du bilan financier, les conducteurs pris en charge "ORION: registraire fiscale FR01K." Le formulaire vous permet de configurer les appareils suivants:

  • Port
    Port série connecté au registre fiscal.
  • La rapidité
    Le taux d'échange de données avec registraire budgétaire.
  • Temps libre
    Le temps de réponse maximal du dossier fiscal.
  • Mot de passe de l'utilisateur
    Utilisateur Mot de passe FR. Il est utilisé pour l'impression de chèques sur DF.
  • Mot de passe administrateur
    Mot de passe administrateur FR. Il est utilisé lors de l'exécution d'un DF nécessitant des privilèges administratifs.
  • Oplata1
    Texte noms d'impression sans numéraire №1 de paiement. Par exemple, "CARTE DE PAIEMENT"
  • Oplata2
    Texte noms d'impression sans numéraire №2 de paiement. Par exemple, le «crédit».

Après avoir modifié les réglages, appuyez sur le bouton "OK" pour enregistrer les modifications apportées ou sur "Annuler" pour abandonner leur conservation.
