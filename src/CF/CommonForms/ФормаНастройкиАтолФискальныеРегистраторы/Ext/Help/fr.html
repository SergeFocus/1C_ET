Réglage forme pilote de gestionnaire "ATOL: Régistrateurs fiscales"

Le formulaire est destiné à définir les paramètres des registraires budgétaires, le pilote prend en charge «ATOL: Driver CMC." Le formulaire vous permet de configurer les appareils suivants:

  • Modèle
     Modèle de l'appareil connecté.
  • Port
    Port série connecté au registre fiscal.
  • La rapidité
    Le taux d'échange de données avec registraire budgétaire.
  • Temps libre
    Le temps de réponse maximal du dossier fiscal.
  • Mot de passe de l'utilisateur
    Utilisateur Mot de passe FR. Il est utilisé pour l'impression de chèques sur DF.
  • Le mot de passe
    L'accès par mot de passe à l'appareil.nécessaire pour travailler avec le CMC dans la transmission des commandes du PC vers la caisse.Attention!Dispositif de mot de passe n'a rien à voir avec les opérateurs de mots de passe CMC.
  • Imprimer taxes en échec

    Si elle est active sur le chèque sera imprimé taux d'imposition.

Après avoir modifié les réglages, appuyez sur le bouton "OK" pour enregistrer les modifications apportées ou sur "Annuler" pour abandonner leur conservation.
