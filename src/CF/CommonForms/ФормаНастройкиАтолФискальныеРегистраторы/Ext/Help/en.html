Setting form handler driver "ATOL: Fiscal registrars"

The form is intended to set the parameters of the fiscal registrars, the driver supports "ATOL: Driver CMC." The form allows you to configure the following devices:

  • Pattern
     Model of the connected device.
  • Port
    Serial port connected to the fiscal register.
  • Speed
    Data exchange rate with fiscal registrar.
  • Timeout
    The maximum response time from the fiscal record.
  • User Password
    User Password FR. It is used for printing checks on DF.
  • Device Password
    Password access to the device.needed to work with the CMC in the transmission of commands from the PC to the cash register.Attention!Password device has nothing to do with passwords operators CMC.
  • Print taxes in check

    If it is active on the check will be printed tax rates.

After changing the settings, press the "OK" button to save the changes made or "Cancel" button to abandon their conservation.
