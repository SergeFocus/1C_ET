Réglage forme pilote de gestionnaire "ATOL: Lecteurs des cartes magnetiques"

Le formulaire est destiné à définir les paramètres des lecteurs de cartes magnétiques pour être liés à un composant externe, «ATOL:. Dispositifs d'entrée de pilote" Le formulaire vous permet de configurer les appareils suivants:

  • Port
    Port série (physique ou émulé) qui est connectée à l'appareil.
  • La rapidité
    Réglage du port auquel l'appareil est connecté; débit de données.
  • Des données de bits
    Réglage du port auquel l'appareil est connecté; nombre de bits de données en un octet transmis.
  • Bit d'arrêt
    Réglage du port auquel l'appareil est connecté; nombre de bits d'arrêt.
  • La parité
    Réglage du port auquel l'appareil est connecté; Procédé de parité.
  • Sensibilité
    Lecteurs magnétiques paramètres de carte connecté via l'interface du clavier; sensibilité.
  • Préfixes
    Préfixe ajouté à la valeur d'un dispositif de compter les cartes. Est un signe de début d'une lecture ou d'une nouvelle piste.
  • Suffixes
    Suffixe ajouté à la valeur de l'appareil à compter les cartes. Il est révélateur de l'achèvement de la piste ou la lecture.
  • Les numéros de piste
    Utilisé pour spécifier la piste sur laquelle les données est conclu entre le préfixe spécifié et la piste de suffixe. Pour suivre les données ont été transférées à la configuration que vous devez cocher la case en face de la piste.

Après avoir modifié les réglages, appuyez sur le bouton "OK" pour enregistrer les modifications apportées ou sur "Annuler" pour abandonner leur conservation.
