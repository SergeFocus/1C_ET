Réglage forme universelle pilote

Le formulaire est destiné à définir les paramètres de l'appareil à raccorder avec une technologie de pilote universel développé par «Compatible Système 1C: Enterprise". Le formulaire vous permet de configurer les paramètres nécessaires de l'appareil.

Après avoir modifié les réglages, appuyez sur le bouton "Enregistrer et fermer" pour enregistrer les modifications apportées ou "Annuler" pour abandonner leur conservation.
