Impostazione del driver gestore del modulo "Terminali di raccolta dei dati"

Il modulo è destinato per impostare i parametri dei terminali di raccolta dati. Il modulo consente di configurare i seguenti dispositivi:

  • Modello
    Modello del dispositivo collegato.
  • Porto
    La porta collegato al terminale di raccolta dati.
  • Velocità
    Il tasso di scambio di dati con il terminale.
  • IR - Stand
    Se terminale di raccolta dati è collegato al computer tramite il IR-stand è necessario installare questa opzione.
  • La base di separazione
    Il simbolo o il codice del carattere utilizzato per separare i campi nel terminale merci.
  • Documento Separatore
    Il simbolo o il codice del carattere utilizzato per separare i campi nel terminale di raccolta dati del documento.
  • Numero base
    Terminale di base Numero di merci, in cui lo scarico.
  • Numero del documento
    Numero documento terminale di raccolta dati da cui il carico.
  • Selezionare scaricare dati di origine
    Se si specifica che è necessario selezionare una sorgente dati, quindi il caricamento del TSD modellerà scelte. Il modulo dovrà specificare dove per avviare i dati: database o documenti.
  • Pulire il documento dopo il download dal TSD
    Se si specifica che è necessario pulire il documento in fase di avvio della TSD, poi dopo aver caricato il documento originale del terminale viene azzerato. Quando si avvia dalla base della base terminale non viene cancellato.

Dopo aver modificato le impostazioni, premere il tasto "OK" per salvare le modifiche apportate o "Annulla" per abbandonare la loro conservazione.
