Установка обработчиков Форма водитель "терминалов сбора данных"

Форма для установки параметров терминала сбора данных. Эта форма позволяет настроить следующие устройства:

  • Стиль
    Модель подключенного устройства.
  • Интерфейс
    Подключение к сбору данных порт терминала.
  • Ставка
    Скорость обмена данными с терминалом.
  • ИК - Бут
    Если терминал сбора данных с помощью инфракрасного колыбель, подключенный к компьютеру, необходимо установить опцию.
  • Сепаратор база
    Символ или код символа используется, чтобы отделить зону грузового терминала.
  • Отдельная документ
    Символ или код символа используется для отдельных документов в области терминал сбора данных.
  • Количество База
    Товары, в котором количество базовой терминала разряда.
  • Регистрационный номер
    Номер файла сбора данных с терминала погрузки.
  • Выберите источник загрузки данных
    Если указать необходимость выбора источника данных, то нагрузка ТСД будет формировать выбор. Форма необходимо будет указать данные загрузки: базы данных или файла.
  • После загрузки документа из чистого ТСД
    Если вы хотите, чтобы очистить ТСД указать файл запуска, а затем загрузить исходные терминалы, он будет очищен. Когда вы начинаете с основания не будет очищена от клеммной базе.

После изменения настроек, нажмите кнопку "OK", чтобы сохранить изменения, или кнопку "Отмена", чтобы отказаться от их защиты.
