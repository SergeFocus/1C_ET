Setarea formă driver handler "Scancode: afișează client"

Forma este destinat pentru a seta parametrii de display client este conectat prin intermediul driver-ul "Scancode:. Driver afișează cumpărător" Forma vă permite să configurați următoarele dispozitive:

  • Model
    Model de dispozitivul conectat.
  • Port
    COM-port de la care dispozitivul este conectat.
  • Viteză
    Rată de biți pentru portul la care dispozitivul este conectat.
  • Utilizare defilare
    Indiferent dacă sunt sau nu de afișare implicit șir salut
  • Ceas de buzunar
    Textul a liniei de bun-venit

După schimbarea setărilor, apăsați pe butonul "Salvare și închidere" pentru a salva modificările făcute sau "Cancel" pentru a abandona conservarea lor.
