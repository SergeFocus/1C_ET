Setting forması handler sürücü "SCANCODE: Customer göstərir"

forma müştəri ekran parametrləri sürücü ilə bağlıdır müəyyən etmək üçün nəzərdə tutulmuşdur "SCANCODE. Driver alıcı göstərir" form aşağıdakı cihazlar konfiqurasiya imkan verir:

  • Model
    Bağlı cihaz Model.
  • Port
    Cihaz bağlı olan COM-port.
  • Sürət
    Cihaz bağlı olan port üçün Bit dərəcəsi.
  • Scrolling istifadə
    Ekran string default təbrik olsun və ya deyil
  • Ticker
    qarşılanma xəttinin mətn

Parametrlərini dəyişdirməzdən sonra, onların qorunması tərk dəyişikliklər saxlamaq və ya "Ləğv et" üçün "Close edin və" düyməsini basın.
