Impostazione modulo autista handler "Scancode: display del cliente"

Il modulo ha lo scopo di impostare i parametri del display cliente è collegato tramite il driver "scancode:. Driver visualizza l'acquirente" Il modulo consente di configurare i seguenti dispositivi:

  • Modello
    Modello del dispositivo collegato.
  • Porto
    COM-porta a cui è collegato il dispositivo.
  • Velocità
    Bit rate per la porta a cui è collegato il dispositivo.
  • Usa scorrimento
    O meno la visualizzazione di saluto stringa predefinito
  • Ticker
    Il testo della linea di accoglienza

Dopo aver modificato le impostazioni, premere il tasto "Salva e chiudi" per salvare le modifiche apportate o "Annulla" per abbandonare la loro conservazione.
