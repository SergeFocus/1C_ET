Форма настройки обработчика драйвера "SCANCODE: Дисплеи покупателя"

Форма предназначена для настройки параметров работы дисплеев покупателя, подключаемых при помощи драйвера "SCANCODE: Драйвер дисплеев покупателя". Форма позволяет настраивать следующие параметры устройства:

  • Модель
    Модель подключаемого устройства.
  • Порт
    COM-порт, к которому подключено устройство.
  • Скорость
    Cкорость передачи данных для порта, к которому подключено устройство.
  • Использовать бегущую строку
    Выводить ли на дисплей строку приветствия по умолчанию
  • Бегущая строка
    Текст приветственной строки

После изменения значений параметров следует нажать кнопку "Записать и закрыть" для того, чтобы сохранить внесённые изменения, или кнопку "Отмена" для того, чтобы отказаться от их сохранения.
