설정 양식 처리기 드라이버 "스캔 코드 : 고객에"

형태는 고객 용 디스플레이의 파라미터는 드라이버를 통해 연결된 설정하도록 구성되는 "스캔 코드 :. 드라이버 구매자 표시" 형태는 다음과 같은 장치를 구성 할 수 있습니다 :

  • 무늬
    연결된 장치의 모델입니다.
  • 포트
    장치가 연결되어있는 COM 포트.
  • 속도
    장치가 접속 된 포트의 비트 레이트.
  • 스크롤 사용
    디스플레이 문자열 기본 인사말되는지 여부
  • 증권 시세 표시기
    환영 라인의 텍스트

설정을 변경 한 후, 자신의 보존을 포기하는 변경 사항을 저장 또는 "취소"를 "닫기 저장"버튼을 누릅니다.
