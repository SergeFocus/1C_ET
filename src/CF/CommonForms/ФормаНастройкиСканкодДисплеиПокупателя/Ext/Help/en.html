Setting form handler driver "SCANCODE: Customer displays"

The form is intended to set the parameters of the customer display is connected via the driver "SCANCODE: Driver displays the buyer." The form allows you to configure the following devices:

  • Pattern
    Model of the connected device.
  • Port
    COM-port to which the device is connected.
  • Speed
    Bit rate for the port to which the device is connected.
  • Use scrolling
    Whether or not the display string default greeting
  • Ticker
    The text of the welcoming line

After changing the settings, press the "Save and Close" to save the changes made or "Cancel" button to abandon their conservation.
