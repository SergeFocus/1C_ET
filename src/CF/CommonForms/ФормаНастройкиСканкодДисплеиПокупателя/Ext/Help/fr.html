Réglage forme pilote de gestionnaire "SCANCODE: affiche des clients"

Le formulaire est destiné à définir les paramètres d'affichage du client est connecté via le pilote "SCANCODE:. Pilote affiche l'acheteur" Le formulaire vous permet de configurer les appareils suivants:

  • Modèle
    Modèle de l'appareil connecté.
  • Port
    COM-port auquel l'appareil est connecté.
  • La rapidité
    Débit pour le port auquel l'appareil est connecté.
  • Utiliser le défilement
    Si oui ou non l'affichage voeux chaîne par défaut
  • Ticker
    Le texte de la ligne de bienvenue

Après avoir modifié les réglages, appuyez sur le bouton "Enregistrer et fermer" pour enregistrer les modifications apportées ou "Annuler" pour abandonner leur conservation.
