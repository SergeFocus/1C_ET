Setting form handler driver "CCR: Customer displays"

The form is intended to set the parameters of the customer display is connected via the drivers' CCF: The driver displays the buyer. " The form allows you to configure the following devices:

  • Pattern
    Model of the connected device.
  • Port
    COM-port to which the device is connected.
  • Speed
    Bit rate for the port to which the device is connected.
  • Parity
    Most models displays serviced driver, allow you to specify the way parity data transfer to the port to which the device is attached.
  • Data bits
    Most models displays serviced driver, allow you to specify the number of bits in the transmitted data byte to the port to which the device is attached.
  • Stop bits
    Most models displays serviced driver, allow you to specify the number of stop bits for the port to which the device is connected.

After changing the settings, press the "Save and Close" to save the changes made or "Cancel" button to abandon their conservation.
