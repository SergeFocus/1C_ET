설정 양식 처리기 드라이버 "CCR : 고객에"

양식은 드라이버 'CCF를 통해 연결되어있는 고객 용 디스플레이의 매개 변수를 설정하기위한 것입니다 : 드라이버가 구매자를 표시합니다. "형태는 다음과 같은 장치를 구성 할 수 있습니다 :

  • 무늬
    연결된 장치의 모델입니다.
  • 포트
    장치가 연결되어있는 COM 포트.
  • 속도
    장치가 접속 된 포트의 비트 레이트.
  • 패리티
    드라이버 서비스되는 대부분의 모델 디스플레이는, 사용자가 디바이스가 부착되는 포트 방식 패리티 데이터 전송을 지정할 수.
  • 데이터 비트
    드라이버 서비스되는 대부분의 모델 디스플레이는, 사용자가 어떤 장치가 연결된 포트로 전송 된 데이터 바이트의 비트 수를 특정 할 수있다.
  • 비트를 중지
    드라이버 서비스되는 대부분의 모델이 표시는 만약 장치가 연결되어있는 포트에 대한 정지 비트의 수를 특정 할 수있다.

설정을 변경 한 후, 자신의 보존을 포기하는 변경 사항을 저장 또는 "취소"를 "닫기 저장"버튼을 누릅니다.
