Réglage forme pilote de gestionnaire "CCR: affiche des clients"

Le formulaire est destiné à définir les paramètres d'affichage du client est connecté via le CCF des pilotes: Le pilote affiche l'acheteur. "Le formulaire vous permet de configurer les appareils suivants:

  • Modèle
    Modèle de l'appareil connecté.
  • Port
    COM-port auquel l'appareil est connecté.
  • La rapidité
    Débit pour le port auquel l'appareil est connecté.
  • La parité
    La plupart des modèles affiche pilote desservis, vous permettent de spécifier le transfert de données de parité de façon de le port auquel le périphérique est connecté.
  • Les bits de données
    La plupart des modèles affiche pilote desservis, vous permettent de spécifier le nombre de bits dans l'octet de données transmis au port auquel le périphérique est connecté.
  • Bits d'arrêt
    La plupart des modèles affiche pilote desservis, vous permettent de spécifier le nombre de bits d'arrêt pour le port auquel l'appareil est connecté.

Après avoir modifié les réglages, appuyez sur le bouton "Enregistrer et fermer" pour enregistrer les modifications apportées ou "Annuler" pour abandonner leur conservation.
