Форма выбора Номенклатуры с вводом количества и цены при наборе документа.

В зависимости от того, откуда вызвана эта форма, Вам будет предложено указать Номенклатуру, ее Количество и Цену.

Номенклатуру можно указывать за счет Штрих-кода (если используется), Артикула номенклатуры (если он используется), поиска по реквизитам и выбора ее из справочника.

По клавише F10 окно закроется с передачей выбранных данных в документ.

Так, для быстрого набора товаров в документе можно вызывать эту форму (F10), сканировать Штрих-код, вводить количество и возвращаться в документ (снова F10).

При вызове этой формы из документа расхода товара, если используется учет по складам и у склада указан склад пополнения, то остаток товара демонстрируется по сумме остатков этих складов.
