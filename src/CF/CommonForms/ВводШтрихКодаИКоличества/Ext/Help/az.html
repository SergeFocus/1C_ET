Bu formu adlı asılı olaraq, siz istenir NomenklaturasıOnun miqdarı və qiyməti.

Nomenklaturası tərəfindən müəyyən edilə bilər Barcode (istifadə əgər), Maddə istifadə əgər nomenklaturası (), məlumat üçün axtarış, və kataloq seçin.

F9 əsas pəncərə sənəd daxil seçilmiş məlumatın köçürülməsi ilə bağlıdır.

Belə ki, bir sənəd məhsullarının tez set üçün, siz barkod scan, bu formu (F9) zəng nömrəsini daxil edin və sənəd (yenidən F9) qayıda bilər.

Sənədin bu formu zəng zaman istehlak mallarıSiz mühasibat istifadə əgər anbarlar Set və anbar fond təchizat, malların qalan bu anbarlarının qalıqlarının məbləği göstərilir.
