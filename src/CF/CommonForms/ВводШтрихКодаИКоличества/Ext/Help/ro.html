Forma de alegerea nomenclatura cu introducerea cantitățile și prețurile stabilite la documentul.

În funcție de cazul în care te-a chemat acest formular, vi se va solicita Nomenclatură, Cantitatea și prețul acestuia.

Nomenclatura pot fi specificate prin Coduri de bare (dacă este folosit), Nomenclatura articolul (dacă este utilizat), căutarea pentru detalii, și selectați-l din directorul.

În fereastra tasta F9 este închis cu transferul de date selectate în document.

Deci, pentru un set rapid de produse într-un document, puteți apela acest formular (F9), scana codul de bare, introduceți numărul și a reveni la documentul (din nou F9).

Când de asteptare această formă a documentului bunurilor de consumDacă utilizați de contabilitate depozite Set și depozit de stoc reconstituirea, restul mărfurilor este prezentată de suma reziduuri ale acestor depozite.
