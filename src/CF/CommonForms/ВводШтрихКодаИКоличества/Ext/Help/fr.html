Formulaire de choix de la nomenclature avec l'introduction de quantités et de prix dans le document de recrutement.

Selon l'endroit où vous avez appelé ce formulaire, vous serez invitéLa nomenclature, La quantité et de prix.

Nomenclature peut être spécifié parCode à barresi elle est utilisée), L'article nomenclature (le cas échéant), la recherche pour les détails et le sélectionner dans le répertoire.

Dans la fenêtre touche F9 est fermé par le transfert des données sélectionnées dans le document.

Ainsi, vous pouvez appeler cette forme (F9), scanner le code barre pour le produit rapidement mis dans le document, entrez le numéro et revenir au document (nouveau F9).

En appelant cette forme de documentsécoulement du produitSi on utilise la comptabilitéentrepôts et l'entrepôt Set entrepôt reconstitution, le résidu de produit est démontrée par la somme des résidus de ces entrepôts.
