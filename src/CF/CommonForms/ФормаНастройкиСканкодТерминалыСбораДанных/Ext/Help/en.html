Setting form handler driver "Terminals for data collection"

The form is intended to set the parameters of the data collection terminals. The form allows you to configure the following devices:

  • Pattern
    Model of the connected device.
  • Port
    The port connected to the data collection terminal.
  • Speed
    The rate of exchange of data with the terminal.
  • IR - Stand
    If data collection terminal is connected to the computer using the IR-stand, you need to install this option.
  • The separator base
    The symbol or code of the character used to separate the fields in the goods terminal.
  • Separator document
    The symbol or code of the character used to separate the fields in the document data collection terminal.
  • Number Base
    Number base terminal of the goods, in which the discharge.
  • Document Number
    Document number data collection terminal from which the loading.
  • Select download source data
    If you specify that you need to select a data source, then the loading of the TSD will shape choices. The form will need to specify where to boot the data: database or document.
  • Clean the document after download from the TSD
    If you specify that you need to clean the document at startup of the TSD, then after loading the original document of the terminal is cleared. When you boot from the base of the terminal base is not cleared.

After changing the settings, press the "OK" button to save the changes made or "Cancel" button to abandon their conservation.
