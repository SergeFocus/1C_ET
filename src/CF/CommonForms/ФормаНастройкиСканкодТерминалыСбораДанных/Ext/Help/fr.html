Réglage forme pilote de gestionnaire "Terminals de collection de données"

Le formulaire est destiné à définir les paramètres des terminaux de collecte de données. Le formulaire vous permet de configurer les appareils suivants:

  • Modèle
    Modèle de l'appareil connecté.
  • Port
    Le port connecté à la borne de collecte de données.
  • La rapidité
    Le taux d'échange de données avec le terminal.
  • IR - Stand
    Si la borne de collecte de données est connecté à l'ordinateur via l'IR-stand, vous devez installer cette option.
  • La base de séparateur
    Le symbole ou code du caractère utilisé pour séparer les champs dans le terminal de marchandises.
  • Document de Separator
    Le symbole ou code du caractère utilisé pour séparer les champs dans le terminal de collecte de données du document.
  • Nombre base
    Base de numéro de terminal de marchandises, dans lequel la décharge.
  • numéro de document
    Numéro du document terminal de collecte de données à partir de laquelle le chargement.
  • Sélectionnez le téléchargement des données de la source
    Si vous spécifiez que vous devez sélectionner une source de données, puis le chargement de la DNT façonnera choix. Le formulaire devra spécifier où démarrer les données: base de données ou un document.
  • Nettoyez le document après le téléchargement de la DNT
    Si vous indiquez que vous avez besoin pour nettoyer le document au démarrage de la DNT, puis après le chargement du document original du terminal est effacé. Lorsque vous démarrez à partir de la base de la base de borne est pas effacée.

Après avoir modifié les réglages, appuyez sur le bouton "OK" pour enregistrer les modifications apportées ou sur "Annuler" pour abandonner leur conservation.
