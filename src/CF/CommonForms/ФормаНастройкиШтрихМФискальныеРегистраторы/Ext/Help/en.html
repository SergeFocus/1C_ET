Setting form handler driver "Barcode M: Fiscal registrars"

The form is intended to set the parameters of the fiscal registrars, the driver supports "Barcode M: FR Driver." The form allows you to configure the following devices:

  • Pattern
     Model of the connected device.
  • Port
    Serial port connected to the fiscal register.
  • Speed
    Data exchange rate with fiscal registrar.
  • Timeout
    The maximum response time from the fiscal record.
  • User Password
    User Password FR. It is used for printing checks on DF.
  • Administrator Password
    Administrator Password FR. It is used when performing a DF requiring administrative privileges.
  • Oplata1
    Text print names cashless payment №1. For example, "PAYMENT CARD"
  • Oplata2
    Text print names cashless payment №2. For example, the "credit".
  • Keep a log of operations Driver
    This property is used in the event of problems with the driver, and the DF. If it is active in the process of file operations will be created that will contain all of the operations performed by the driver, as well as results of operations.

After changing the settings, press the "OK" button to save the changes made or "Cancel" button to abandon their conservation.
