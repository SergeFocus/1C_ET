Форма настройки обработчика драйвера "Штрих-М: Фискальные регистраторы"

Форма предназначена для настройки параметров работы фискальных регистраторов, поддерживаемых драйвером «Штрих-М: Драйвер ФР». Форма позволяет настраивать следующие параметры устройства:

  • Модель
    Модель подключаемого устройства.
  • Порт
    Номер последовательного порта, к которому подключен фискальный регистратор.
  • Скорость
    Скорость обмена данными с фискальным регистратором.
  • Таймаут
    Максимальное время ожидания ответа от фискального регистратора.
  • Пароль пользователя
    Пароль пользователя ФР. Применяется при печати чеков на ФР.
  • Пароль администратора
    Пароль администратора ФР. Применяется при выполнении операций с ФР, требующих административных привелегий.
  • Оплата1
    Текст для печати наименования безналичной оплаты №1. Например, "ПЛАТЕЖНАЯ КАРТА"
  • Оплата2
    Текст для печати наименования безналичной оплаты №2. Например, "КРЕДИТ".
  • Вести журнал операций драйвера
    Данное свойство применяется при возникновении проблем при работе с драйвером и ФР. При включенном свойстве в процессе работы будет создаваться файл операций, куда будут записаны все операции, совершаемые с драйвером, а также результаты операций.

После изменения значений параметров следует нажать кнопку "ОК" для того, чтобы сохранить внесённые изменения, или кнопку "Отмена" для того, чтобы отказаться от их сохранения.
