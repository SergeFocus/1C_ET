Impostazione modulo autista handler "Barcode M: Registratori fiscali"

Il modulo ha lo scopo di impostare i parametri dei registrar fiscali, il driver supporta "Barcode M:. FR Driver" Il modulo consente di configurare i seguenti dispositivi:

  • Modello
     Modello del dispositivo collegato.
  • Porto
    Porta seriale collegata al registro fiscale.
  • Velocità
    Il tasso di cambio di dati con registrar fiscale.
  • Pausa
    Il tempo massimo di risposta dal record fiscale.
  • Password utente
    Utente Password FR. E 'utilizzato per la stampa controlli sui DF.
  • Password amministratore
    Password amministratore FR. Viene utilizzato quando si esegue una DF che richiede privilegi amministrativi.
  • Oplata1
    Nomi Stampa testo senza contanti â1 pagamento. Ad esempio, "PAGAMENTO CON CARTA"
  • Oplata2
    Nomi Stampa testo senza contanti â2 pagamento. Ad esempio, il "credito".
  • Tenere un registro delle operazioni di driver
    Questa proprietà viene utilizzata in caso di problemi con il driver, e il DF. Se è attivo nel processo di operazioni sui file saranno creati che conterranno tutte le operazioni eseguite dal conducente, nonché risultati delle operazioni.

Dopo aver modificato le impostazioni, premere il tasto "OK" per salvare le modifiche apportate o "Annulla" per abbandonare la loro conservazione.
