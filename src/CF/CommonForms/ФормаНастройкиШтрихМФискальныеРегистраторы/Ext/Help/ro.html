Setarea formă driver handler "coduri de bare M: Registratori fiscale"

Forma este destinat pentru a seta parametrii registratorii fiscale, șoferul acceptă "coduri de bare M:. FR Driver" Forma vă permite să configurați următoarele dispozitive:

  • Model
     Model de dispozitivul conectat.
  • Port
    Port serial conectat la registrul fiscal.
  • Viteză
    Rata de schimb de date cu registratorul fiscal.
  • Pauză
    Timpul maxim de răspuns din cazierul fiscal.
  • Parolă de utilizator
    Utilizator Parola FR. Acesta este utilizat pentru imprimarea controalelor la DF.
  • Administrator Password
    Administrator Password FR. Acesta este utilizat atunci când efectuează o DF necesită privilegii administrative.
  • Oplata1
    Text nume de imprimare №1 plată fără numerar. De exemplu, "card de plată"
  • Oplata2
    Text nume de imprimare nr.2 plată fără numerar. De exemplu, "credit".
  • Păstrați un jurnal de operațiuni pilot
    Această proprietate este folosită în cazul unor probleme cu driver-ul, și DF. În cazul în care este activ în procesul de operațiuni de fișier va fi creat, care va conține toate operațiunile efectuate de către conducătorul auto, precum și rezultatele operațiunilor.

După schimbarea setărilor, apăsați pe butonul "OK" pentru a salva modificările făcute sau "Cancel" pentru a abandona conservarea lor.
