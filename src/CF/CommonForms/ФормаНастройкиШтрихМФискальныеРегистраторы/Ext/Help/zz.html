Установка драйверов форма обработчик "Штрих М: административный регистратор"

Эта форма для установки параметров административного записи, драйвер поддерживает "Штрих М: водитель FR." Эта форма позволяет настроить следующие устройства:

  • Стиль
     Модель подключенного устройства.
  • Интерфейс
    Последовательное подключение к финансовым регистров.
  • Ставка
    Скорость обмена данными и финансовый директор.
  • Время
    Максимальное время реакции от финансовых отчетах.
  • Пароль
    Пароль FR. Он используется для проверки DF печати.
  • Пароль администратора
    Пароль администратора FR. DF Время, необходимое для выполнения только с правами администратора.
  • Oplata1
    Текст для печати безналичный расчет №1. Например, "Платежная карта"
  • Oplata2
    Текст для печати безналичный расчет №2. Например, "кредит".
  • Провести операцию журнал драйверов
    Это свойство используется в случае проблемных драйверов и событий DF. Если он активен во время файловых операций будут созданы, который будет содержать все операции, выполняемые водителем и операционные результаты.

После изменения настроек, нажмите кнопку "OK", чтобы сохранить изменения, или кнопку "Отмена", чтобы отказаться от их защиты.
