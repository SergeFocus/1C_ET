Nastavení ovladače formulář handler "Barcode M: registrátory fiksální"

Formulář slouží k nastavení parametrů fiskálních registrátorů, ovladač podporuje "Čárový kód M:. FR Driver" Formulář umožňuje nakonfigurovat následující zařízení:

  • Pattern
     Model připojeného zařízení.
  • Port
    Sériový port připojený k fiskální rejstříku.
  • Rychlost
    Směnný kurz dat s fiskálním registrátora.
  • Čas vypršel
    Maximální doba odezvy od fiskálního záznamu.
  • Uživatel Heslo
    User Password FR. Používá se pro tisk kontroly DF.
  • Heslo správce
    Heslo správce FR. Používá se při provádění DF vyžadující oprávnění správce.
  • Oplata1
    Textu Vytisknout jména bezhotovostní platba №1. Například, "PLATEBNÍ KARTY"
  • Oplata2
    Textu Vytisknout jména bezhotovostní platba №2. Například, "úvěr".
  • Mějte protokolem o operacích Driver
    Tato vlastnost se používá v případě problémů s řidičem, a DF. Pokud je aktivní v procesu operací soubor bude vytvořen, která bude obsahovat všechny operace prováděné řidiče, jakož i výsledky operací.

Po změně nastavení stiskněte tlačítko "OK" pro uložení změny nebo tlačítko "Zrušit" opouštět jejich zachování.
