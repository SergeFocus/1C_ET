"Məlumatların toplanması üçün Terminals" şəklində handler sürücü qurulması

forma məlumatların toplanması terminalların parametrləri təyin etmək üçün nəzərdə tutulmuşdur. form aşağıdakı cihazlar konfiqurasiya imkan verir:

  • Model
    Bağlı cihaz Model.
  • Port
    məlumatların toplanması terminal bağlı port.
  • Sürət
    terminalı ilə məlumat mübadiləsi dərəcəsi.
  • IR - Stand
    Məlumatların toplanması terminal IR-stand istifadə kompüter bağlı varsa, siz bu seçimi yüklemek üçün lazımdır.
  • separator bazası
    simvolu və ya xarakterli kodu malları terminalında sahələri ayırmaq üçün istifadə olunur.
  • Separator sənəd
    simvolu və ya xarakterli kodu sənəd məlumatların toplanması terminal sahələri ayırmaq üçün istifadə olunur.
  • Number Base
    Malları, axıdılması sayı bazası terminal.
  • Document sayı
    Document sayı məlumatların toplanması terminal olan loading olan.
  • Select yukle mənbə məlumat
    Siz məlumat mənbəyi seçmək lazımdır ki, müəyyən, onda TSD yüklənməsi seçim formalaşdırmaq olacaq. məlumat açılış harada forma müəyyən etmək lazımdır: verilənlər bazası və ya sənəd.
  • TSD yükləmək sonra sənəd təmizləmək
    Siz TSD başlanğıc sənəd təmizləmək lazımdır ki, müəyyən, onda terminal orijinal sənəd loading sonra təmizlənib. Siz terminal bazasının bazasının açılış zaman rəsmiləşdirilməyib olunur.

Parametrlərini dəyişdirməzdən sonra, onların qorunması tərk düyməsinə dəyişikliklər saxlamaq və ya "Ləğv et" üçün "OK" düyməsini basın.
