"데이터 수집 터미널"양식 처리기 드라이버 설정

폼은 데이터 수집 단말기 매개 변수를 설정하기위한 것이다. 형태는 다음과 같은 장치를 구성 할 수 있습니다 :

  • 무늬
    연결된 장치의 모델입니다.
  • 포트
    데이터 수집 단자에 접속 포트.
  • 속도
    단말기와 데이터 교환의 속도입니다.
  • IR - 스탠드
    데이터 수집 단말기는 IR-스탠드를 사용하여 컴퓨터에 연결되어있는 경우,이 옵션을 설치해야합니다.
  • 분리베이스
    기호 나 문자의 코드는 제품 단말기의 필드를 구분하는 데 사용.
  • 구분 문서
    기호 나 문자 코드는 문서 데이터 수집 단말기의 필드를 구분하는데 사용.
  • 번호 자료
    상품에 방전 수의베이스 단자.
  • 문서 번호
    문서 번호 데이터 수집 터미널있는로드에서.
  • 선택 다운로드 소스 데이터
    당신이 데이터 소스를 선택해야한다는 것을 지정하면, TSD의 로딩은 선택을 형성 할 것이다. 데이터를 부팅 할 경우 양식은 지정해야합니다 : 데이터베이스 또는 문서를.
  • TSD에서 다운로드 한 후 문서를 청소
    당신이 TSD의 시작에서 문서를 청소해야한다는 것을 지정하면, 단말기의 원본 문서를로드 한 후 삭제됩니다. 터미널베이스의베이스에서 부팅 할 때 지워지지 않습니다.

설정을 변경 한 후, 자신의 보존을 포기 버튼을 변경 한 내용을 저장하거나 "취소"할 수있는 "확인"버튼을 누릅니다.
