Форма налаштування обробника драйвера "КасбіФР01К: Фіскальні реєстратори"

Форма призначена для налаштування параметрів роботи фіскальних реєстраторів, підтримуваних драйвером «Штрих-М: Драйвер ФР». Форма дозволяє настроювати наступні параметри пристрою:

  • Модель
     Модель підключається.
  • Порт
    Номер послідовного порту, до якого підключений фіскальний реєстратор.
  • Швидкість
    Швидкість обміну даними з фіскальним реєстратором.
  • Таймаут
    Максимальний час очікування відповіді від фіскального реєстратора.
  • Пароль користувача
    Пароль користувача ФР. Застосовується при друку чеків на ФР.
  • Пароль адміністратора
    Пароль адміністратора ФР. Застосовується при виконанні операцій з ФР, що вимагають адміністративних привелегий.
  • Оплата1
    Текст для друку найменування безготівкової оплати №1. Наприклад, "ПЛАТІЖНА КАРТА"
  • Оплата2
    Текст для друку найменування безготівкової оплати №2. Наприклад, "КРЕДИТ".
  • Вести журнал операцій драйвера
    Дана властивість застосовується при виникненні проблем при роботі з драйвером і ФР. При включеному властивості в процесі роботи буде створюватися файл операцій, куди будуть записані всі операції, що здійснюються з драйвером, а також результати операцій.

Після зміни значень параметрів слід натиснути кнопку "ОК" для того, щоб зберегти внесені зміни, або кнопку "Скасувати" для того, щоб відмовитися від їх збереження.
