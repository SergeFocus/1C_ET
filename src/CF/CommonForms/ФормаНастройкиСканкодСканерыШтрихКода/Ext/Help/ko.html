설정 양식 처리기 드라이버 "스캔 코드 : 바코드 스캐너"

양식은 외국의 구성 요소를 사용하여 연결된 바코드 스캐너의 매개 변수를 설정하기위한 것입니다 "스캔 코드 :. 바코드 스캐너를" 형태는 다음과 같은 장치를 구성 할 수 있습니다 :

  • 포트
    바코드 스캐너에 접속되는 직렬 포트 (물리적 또는 에뮬레이트).
  • 속도
    포트를 설정하는 것은 스캐너에 접속된다; 통신 속도.
  • 비트 데이터
    포트를 설정하는 것은 스캐너에 접속된다; 전송 된 바이트의 데이터 비트의 수.
  • 정지 비트
    포트를 설정하는 것은 스캐너에 접속된다; 정지 비트의 수.
  • 접두사
    접두사 스캐너 바코드 판독 값에 추가됩니다. 일반적으로, 바코드 판독기의 시작의 상징.
  • 접미사
    접미사 바코드 스캐너에 의해 판독 된 값에 추가된다. 이는 바코드의 판독의 완료를 나타낸다.

설정을 변경 한 후, 자신의 보존을 포기하는 변경 사항을 저장 또는 "취소"를 "닫기 저장"버튼을 누릅니다.
