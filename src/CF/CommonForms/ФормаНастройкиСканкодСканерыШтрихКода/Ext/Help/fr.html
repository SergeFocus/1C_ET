Réglage forme pilote de gestionnaire "SCANCODE: scanners de codes à barres"

Le formulaire est destiné à définir les paramètres des scanners de codes à barres connectés en utilisant des composants étrangers "SCANCODE:. Scanner de codes barres" Le formulaire vous permet de configurer les appareils suivants:

  • Port
    Port série (physique ou émulé) qui est connecté à un scanner de code à barres.
  • La rapidité
    Réglage du port est connecté au scanner; débit de données.
  • Des données de bits
    Réglage du port est connecté au scanner; nombre de bits de données en un octet transmis.
  • Bit d'arrêt
    Réglage du port est connecté au scanner; nombre de bits d'arrêt.
  • Préfixe
    Préfixe ajouté à la valeur lue par le code à barres du scanner. Typiquement, un signe de début de la lecture de code à barres.
  • Suffixe
    Suffixe à être ajouté à la valeur lue par le code à barres du scanner. Il est indicatif de l'achèvement de la lecture du code à barres.

Après avoir modifié les réglages, appuyez sur le bouton "Enregistrer et fermer" pour enregistrer les modifications apportées ou "Annuler" pour abandonner leur conservation.
