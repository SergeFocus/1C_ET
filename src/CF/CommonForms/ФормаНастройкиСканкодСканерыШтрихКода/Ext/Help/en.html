Setting form handler driver "SCANCODE: barcode scanners"

The form is intended to set the parameters of the bar code scanners connected using foreign components "SCANCODE: barcode scanner." The form allows you to configure the following devices:

  • Port
    Serial port (physical or emulated) that is connected to a barcode scanner.
  • Speed
    Setting the port is connected to the scanner; data rate.
  • Bit data
    Setting the port is connected to the scanner; number of data bits in a transmitted byte.
  • Stop bit
    Setting the port is connected to the scanner; number of stop bits.
  • Prefix
    Prefix added to the value read by the scanner barcode. Typically, a sign of the beginning of the barcode reader.
  • Suffix
    Suffix to be added to the value read by the scanner barcode. It is indicative of the completion of the reading of the barcode.

After changing the settings, press the "Save and Close" to save the changes made or "Cancel" button to abandon their conservation.
