טופס נהג הגדרת מטפל "ATOL: מציג לקוחות"

הטופס מיועד לקבוע את הפרמטרים של תצוגת לקוח מחוברים דרך הנהג "ATOL:. נהג מציג את הקונה" הטופס מאפשר לך להגדיר את המכשירים הבאים:

  • תבנית
    דגם של ההתקן המחובר.
  • נמל
    הדגמים מציג את רוב שירות נהג, מאפשר לך לציין את היציאה שלמכשיר מחובר.
  • מהירות
    הדגמים מציג את רוב שירות נהג, מאפשר לך לציין את קצב הנתונים לנמל שלמכשיר מצורף.
  • זוגיות
    הדגמים מציג את רוב שירות נהג, מאפשר לך לציין העברת נתונים זוגיות הדרך לנמל שאליו המכשיר מחובר.
  • ביטים של נתונים
    הדגמים מציג את רוב שירות נהג, מאפשר לך לציין את מספר הסיביות בבתי נתונים המועברים לנמל שאליו המכשיר מחובר.
  • תפסיק ביטים
    הדגמים מציג את רוב שירות נהג, מאפשר לך לציין את מספר סיביות תחנה לנמל שלהתקן מחובר.

לאחר שינוי ההגדרות, לחץ על הכפתור "אישור" כדי לשמור את השינויים שנעשו או "ביטול" כפתור לנטוש שימורם.
