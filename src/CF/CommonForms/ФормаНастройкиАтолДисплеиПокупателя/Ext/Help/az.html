Setting forması handler sürücü "ATOL: Customer göstərir"

forma müştəri ekran parametrləri sürücü ilə bağlıdır müəyyən etmək üçün nəzərdə tutulmuşdur "ATOL. Driver alıcı göstərir" form aşağıdakı cihazlar konfiqurasiya imkan verir:

  • Model
    Bağlı cihaz Model.
  • Port
    Sürücü xidmət ən modelləri göstərir, cihaz əlavə olunur hansı port müəyyən etməyə imkan verir.
  • Sürət
    Sürücü xidmət ən modelləri göstərir, cihaz əlavə olunur port üçün məlumat dərəcəsi müəyyən etməyə imkan verir.
  • Bərabərlik
    Sürücü xidmət ən modelləri göstərir, cihaz əlavə olunur olan limanına yol paritet ötürülməsi müəyyən etməyə imkan verir.
  • Məlumat bit
    Sürücü xidmət ən modelləri göstərir, hansı cihaz əlavə etmək limanına ötürülən məlumatların byte bit sayı müəyyən imkan verir.
  • Bit Stop
    Sürücü xidmət ən modelləri göstərir, siz cihaz bağlı port üçün stop bit sayı müəyyən imkan verir.

Parametrlərini dəyişdirməzdən sonra, onların qorunması tərk düyməsinə dəyişikliklər saxlamaq və ya "Ləğv et" üçün "OK" düyməsini basın.
