Impostazione modulo autista handler "ATOL: display del cliente"

Il modulo ha lo scopo di impostare i parametri del display cliente è collegato tramite il driver "ATOL:. Driver visualizza l'acquirente" Il modulo consente di configurare i seguenti dispositivi:

  • Modello
    Modello del dispositivo collegato.
  • Porto
    La maggior parte dei modelli di display con servizi autista, consentono di specificare la porta a cui è collegato il dispositivo.
  • Velocità
    La maggior parte dei modelli di display con servizi autista, consentono di specificare la velocità dei dati per la porta a cui il dispositivo è collegato.
  • Parità
    La maggior parte dei modelli di display con servizi autista, consentono di specificare il trasferimento dei dati di parità fino alla porta a cui è collegato il dispositivo.
  • Bit di dati
    La maggior parte dei modelli di display con servizi autista, consentono di specificare il numero di bit nel byte di dati trasmessi alla porta a cui è collegato il dispositivo.
  • Bit di stop
    La maggior parte dei modelli di display con servizi autista, consentono di specificare il numero di bit di stop per la porta a cui il dispositivo è collegato.

Dopo aver modificato le impostazioni, premere il tasto "OK" per salvare le modifiche apportate o "Annulla" per abbandonare la loro conservazione.
