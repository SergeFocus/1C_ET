Форма налаштування обробника драйвера "АТОЛ: Дисплеї покупця"

Форма призначена для налаштування параметрів роботи дисплеїв покупця, що підключаються за допомогою драйвера "АТОЛ: Драйвер дисплеїв покупця". Форма дозволяє настроювати наступні параметри пристрою:

  • Модель
    Модель підключається.
  • Порт
    Більшість моделей дисплеїв, що обслуговуються драйвером, дозволяють вказати порт, до якого підключений.
  • Швидкість
    Більшість моделей дисплеїв, що обслуговуються драйвером, дозволяють вказати швидкість передачі даних для порту, до якого підключений.
  • Парність
    Більшість моделей дисплеїв, що обслуговуються драйвером, дозволяють вказати спосіб контролю парності при передачі даних для порту, до якого підключений.
  • Біти даних
    Більшість моделей дисплеїв, що обслуговуються драйвером, дозволяють вказати кількість біт даних в переданому байті для порту, до якого підключений.
  • Стоп біти
    Більшість моделей дисплеїв, що обслуговуються драйвером, дозволяють вказати кількість стоп-біт для порту, до якого підключений.

Після зміни значень параметрів слід натиснути кнопку "ОК" для того, щоб зберегти внесені зміни, або кнопку "Скасувати" для того, щоб відмовитися від їх збереження.
