설정 양식 처리기 드라이버 "ATOL : 고객에"

형태는 고객 용 디스플레이의 파라미터는 드라이버를 통해 연결된 설정하고자 "ATOL :. 드라이버 구매자 표시" 형태는 다음과 같은 장치를 구성 할 수 있습니다 :

  • 무늬
    연결된 장치의 모델입니다.
  • 포트
    드라이버를 서비스 대부분의 모델 표시, 당신이 장치가 연결된 포트를 지정할 수 있습니다.
  • 속도
    드라이버 서비스되는 대부분의 모델 디스플레이는, 사용자가 디바이스가 연결된 포트에 대한 데이터 레이트를 지정할 수.
  • 패리티
    드라이버 서비스되는 대부분의 모델 디스플레이는, 사용자가 디바이스가 부착되는 포트 방식 패리티 데이터 전송을 지정할 수.
  • 데이터 비트
    드라이버 서비스되는 대부분의 모델 디스플레이는, 사용자가 어떤 장치가 연결된 포트로 전송 된 데이터 바이트의 비트 수를 특정 할 수있다.
  • 비트를 중지
    드라이버 서비스되는 대부분의 모델이 표시는 만약 장치가 연결되어있는 포트에 대한 정지 비트의 수를 특정 할 수있다.

설정을 변경 한 후, 자신의 보존을 포기 버튼을 변경 한 내용을 저장하거나 "취소"할 수있는 "확인"버튼을 누릅니다.
