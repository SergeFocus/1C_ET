Форма настройки обработчика драйвера "АТОЛ: Дисплеи покупателя"

Форма предназначена для настройки параметров работы дисплеев покупателя, подключаемых при помощи драйвера "АТОЛ: Драйвер дисплеев покупателя". Форма позволяет настраивать следующие параметры устройства:

  • Модель
    Модель подключаемого устройства.
  • Порт
    Большинство моделей дисплеев, обслуживаемых драйвером, позволяют указать порт, к которому подключено устройство.
  • Скорость
    Большинство моделей дисплеев, обслуживаемых драйвером, позволяют указать скорость передачи данных для порта, к которому подключено устройство.
  • Чётность
    Большинство моделей дисплеев, обслуживаемых драйвером, позволяют указать способ контроля чётности при передаче данных для порта, к которому подключено устройство.
  • Биты данных
    Большинство моделей дисплеев, обслуживаемых драйвером, позволяют указать количество бит данных в передаваемом байте для порта, к которому подключено устройство.
  • Стоп биты
    Большинство моделей дисплеев, обслуживаемых драйвером, позволяют указать количество стоп-бит для порта, к которому подключено устройство.

После изменения значений параметров следует нажать кнопку "ОК" для того, чтобы сохранить внесённые изменения, или кнопку "Отмена" для того, чтобы отказаться от их сохранения.
