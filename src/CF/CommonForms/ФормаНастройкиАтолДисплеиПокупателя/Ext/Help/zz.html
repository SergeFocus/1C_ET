Установка формы обработчик драйвера "АТОЛ: Дисплей покупателя"

Форма для установки параметров дисплея покупателя на связи водителя "АТОЛ: Покупатель управляемой шоу" Эта форма позволяет настроить следующие устройства:

  • Стиль
    Модель подключенного устройства.
  • Интерфейс
    Большинство моделей показать водителю сервис, позволяющий указать порт, к которому подключен прибор.
  • Ставка
    Большинство моделей показать водителю службы, он позволяет указать скорость передачи данных порта для подключения устройств.
  • Соотношение
    Большинство моделей показать водителю службы, в котором можно указать путь данные четности к порту подключено устройство.
  • Биты данных
    Большинство моделей показать водителю службы, он позволяет указать байт данных на число битов, передаваемых в устройства, подключенного к порту.
  • Стоп бит
    Большинство моделей показать водителю службы, он позволяет указать количество стоп-битов порта для подключения устройств.

После изменения настроек, нажмите кнопку "OK", чтобы сохранить изменения, или кнопку "Отмена", чтобы отказаться от их защиты.
