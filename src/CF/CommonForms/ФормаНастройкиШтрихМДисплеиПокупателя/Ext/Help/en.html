Setting form handler drivers "Bar-M: Displays the buyer"

The form is intended to set the parameters of the customer display is connected via the driver "Bar-M: driver displays the buyer." The form allows you to configure the following devices:

  • Pattern
    Model of the connected device.
  • Port
    COM-port to which the device is connected.
  • Timeout
     Response time of the device.
  • The number of repetitions
     The maximum number of connection attempts to the device
  • Appears on the display
    Because the device is only one row, and at most - two, it is possible to choose to display either the first or second. The first line is usually specified product name, and the second - the total amount.

After changing the settings, press the "Save and Close" to save the changes made or "Cancel" button to abandon their conservation.
