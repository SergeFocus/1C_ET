Réglage formulaire pilotes de gestionnaire "Bar-M: Affiche l'acheteur"

Le formulaire est destiné à définir les paramètres d'affichage du client est connecté via le pilote "Bar-M:. Pilote affiche l'acheteur" Le formulaire vous permet de configurer les appareils suivants:

  • Modèle
    Modèle de l'appareil connecté.
  • Port
    COM-port auquel l'appareil est connecté.
  • Temps libre
     Temps de réponse de l'appareil.
  • Le nombre de répétitions
     Le nombre maximum de tentatives de connexion à l'appareil
  • Apparaît sur l'écran
    Parce que le dispositif est une seule ligne, et au plus - deux, il est possible de choisir d'afficher soit la première ou la deuxième. La première ligne est généralement spécifiée nom du produit, et le second - le montant total.

Après avoir modifié les réglages, appuyez sur le bouton "Enregistrer et fermer" pour enregistrer les modifications apportées ou "Annuler" pour abandonner leur conservation.
