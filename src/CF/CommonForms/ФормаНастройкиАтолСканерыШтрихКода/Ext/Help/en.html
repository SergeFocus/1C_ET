Setting form handler driver "ATOL: barcode scanners"

The form is intended to set the parameters of the bar code scanner, is connected via the external components "ATOL: Driver input devices." The form allows you to configure the following devices:

  • Pattern
    Model of the connected device.
  • Port
    The port is connected to the scanner.
  • Speed
    Setting the port is connected to the scanner; data rate.
  • Bit data
    Setting the port is connected to the scanner; number of data bits in a transmitted byte.
  • Stop bit
    Setting the port is connected to the scanner; number of stop bits.
  • Parity
    Setting the port is connected to the scanner; parity method.
  • Sensitivity
    Parameter barcode scanner connected via the keyboard interface; sensitivity.
  • Prefix
    Prefix scanner shtrihkooda to the value of the read barcodes.
  • Suffix
    Suffix to be added to the value of the barcode scanner of the read barcodes. It is indicative of the completion of the reading of the barcode.

After changing the settings, press the "OK" button to save the changes made or "Cancel" button to abandon their conservation.
