Réglage forme pilote de gestionnaire "ATOL: scanners de codes à barres"

Le formulaire est destiné à définir les paramètres du scanner de code à barres, est reliée par les composants externes "ATOL: les dispositifs d'entrée du pilote". Le formulaire vous permet de configurer les appareils suivants:

  • Modèle
    Modèle de l'appareil connecté.
  • Port
    Le port est connecté au scanner.
  • La rapidité
    Réglage du port est connecté au scanner; débit de données.
  • Des données de bits
    Réglage du port est connecté au scanner; nombre de bits de données en un octet transmis.
  • Bit d'arrêt
    Réglage du port est connecté au scanner; nombre de bits d'arrêt.
  • La parité
    Réglage du port est connecté au scanner; Procédé de parité.
  • Sensibilité
    Paramètre lecteur de codes barres connecté via l'interface de clavier; sensibilité.
  • Préfixe
    Préfixe scanner shtrihkooda à la valeur des codes à barres lus.
  • Suffixe
    Suffixe à être ajouté à la valeur du scanner de codes barres des codes à barres lus. Il est indicatif de l'achèvement de la lecture du code à barres.

Après avoir modifié les réglages, appuyez sur le bouton "OK" pour enregistrer les modifications apportées ou sur "Annuler" pour abandonner leur conservation.
