Установка формы обработчик драйвера "АТОЛ: сканер штрих-кода"

Форма для установки параметров сканера штрих-кода, подключенного через внешнего элемента "АТОЛ: Драйвер устройств ввода" Эта форма позволяет настроить следующие устройства:

  • Стиль
    Модель подключенного устройства.
  • Интерфейс
    Порт подключен к сканеру.
  • Ставка
    Установите порт подключен к сканеру; скорость передачи данных.
  • Бит данных
    Установите порт подключен к сканеру; число битов данных в передаваемых данных.
  • Стоп бит
    Установите порт подключен к сканеру; стоп-бит.
  • Соотношение
    Установите порт подключен к сканеру; методы контроля четности.
  • Чувствительность
    Штрих параметры интерфейса сканера через клавиатуру; чувствительность.
  • Префикс
    Приставка для чтения значения сканер штрих-кода shtrihkooda.
  • Суффикс
    Суффикс для добавления к значению штрих-кода чтения сканером штрих-кодов. Она представляет собой конечный штрих-кода считывается.

После изменения настроек, нажмите кнопку "OK", чтобы сохранить изменения, или кнопку "Отмена", чтобы отказаться от их защиты.
