Setarea formă driver handler "Atol: Scanere coduri de bare"

Forma este destinat pentru a seta parametrii de cititor de coduri de bare, este conectat prin componentele externe "ATOL:. Dispozitivele de intrare Driver" Forma vă permite să configurați următoarele dispozitive:

  • Model
    Model de dispozitivul conectat.
  • Port
    Portul este conectat la scaner.
  • Viteză
    Setarea portului este conectat la scaner; rata de date.
  • Date Bit
    Setarea portului este conectat la scaner; Numărul de biți de date într-un octet transmis.
  • Bit de stop
    Setarea portului este conectat la scaner; numărul de biți de stop.
  • Paritate
    Setarea portului este conectat la scaner; Metoda paritate.
  • Sensibilitate
    Cititoare coduri de bare parametru conectat prin interfața tastaturii; sensibilitate.
  • Prefix
    Prefix shtrihkooda scanner la valoarea codurilor de bare de citire.
  • Sufix
    Sufix care se adaugă la valoarea scanerului de coduri de bare de codurile de bare citite. Acesta este un indicator al finalizarea citirea codului de bare.

După schimbarea setărilor, apăsați pe butonul "OK" pentru a salva modificările făcute sau "Cancel" pentru a abandona conservarea lor.
