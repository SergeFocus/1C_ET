설정 양식 처리기 드라이버 "ATOL : 바코드 스캐너"

폼은 외부 구성 요소를 통해 연결되어, 바코드 스캐너의 파라미터를 설정하기위한 것이다 "ATOL :. 드라이버 입력 장치" 형태는 다음과 같은 장치를 구성 할 수 있습니다 :

  • 무늬
    연결된 장치의 모델입니다.
  • 포트
    포트 스캐너에 접속된다.
  • 속도
    포트를 설정하는 것은 스캐너에 접속된다; 통신 속도.
  • 비트 데이터
    포트를 설정하는 것은 스캐너에 접속된다; 전송 된 바이트의 데이터 비트의 수.
  • 정지 비트
    포트를 설정하는 것은 스캐너에 접속된다; 정지 비트의 수.
  • 패리티
    포트를 설정하는 것은 스캐너에 접속된다; 패리티 방법.
  • 감광도
    키보드 인터페이스를 통해 접속 파라미터 바코드 스캐너; 감도.
  • 접두사
    읽기 바코드의 값에 접두사 스캐너 shtrihkooda.
  • 접미사
    접미사 판독 된 바코드의 바코드 스캐너의 값에 추가한다. 이는 바코드의 판독의 완료를 나타낸다.

설정을 변경 한 후, 자신의 보존을 포기 버튼을 변경 한 내용을 저장하거나 "취소"할 수있는 "확인"버튼을 누릅니다.
