Forme paramètres du pilote équipement connecté "Barcode M: Caisse"

Le formulaire est destiné à définir les paramètres de l'équipement pouvant être connectés avec les pilotes de la technologie développés "Compatible Système 1C: Enterprise". Le formulaire vous permet de configurer les paramètres nécessaires de l'appareil.

Après avoir modifié les réglages, appuyez sur le bouton "Enregistrer et fermer" pour enregistrer les modifications apportées ou "Annuler" pour abandonner leur conservation.
